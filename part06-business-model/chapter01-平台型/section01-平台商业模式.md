# 平台商业模式


## 核心特征

- 网络效应驱动
- 双边/多边市场
- 边际成本极低
- 赢家通吃

## 盈利模式

| 模式 | 例子 |
|------|------|
| 佣金 | Airbnb、Uber |
| 广告 | Google、Meta |
| 订阅 | LinkedIn Premium |
| 增值服务 | App Store |

## 护城河

- 网络效应
- 数据优势
- 转换成本

2. 为什么平台型公司容易形成赢家通吃？
