# 订阅商业模式


## 核心特征

- 经常性收入（ARR/MRR）
- 高客户留存率
- 可预测现金流
- 高客户生命周期价值

## 关键指标

| 指标 | 公式 |
|------|------|
| ARR | 年度经常性收入 |
| MRR | 月度经常性收入 |
| Churn Rate | 客户流失率 |
| NRR | 净收入留存率 |
| LTV/CAC | 客户终身价值/获客成本 |

## 优势

- 收入可预测
- 客户粘性高
- 规模效应强
