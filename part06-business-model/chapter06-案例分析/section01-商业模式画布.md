# 商业模式画布


## 九个要素

| 要素 | 问题 |
|------|------|
| 价值主张 | 为客户解决什么问题？ |
| 客户细分 | 目标客户是谁？ |
| 渠道通路 | 如何触达客户？ |
| 客户关系 | 如何维护客户？ |
| 收入来源 | 如何赚钱？ |
| 核心资源 | 需要什么资源？ |
| 关键业务 | 需要做什么？ |
| 重要合作 | 需要什么合作伙伴？ |
| 成本结构 | 主要成本是什么？ |
