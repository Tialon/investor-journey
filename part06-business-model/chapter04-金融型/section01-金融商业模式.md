# 金融商业模式

## 核心特征

- 利差/手续费驱动
- 高杠杆
- 监管严格
- 周期性强

## 盈利模式

| 类型 | 模式 |
|------|------|
| 银行 | 存贷利差 |
| 保险 | 承保利润+投资收益 |
| 支付 | 交易手续费 |
| 资管 | 管理费 |
