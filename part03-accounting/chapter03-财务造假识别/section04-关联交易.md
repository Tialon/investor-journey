# 关联交易

## 定义

> **关联交易**是指公司与关联方（控股股东、管理层、子公司等）之间的交易。

## 关联方类型

| 类型 | 说明 |
|------|------|
| 控股股东 | 持有公司5%以上股份的股东 |
| 管理层 | CEO、CFO等高管 |
| 子公司 | 公司控制的企业 |
| 关联企业 | 与控股股东有关联的企业 |
| 家庭成员 | 高管的配偶、子女等 |

## 关联交易的风险

### 1. 利益输送

通过不公平的关联交易，将公司利益转移给关联方。

| 手法 | 说明 |
|------|------|
| 高价采购 | 向关联方高价采购原材料 |
| 低价销售 | 向关联方低价出售产品 |
| 高价租赁 | 向关联方高价租赁物业 |
| 虚假服务 | 支付高额咨询费等 |

### 2. 虚增收入

通过关联交易虚增收入。

| 手法 | 说明 |
|------|------|
| 虚假销售 | 向关联方虚假销售 |
| 循环交易 | 关联方买回来 |
| 提前确认 | 不合理的收入确认 |

### 3. 隐藏债务

通过关联方隐藏公司真实负债。

## 识别方法

### 定量信号

| 信号 | 说明 |
|------|------|
| 关联交易占比高 | > 30% 需警惕 |
| 关联交易毛利率异常 | 与非关联交易差异大 |
| 应收关联方款项大 | 可能是利益输送 |
| 关联交易增长快 | 可能在扩大利益输送 |

### 定性信号

| 信号 | 说明 |
|------|------|
| 关联交易披露不充分 | 可能在隐藏问题 |
| 关联交易频繁变动 | 可能在规避监管 |
| 管理层薪酬异常高 | 可能是利益输送 |
| 关联方财务状况差 | 可能无法偿还 |

## 典型案例

### 案例：利益输送

某上市公司：
- 向控股股东的公司高价采购
- 采购价格高于市场价30%
- 关联交易占总采购的60%

**实质**：控股股东通过关联交易转移公司利润。

### 案例：虚增收入

某公司：
- 向关联方销售产品
- 关联方再卖回给公司
- 双方都确认收入

**实质**：循环交易虚增收入。

## 分析框架

```
1. 关联交易规模
   ├── 占收入/采购比例
   ├── 与同行对比
   └── 趋势变化

2. 关联交易定价
   ├── 与市场价格对比
   ├── 毛利率差异
   └── 是否有第三方评估

3. 关联交易披露
   ├── 是否充分披露
   ├── 是否有独立董事意见
   └── 是否有审计师意见

4. 关联方状况
   ├── 关联方财务状况
   ├── 关联方业务
   └── 关联方与管理层关系
```
