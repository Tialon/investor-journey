# 商誉减值风险

## 什么是商誉

$$\text{商誉} = \text{收购价格} - \text{被收购公司净资产公允价值}$$

**例子**：用100亿收购一家净资产50亿的公司 → 商誉 = 50亿

## 商誉减值的触发条件

| 条件 | 说明 |
|------|------|
| 被收购公司业绩不达预期 | 盈利能力下降 |
| 行业环境恶化 | 市场萎缩 |
| 管理层变动 | 核心团队离开 |
| 协同效应未实现 | 收购目的未达成 |

## 商誉减值的影响

| 影响 | 说明 |
|------|------|
| 净利润大幅下降 | 一次性大额减值 |
| 股东权益减少 | 资产负债表恶化 |
| ROE被动提高 | 分母（权益）减少 |
| 不影响现金流 | 非现金费用 |

## 商誉减值风险评估

### 高风险信号

| 信号 | 说明 |
|------|------|
| 商誉占净资产比例高 | > 50% 需警惕 |
| 频繁高溢价收购 | 管理层激进 |
| 被收购公司业绩下滑 | 减值风险增加 |
| 行业环境恶化 | 收购标的贬值 |
| 核心团队离开 | 协同效应可能落空 |

### 定量指标

| 指标 | 高风险 | 低风险 |
|------|--------|--------|
| 商誉/净资产 | > 50% | < 20% |
| 商誉/总资产 | > 30% | < 10% |
| 减值/净利润 | > 50% | < 10% |

## 商誉减值 vs 摊销

| 方式 | 说明 | 会计准则 |
|------|------|----------|
| **减值测试** | 每年测试，有减值才计提 | 美国GAAP |
| **摊销** | 按年分摊 | 部分国家准则 |

## 典型案例

### 案例：AOL时代华纳

- 2000年AOL收购时代华纳
- 商誉高达1500亿美元
- 2002年减值990亿美元
- 史上最大商誉减值

### 案例：科技公司收购

某科技公司：
- 高溢价收购AI公司
- 商誉50亿，净资产50亿
- 被收购公司业绩不达标
- 3年后减值30亿

## 分析框架

```
1. 商誉规模
   ├── 占净资产比例
   ├── 占总资产比例
   └── 与同行对比

2. 收购历史
   ├── 收购频率
   ├── 收购溢价
   └── 整合效果

3. 被收购公司状况
   ├── 业绩表现
   ├── 管理层稳定性
   └── 行业环境

4. 减值风险评估
   └── 商誉减值测试方法
```
