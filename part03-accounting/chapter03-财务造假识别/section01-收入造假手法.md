# 收入造假手法

## 常见手法

### 1. 提前确认收入

在商品未交付或服务未完成时确认收入。

| 手法 | 说明 |
|------|------|
| 发货即确认 | 不管客户是否签收 |
| 跨期确认 | 把下期收入挪到本期 |
| 附条件销售 | 有退货权但全额确认 |

### 2. 虚构收入

完全不存在的交易。

| 手法 | 说明 |
|------|------|
| 虚假客户 | 编造不存在的客户 |
| 关联交易 | 与关联方虚假交易 |
| 循环交易 | A卖给B，B卖回A |

### 3. 渠道填塞

向经销商大量发货，确认收入，但经销商并未卖给终端消费者。

| 特征 | 说明 |
|------|------|
| 季末收入暴增 | 为完成目标突击发货 |
| 渠道库存高 | 经销商货卖不出去 |
| 后续退货多 | 产品回流 |

### 4. 提前开票

在服务未完成时就开票确认收入。

| 例子 | 说明 |
|------|------|
| 年度服务 | 一次性确认全年收入 |
| 长期合同 | 过早确认收入 |

## 识别方法

### 定量信号

| 信号 | 可能含义 |
|------|----------|
| 应收增速 >> 收入增速 | 可能在虚增收入 |
| 季末收入占比异常高 | 可能在渠道填塞 |
| 收入增速远超同行 | 需要验证 |
| 退货率上升 | 之前确认的收入有问题 |
| 经营现金流 << 净利润 | 利润质量差 |

### 定性信号

| 信号 | 可能含义 |
|------|----------|
| 频繁更换审计师 | 可能有分歧 |
| 关联交易复杂 | 可能有利益输送 |
| 管理层频繁变动 | 可能有内部问题 |
| 业绩"踩线"达标 | 可能在操纵 |

## 典型案例

### Enron（安然）

- 通过特殊目的实体（SPE）隐藏债务
- 虚构收入和利润
- 最终破产

### WorldCom（世通）

- 将运营费用资本化
- 虚增利润数十亿美元
- 最终破产

### Luckin Coffee（瑞幸咖啡）

- 虚构交易额
- 关联方虚假采购
- 被做空后承认造假

## 防范框架

```
1. 收入质量检查
   ├── 应收增速 vs 收入增速
   ├── 经营现金流 vs 净利润
   └── 季度收入分布

2. 关联交易检查
   ├── 关联交易金额占比
   └── 关联交易定价是否合理

3. 同行对比
   ├── 收入增速是否异常
   └── 毛利率是否异常

4. 外部验证
   ├── 行业数据
   ├── 终端数据
   └── 做空报告
```

## 思考题

1. 为什么说"应收账款增速超过收入增速"是危险信号？
2. 如何区分"渠道填塞"和"正常的渠道扩张"？
3. 做空报告一定可信吗？如何验证？
