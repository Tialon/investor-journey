# 现金流与利润背离

## 核心逻辑

$$\text{净利润} \approx \text{经营现金流} + \text{营运资本变动} + \text{非现金费用}$$

长期来看，**净利润应该接近经营现金流**。如果两者持续背离，可能存在问题。

## 背离类型

### 类型一：净利润高，CFO低

| 原因 | 说明 |
|------|------|
| 应收账款暴增 | 卖了货没收到钱 |
| 存货暴增 | 买了材料没卖出去 |
| 预付账款暴增 | 提前付了钱 |
| 收入虚增 | 虚构收入没有现金流 |

**危险程度**：高

### 类型二：CFO高，净利润低

| 原因 | 说明 |
|------|------|
| 大额折旧 | 非现金费用减少利润 |
| 预收账款增加 | 收了钱没确认收入 |
| 应付账款增加 | 欠着供应商的钱 |

**危险程度**：低（通常是正常现象）

## 关键比率

$$\text{CFO / 净利润} = \text{利润质量比率}$$

| 比率 | 评价 |
|------|------|
| > 1 | 优秀 |
| 0.8-1 | 良好 |
| 0.5-0.8 | 需关注 |
| < 0.5 | 危险 |
| < 0 | 极度危险 |

## 识别方法

### 步骤一：计算比率

```
净利润：100亿
CFO：40亿
比率：0.4（危险）
```

### 步骤二：找原因

```
应收账款增加：-40亿
存货增加：-20亿
折旧：+20亿
应付增加：+10亿
预收增加：+10亿
其他：-10亿
→ CFO = 100 - 40 - 20 + 20 + 10 + 10 - 10 = 70亿
→ 实际约40亿（简化）
```

### 步骤三：判断性质

| 原因 | 性质 |
|------|------|
| 应收暴增 | 可能虚增收入 |
| 存货暴增 | 可能产品滞销 |
| 折旧增加 | 正常 |
| 预收增加 | 好信号 |

## 典型案例

### 案例：利润虚增

某公司连续3年：
- 净利润：50亿、60亿、70亿
- CFO：10亿、5亿、-10亿
- 应收账款：100亿、180亿、280亿

**真相**：收入可能虚增，应收账款堆积。

### 案例：正常背离

某公司：
- 净利润：30亿
- CFO：60亿
- 折旧：25亿
- 预收增加：10亿

**解读**：非现金费用+预收增加，正常现象。

## 分析框架

```
1. CFO vs 净利润比率
   └── < 0.8 → 需要分析

2. 差异原因
   ├── 应收变动
   ├── 存货变动
   ├── 应付变动
   ├── 预收变动
   └── 非现金费用

3. 趋势分析
   └── 背离是否在扩大

4. 同行对比
   └── 比率是否低于同行

5. 判断性质
   ├── 正常背离
   ├── 经营问题
   └── 可能造假
```
