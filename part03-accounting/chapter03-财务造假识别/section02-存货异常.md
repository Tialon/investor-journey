# 存货异常

## 存货的构成

| 项目 | 说明 |
|------|------|
| 原材料 | 未加工的材料 |
| 在产品 | 正在生产中 |
| 产成品 | 已完成待销售 |

## 异常信号

### 1. 存货增速远超收入增速

| 情况 | 可能含义 |
|------|----------|
| 存货+30%，收入+5% | 产品滞销 |
| 存货+50%，收入+10% | 可能在囤货或造假 |

### 2. 存货周转率下降

$$\text{存货周转率} = \frac{\text{营业成本}}{\text{平均存货}}$$

$$\text{存货周转天数} = \frac{365}{\text{存货周转率}}$$

| 趋势 | 含义 |
|------|------|
| 周转率上升 | 存货管理改善 |
| 周转率下降 | 产品可能滞销 |

### 3. 存货减值准备不足

| 信号 | 可能含义 |
|------|----------|
| 减值准备比例低于同行 | 可能高估存货价值 |
| 存货老化但不减值 | 可能在掩盖问题 |

### 4. 存货结构异常

| 信号 | 可能含义 |
|------|----------|
| 产成品占比异常高 | 产品卖不出去 |
| 原材料异常增加 | 可能虚增资产 |

## 存货造假手法

| 手法 | 说明 |
|------|------|
| 虚增存货数量 | 账面数量多于实际 |
| 高估存货价值 | 计价高于实际价值 |
| 少计存货减值 | 不计提应有的减值 |
| 虚假采购 | 虚构采购增加存货 |

## 识别方法

| 方法 | 操作 |
|------|------|
| **趋势分析** | 存货增速 vs 收入增速 |
| **同行对比** | 存货周转率 vs 行业平均 |
| **结构分析** | 原材料/在产品/产成品占比 |
| **减值政策** | 与同行对比减值比例 |
| **现金流验证** | 采购现金流 vs 账面存货增加 |

## 典型案例

### 案例：獐子岛

- 声称"扇贝跑了"
- 存货异常消失
- 实际是存货造假

### 案例：存货积压

某电子产品公司：
- 存货从50亿增长到120亿
- 收入只增长10%
- 后来计提大额减值

## 分析框架

```
1. 存货增速 vs 收入增速
   └── 差距大 → 警惕

2. 存货周转率趋势
   └── 下降 → 产品可能滞销

3. 存货结构
   └── 产成品占比高 → 卖不出去

4. 减值政策
   └── 低于同行 → 可能高估

5. 现金流验证
   └── 采购现金流 vs 存货增加
```
