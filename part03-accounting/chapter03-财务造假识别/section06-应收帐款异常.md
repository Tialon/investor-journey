# 应收账款异常

## 应收账款的本质

应收账款是公司**已确认收入但未收到现金**的部分。它是连接利润表和现金流量表的关键桥梁。

## 异常信号

### 1. 应收增速远超收入增速

| 情况 | 可能含义 |
|------|----------|
| 应收+30%，收入+5% | 可能在虚增收入 |
| 应收+50%，收入+10% | 可能在渠道填塞 |
| 应收持续高增长 | 可能信用政策放宽 |

### 2. 应收账款周转率下降

$$\text{应收周转率} = \frac{\text{营业收入}}{\text{平均应收账款}}$$

$$\text{应收周转天数} = \frac{365}{\text{应收周转率}}$$

| 趋势 | 含义 |
|------|------|
| 周转率上升 | 回款加快 |
| 周转率下降 | 回款变慢，可能有问题 |

### 3. 账龄结构恶化

| 账龄 | 风险 |
|------|------|
| 0-90天 | 低 |
| 90-180天 | 中 |
| 180-365天 | 高 |
| > 1年 | 极高 |

**信号**：长账龄占比上升。

### 4. 坏账准备不足

| 信号 | 说明 |
|------|------|
| 坏账比例低于同行 | 可能高估应收 |
| 坏账政策突然放宽 | 可能在掩盖问题 |

## 应收账款造假手法

| 手法 | 说明 |
|------|------|
| 虚假销售 | 虚构客户确认收入 |
| 提前开票 | 未满足条件就开票 |
| 关联方应收 | 向关联方虚假销售 |
| 出售应收 | 将应收卖给金融机构 |

## 识别方法

### 定量检查

| 检查项 | 方法 |
|--------|------|
| 应收增速 vs 收入增速 | 差距大需警惕 |
| 应收周转天数趋势 | 上升需警惕 |
| 账龄结构 | 长账龄占比上升需警惕 |
| 坏账比例 vs 同行 | 低于同行需警惕 |
| 应收/收入比例 | 上升需警惕 |

### 定性检查

| 检查项 | 方法 |
|--------|------|
| 客户集中度 | 是否过度依赖少数客户 |
| 信用政策 | 是否突然放宽 |
| 关联方应收 | 是否有大额关联方应收 |
| 行业对比 | 是否与行业趋势一致 |

## 典型案例

### 案例：虚增收入

某公司：
- 收入增长20%
- 应收账款增长80%
- 坏账准备比例不变
- 经营现金流为负

**真相**：可能通过虚假销售虚增收入。

### 案例：渠道填塞

某公司：
- Q4收入占全年45%
- Q4应收大幅增加
- 次年Q1收入下降30%

**真相**：季末突击发货，渠道库存积压。

## 分析框架

```
1. 应收增速 vs 收入增速
   └── 差距大 → 警惕

2. 应收周转天数趋势
   └── 上升 → 回款变慢

3. 账龄结构
   └── 长账龄占比上升 → 坏账风险

4. 坏账准备
   └── 低于同行 → 可能高估

5. 现金流验证
   └── 应收增加 → CFO减少
```
