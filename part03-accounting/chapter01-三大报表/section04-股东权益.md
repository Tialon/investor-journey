# 股东权益

## 定义

> **股东权益（Shareholders' Equity）**是公司总资产减去总负债后的余额，代表股东对公司净资产的所有权。

$$\text{股东权益} = \text{总资产} - \text{总负债}$$

## 组成部分

| 项目 | 定义 | 说明 |
|------|------|------|
| **股本（Paid-in Capital）** | 股东投入的资本 | IPO和增发募集的资金 |
| **资本公积（Capital Surplus）** | 超过面值的投入 | 股票发行溢价 |
| **留存收益（Retained Earnings）** | 历年累积的未分配利润 | 公司自我积累的核心 |
| **其他综合收益（OCI）** | 未实现的损益 | 外币折算、金融资产公允价值变动 |
| **库存股（Treasury Stock）** | 回购但未注销的股票 | 减少股东权益 |

## 留存收益的重要性

$$\text{期末留存收益} = \text{期初留存收益} + \text{净利润} - \text{分红}$$

留存收益是公司**自我积累**的核心来源：

| 情况 | 含义 |
|------|------|
| 留存收益持续增长 | 公司在积累财富 |
| 留存收益不增长 | 赚的钱全部分红或亏损 |
| 留存收益为负 | 累积亏损超过累积盈利 |

## 股东权益变动

```
股东权益变动
├── (+) 净利润
├── (+) 增发股票
├── (-) 分红
├── (-) 回购股票
├── (+/-) 其他综合收益
└── (+/-) 股权激励
```

## 关键指标

### 每股净资产（Book Value Per Share）

$$\text{BVPS} = \frac{\text{股东权益}}{\text{总股本}}$$

### 市净率（P/B Ratio）

$$\text{P/B} = \frac{\text{股价}}{\text{每股净资产}}$$

| P/B | 含义 |
|-----|------|
| < 1 | 股价低于净资产（可能被低估或有问题） |
| 1-3 | 合理 |
| > 3 | 市场给予高溢价 |

### ROE

$$\text{ROE} = \frac{\text{净利润}}{\text{股东权益}}$$

## 库存股的影响

公司回购股票但不注销：

| 效果 | 说明 |
|------|------|
| 减少总股本 | 提高EPS |
| 减少股东权益 | 提高ROE |
| 不影响资产 | 只是权益内部调整 |

## 股东权益的质量

| 信号 | 好 | 坏 |
|------|------|------|
| 留存收益占比 | 高（自我积累） | 低（依赖融资） |
| 其他综合收益 | 小 | 大（波动风险） |
| 商誉占比 | 低 | 高（减值风险） |
| 库存股 | 适度回购 | 大量回购但业绩不增长 |

## 思考题

1. 为什么说"留存收益是公司最重要的资金来源"？
2. 一家公司P/B < 1，一定是被低估吗？
3. 回购股票和分红，对股东权益的影响有什么不同？
