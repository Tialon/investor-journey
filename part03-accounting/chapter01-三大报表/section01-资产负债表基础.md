# 资产负债表基础

## 核心公式

$$\text{资产} = \text{负债} + \text{股东权益}$$

这个等式永远平衡，因此叫 Balance Sheet。

## 三大部分

| 部分 | 定义 | 例子 |
|------|------|------|
| **资产** | 公司拥有的经济资源 | 现金、设备、专利 |
| **负债** | 公司欠他人的债务 | 贷款、应付账款 |
| **股东权益** | 资产减负债后的净值 | 股本、留存收益 |

## 报表结构

```
资产                              负债+股东权益
┌──────────────────┐             ┌──────────────────┐
│ 流动资产          │             │ 流动负债          │
│   现金            │             │   应付账款        │
│   应收账款        │             │   短期借款        │
│   存货            │             ├──────────────────┤
├──────────────────┤             │ 非流动负债        │
│ 非流动资产        │             │   长期借款        │
│   固定资产        │             │   应付债券        │
│   无形资产        │             ├──────────────────┤
│   商誉            │             │ 股东权益          │
│                   │             │   股本            │
│                   │             │   留存收益        │
└──────────────────┘             └──────────────────┘
    总资产                            总计
```

## 关键概念

### 时点报表

资产负债表反映的是**某一特定日期**的财务状况（如12月31日），不是一段时期。

### 历史成本 vs 公允价值

| 计量方式 | 说明 | 例子 |
|----------|------|------|
| 历史成本 | 按购买时的价格 | 固定资产（原值） |
| 公允价值 | 按当前市场价格 | 金融资产 |

### 会计恒等式的含义

- **资产** = 公司用这些资源赚钱
- **负债** = 债权人的索取权
- **股东权益** = 股东的索取权

## 阅读要点

1. **现金多不多？** — 流动性
2. **负债高不高？** — 杠杆风险
3. **商誉大不大？** — 收购风险
4. **留存收益增长了吗？** — 盈利积累

## 思考题

1. 为什么资产负债表必须"平衡"？
2. 一家公司资产100亿、负债90亿，净资产只有10亿，这健康吗？
3. 历史成本和公允价值哪个更能反映公司真实价值？
