# 营业利润 vs 净利润

## 定义

| 指标 | 公式 | 含义 |
|------|------|------|
| **营业利润** | 毛利 - 运营费用 | 核心业务的盈利能力 |
| **净利润** | 营业利润 ± 利息/其他 - 税 | 最终归属股东的利润 |

## 两者之间的差异

```
营业利润（EBIT）
  ± 利息收入/支出
  ± 投资收益/损失
  ± 资产处置收益/损失
  ± 其他非经营损益
━━━━━━━━━━━━━━━━━━━
= 税前利润（EBT）
  - 所得税
━━━━━━━━━━━━━━━━━━━
= 净利润
```

## 关注重点

### 营业利润更重要

| 维度 | 营业利润 | 净利润 |
|------|----------|--------|
| 可预测性 | 高 | 低（受非经营项目影响） |
| 可持续性 | 高 | 可能被一次性项目扭曲 |
| 可比性 | 高（跨公司可比） | 低（不同资本结构） |
| 分析价值 | 反映核心业务 | 反映最终结果 |

### 两者差距大时需要关注

| 差距原因 | 说明 |
|----------|------|
| 高利息支出 | 杠杆过高 |
| 大额投资收益 | 非经营性，不可持续 |
| 资产处置收益 | 一次性 |
| 非经常性损益 | 需要剔除 |

## 实例分析

### 情况一：营业利润高，净利润低

```
营业利润 100亿
  - 利息支出 30亿（高杠杆）
  - 所得税 20亿
= 净利润 50亿
```

**问题**：杠杆过高，侵蚀利润。

### 情况二：营业利润低，净利润高

```
营业利润 30亿
  + 投资收益 50亿（卖了子公司）
  - 所得税 10亿
= 净利润 70亿
```

**问题**：依赖一次性收益，不可持续。

### 情况三：两者接近

```
营业利润 80亿
  ± 利息/其他 -5亿
  - 所得税 20亿
= 净利润 55亿
```

**健康**：核心业务驱动利润。

## 分析框架

```
1. 营业利润是否稳定增长？
   └── 核心业务是否健康

2. 营业利润与净利润差距大吗？
   └── 差距大 → 找原因

3. 非经营损益是什么？
   └── 一次性还是持续性

4. 所得税率是否合理？
   └── 异常低 → 可能有税务优惠（需关注可持续性）
```

## 思考题

1. 为什么分析师更关注营业利润而非净利润？
2. 一家公司净利润增长30%但营业利润只增长5%，可能是什么原因？
3. 如何判断非经营损益是否可持续？
