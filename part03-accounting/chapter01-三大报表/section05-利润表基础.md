# 利润表基础

## 核心等式

$$\text{收入} - \text{费用} = \text{利润}$$

## 报表结构

```
营业收入（Revenue）
  - 营业成本（COGS）
━━━━━━━━━━━━━━━━━━━
= 毛利（Gross Profit）
  - 销售费用（Selling）
  - 管理费用（G&A）
  - 研发费用（R&D）
  - 折旧摊销（D&A）
━━━━━━━━━━━━━━━━━━━
= 营业利润（Operating Income / EBIT）
  ± 利息收支
  ± 其他收入/支出
━━━━━━━━━━━━━━━━━━━
= 税前利润（EBT）
  - 所得税
━━━━━━━━━━━━━━━━━━━
= 净利润（Net Income）
```

## 期间报表

与资产负债表不同，利润表反映的是**一段时期**（如一个季度或一年）的经营成果。

## 权责发生制

利润表使用**权责发生制**：
- 收入在**赚取时**确认，不是收到现金时
- 费用在**发生时**确认，不是支付现金时

这是净利润与现金流不一致的根本原因。

## 关键利润层级

| 层级 | 公式 | 关注点 |
|------|------|--------|
| 毛利 | 收入 - COGS | 产品基础盈利能力 |
| 营业利润 | 毛利 - 运营费用 | 核心业务盈利 |
| 税前利润 | 营业利润 ± 利息/其他 | 考虑财务成本 |
| 净利润 | 税前利润 - 税 | 最终归属股东 |

## 阅读要点

1. **收入增速** — 成长性
2. **毛利率趋势** — 竞争力
3. **费用率变化** — 运营效率
4. **非经常性损益** — 利润质量
5. **EPS vs 净利润** — 股数变化

## 思考题

1. 为什么利润表用权责发生制而不是收付实现制？
2. 毛利率和净利率的差距说明了什么？
3. 如何判断一家公司的利润质量？
