# 经营现金流 vs 净利润

## 核心关系

$$\text{经营现金流} \approx \text{净利润} + \text{非现金费用} - \text{营运资本变动}$$

## 为什么两者不一致？

| 原因 | 对经营现金流的影响 |
|------|-------------------|
| **折旧摊销** | 加回（非现金费用） |
| **应收账款增加** | 减少（卖了货没收到钱） |
| **存货增加** | 减少（买了材料没卖出去） |
| **应付账款增加** | 增加（欠着供应商的钱） |
| **预收账款增加** | 增加（收了钱没确认收入） |

## 关键比率

$$\text{CFO / 净利润} = \text{利润质量比率}$$

| 比率 | 评价 |
|------|------|
| > 1 | 优秀（利润质量高） |
| 0.8-1 | 良好 |
| 0.5-0.8 | 需要关注 |
| < 0.5 | 利润质量差 |
| < 0 | 危险信号 |

## 常见情况分析

### 情况一：CFO > 净利润

```
净利润 50亿
+ 折旧 30亿
- 应收增加 5亿
+ 应付增加 5亿
+ 预收增加 10亿
- 存货增加 5亿
= CFO 85亿
```

**解读**：利润质量高，折旧加回，预收增加（产品抢手）。

### 情况二：CFO < 净利润

```
净利润 80亿
+ 折旧 20亿
- 应收增加 40亿（大量赊销）
- 存货增加 30亿（产品积压）
+ 应付增加 5亿
= CFO 35亿
```

**解读**：利润质量差，应收账款和存货堆积。

### 情况三：CFO为负，净利润为正

```
净利润 10亿
- 应收增加 60亿
- 存货增加 40亿
= CFO -80亿
```

**解读**：纸面盈利但实际在烧钱，极度危险。

## 利润质量分析框架

```
1. CFO vs 净利润比率
   └── > 0.8 为健康

2. 差异的原因是什么？
   ├── 折旧（正常）
   ├── 应收增加（关注）
   ├── 存货增加（关注）
   └── 预收增加（好信号）

3. 趋势如何？
   └── 比率是否在恶化

4. 与同行对比
   └── 比率是否低于同行
```

## 红旗信号

| 信号 | 可能含义 |
|------|----------|
| CFO持续 < 净利润 | 利润质量差 |
| CFO为负但净利润为正 | 可能在操纵利润 |
| 应收增速 >> 收入增速 | 可能虚增收入 |
| 存货增速 >> 收入增速 | 产品可能滞销 |

## 思考题

1. 为什么说"净利润是意见，现金流是事实"？
2. 一家公司CFO持续低于净利润，但收入在增长，可能是什么原因？
3. 如何区分"正常的营运资本变动"和"利润操纵"？
