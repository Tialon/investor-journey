# 非经常性损益

## 定义

> **非经常性损益**是指与公司正常经营业务**无直接关系**，且**不具有持续性**的收入或支出。它们可能大幅扭曲净利润，需要在分析时剔除。

## 常见类型

### 非经常性收入

| 类型 | 说明 | 例子 |
|------|------|------|
| 资产处置收益 | 卖资产赚的钱 | 出售子公司、房产 |
| 投资收益 | 买卖股票/债券赚的钱 | 持股公允价值上升 |
| 政府补助 | 一次性补贴 | 搬迁补偿、税收返还 |
| 诉讼赔偿 | 打官司赢的钱 | 专利侵权赔偿 |
| 债务重组收益 | 少还了钱 | 债务减免 |

### 非经常性支出

| 类型 | 说明 | 例子 |
|------|------|------|
| 资产减值 | 资产价值下降 | 商誉减值、存货减值 |
| 重组费用 | 裁员、关厂 | 人员优化费用 |
| 诉讼赔偿 | 打官司输的钱 | 被判赔偿 |
| 债务重组损失 | 多还了钱 | 提前还款罚金 |
| 汇兑损失 | 汇率波动 | 外币资产贬值 |

## 扣非净利润

$$\text{扣非净利润} = \text{净利润} - \text{非经常性损益}$$

**扣非净利润**才是衡量公司**真实盈利能力**的关键指标。

## 典型案例

### 情况一：靠卖资产"保利润"

```
营业利润：-10亿（亏损）
+ 出售子公司收益：20亿
= 净利润：10亿（盈利）
```

**真相**：核心业务亏损，靠卖资产掩盖。

### 情况二：商誉减值"洗大澡"

```
营业利润：5亿
- 商誉减值：30亿
= 净利润：-25亿
```

**目的**：一次性出清风险，未来轻装上阵。

### 情况三：稳定的非经常性收入

```
每年获得政府补助：2亿
```

**分析**：虽然叫"非经常性"，但如果每年都有的话，需要关注可持续性。

## 如何识别非经常性损益

| 方法 | 说明 |
|------|------|
| **看财报附注** | 非经常性项目会在附注中详细说明 |
| **对比扣非净利润** | 与净利润的差异 |
| **看利润表结构** | 大额"其他收入/支出" |
| **看现金流** | 非现金的非经常性项目 |

## 非经常性损益的陷阱

| 陷阱 | 说明 |
|------|------|
| **年年"非经常"** | 每年都有类似项目，实际是经常性的 |
| **金额巨大** | 扭曲真实盈利 |
| **管理层解释** | "下不为例"但年年有 |
| **隐藏在"其他"** | 不单独披露 |

## 分析框架

```
1. 今年有多少非经常性损益？
   └── 财报附注

2. 扣非净利润是多少？
   └── 真实盈利能力

3. 去年也有类似项目吗？
   └── 是否真的"非经常"

4. 非经常性损益的性质是什么？
   └── 一次性 vs 可能重复
```

## 思考题

1. 为什么说"扣非净利润"比净利润更重要？
2. 一家公司年年都有大额"非经常性"损益，这意味着什么？
3. 商誉减值是"洗大澡"还是"真实损失"？
