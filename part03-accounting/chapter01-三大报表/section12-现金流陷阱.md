# 现金流陷阱

## 定义

> **现金流陷阱**是指公司现金流表面上看起来健康，但实际上存在隐藏风险的情况。

## 常见陷阱

### 1. 延迟付款制造的现金流

公司通过延迟支付供应商货款来美化经营现金流。

```
正常的CFO：100亿
延迟付款：+50亿（少付供应商）
表面CFO：150亿
```

**识别方法**：应付账款增速远超收入增速。

### 2. 出售应收账款（Factoring）

公司将应收账款卖给金融机构，提前获得现金。

```
正常的CFO：100亿
出售应收：+30亿
表面CFO：130亿
```

**识别方法**：应收账款异常下降，但收入未降。

### 3. 融资租赁资本化

将经营租赁转为融资租赁，租金支出变成CapEx（不影响CFO）。

**识别方法**：比较经营租赁和融资租赁的比例。

### 4. 收购带来的现金流

收购一家公司，获得其现金和业务，CFO增加。

```
自身CFO：80亿
被收购公司CFO：30亿
合并CFO：110亿
```

**识别方法**：分析有机增长 vs 并购增长。

### 5. 资本支出不足

公司削减维护性CapEx来提高FCF，但长期会损害竞争力。

```
正常FCF：50亿
削减CapEx：+20亿
表面FCF：70亿
```

**识别方法**：CapEx < 折旧（维护性CapEx的下限）。

### 6. 股权激励的现金效应

股权激励是非现金费用，在间接法下会加回CFO。

```
净利润：50亿
+ 股权激励：20亿（加回）
= CFO：70亿
```

**识别方法**：股权激励占CFO的比例。

## 现金流质量检查清单

| 检查项 | 方法 |
|--------|------|
| 应付账款是否异常增加？ | 对比收入增速 |
| 应收账款是否异常减少？ | 检查是否有Factoring |
| CapEx是否低于折旧？ | 可能在削减投资 |
| 股权激励是否过高？ | 评估真实CFO |
| 是否有大额收购？ | 分析有机CFO |
| 经营租赁是否转融资租赁？ | 检查租赁政策 |

## 典型案例

### 案例：延迟付款

某公司连续3年CFO增长：
- Year 1：100亿
- Year 2：130亿（应付+20亿）
- Year 3：160亿（应付+30亿）

**真相**：CFO增长主要靠延迟付款，不可持续。

### 案例：削减CapEx

某公司FCF从50亿增长到80亿：
- CapEx从60亿降到30亿
- 但折旧为50亿

**真相**：CapEx不足折旧，长期损害竞争力。

## 分析框架

```
1. CFO增长的驱动因素是什么？
   ├── 利润增长（好）
   ├── 营运资本变动（需分析）
   └── 非现金费用加回（需分析）

2. 是否有美化现金流的行为？
   ├── 延迟付款
   ├── 出售应收
   ├── 削减CapEx
   └── 股权激励

3. FCF是否可持续？
   └── CapEx vs 折旧
```

## 思考题

1. 为什么说"CFO增长"不一定等于"经营改善"？
2. 如何区分"正常的营运资本优化"和"现金流美化"？
3. 一家公司FCF很高但CapEx低于折旧，你会投资吗？
