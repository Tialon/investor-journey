# 现金流量表基础

## 定义

> **现金流量表**反映公司在**一段时期**内现金的流入和流出情况。它揭示公司实际收到了多少现金、花出了多少现金。

## 三大活动分类

### 1. 经营活动现金流（CFO）

公司**主营业务**产生的现金。

| 流入 | 流出 |
|------|------|
| 收到客户货款 | 支付供应商 |
| 收到利息/股息 | 支付工资 |
| | 支付税费 |

**核心问题：主业能养活自己吗？**

### 2. 投资活动现金流（CFI）

公司**投资和资产处置**产生的现金。

| 流入 | 流出 |
|------|------|
| 出售资产 | 购买固定资产（CapEx） |
| 出售投资 | 收购公司 |

**核心问题：公司在扩张还是收缩？**

### 3. 筹资活动现金流（CFF）

公司**与股东和债权人**之间的现金流。

| 流入 | 流出 |
|------|------|
| 发行股票 | 回购股票 |
| 借款 | 偿还债务 |
| | 支付股息 |

**核心问题：公司在借钱还是还钱？**

## 两种编制方法

### 直接法

直接列示各项现金收支。

| 项目 | 金额 |
|------|------|
| 收到客户货款 | +1000 |
| 支付供应商 | -600 |
| 支付工资 | -200 |
| 支付税费 | -100 |
| **经营活动现金流** | **100** |

### 间接法

从净利润出发，调整为现金流。

```
净利润 80
+ 折旧摊销 20（非现金费用）
- 应收账款增加 10（未收到现金）
+ 应付账款增加 5（未支付现金）
+ 存货减少 5（卖掉了）
━━━━━━━━━━━━━━━━
= 经营活动现金流 100
```

## 现金流组合分析

| 经营 | 投资 | 筹资 | 典型阶段 |
|------|------|------|----------|
| + | - | + | 成长期 |
| + | - | - | 成熟期（最佳） |
| + | + | - | 收缩期 |
| - | - | + | 初创期 |
| - | + | + | 危险 |
| - | - | - | 衰退期 |

## 阅读要点

1. **经营现金流是否为正？** — 主业造血能力
2. **经营现金流 vs 净利润** — 利润质量
3. **自由现金流** — 可支配现金
4. **现金流组合** — 公司处于什么阶段

## 思考题

1. 为什么间接法更常用？
2. 经营现金流为正但净利润为负，可能是什么原因？
3. 一家公司经营现金流持续为负但股价涨，可能是什么原因？
