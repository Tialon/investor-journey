# 资产负债率

## 公式

$$\text{资产负债率} = \frac{\text{总负债}}{\text{总资产}} \times 100\%$$

## 含义

衡量公司**用借来的钱**占总资产的比例。

## 水平参考

| 水平 | 评价 | 典型行业 |
|------|------|----------|
| < 30% | 保守 | 科技、消费品 |
| 30-60% | 适中 | 多数行业 |
| 60-80% | 激进 | 地产、公用事业 |
| > 80% | 高风险 | 银行（正常） |

## 相关指标

### 有息负债率

$$\text{有息负债率} = \frac{\text{有息负债}}{\text{总资产}} \times 100\%$$

有息负债 = 短期借款 + 长期借款 + 应付债券

**比资产负债率更准确**，因为排除了无息的经营负债（应付账款、预收款）。

### 净负债率

$$\text{净负债率} = \frac{\text{有息负债} - \text{现金}}{\text{股东权益}} \times 100\%$$

| 水平 | 评价 |
|------|------|
| < 0 | 现金多于负债（非常安全） |
| 0-50% | 适中 |
| > 100% | 激进 |

### 权益乘数

$$\text{权益乘数} = \frac{\text{总资产}}{\text{股东权益}} = \frac{1}{1 - \text{资产负债率}}$$

## 好负债 vs 坏负债

| 类型 | 特征 | 例子 |
|------|------|------|
| **好负债** | 低利率、用于高回报投资 | 低息贷款扩产 |
| **坏负债** | 高利率、用于维持运营 | 高息借款发工资 |
| **经营负债** | 无息、正常经营 | 应付账款、预收款 |

## 不同行业的负债结构

| 行业 | 资产负债率 | 原因 |
|------|-----------|------|
| 银行 | 90%+ | 业务模式依赖杠杆 |
| 地产 | 70-80% | 高杠杆开发 |
| 公用事业 | 60-70% | 稳定现金流支撑 |
| 科技 | 20-40% | 轻资产、现金多 |
| 消费品 | 30-50% | 适度杠杆 |

## 红旗信号

| 信号 | 可能含义 |
|------|----------|
| 资产负债率持续上升 | 杠杆风险增加 |
| 有息负债率远高于同行 | 融资依赖度高 |
| 短期借款占比高 | 还款压力大 |
| 现金少但负债高 | 流动性风险 |

## 思考题

1. 为什么银行的资产负债率可以超过90%？
2. 有息负债率和资产负债率哪个更能反映真实风险？
3. 一家公司资产负债率只有10%，这是好事还是坏事？
