# 利息覆盖倍数

## 公式

$$\text{利息覆盖倍数} = \frac{\text{EBIT}}{\text{利息费用}}$$

EBIT = 营业利润（税息前利润）

## 含义

衡量公司**用经营利润支付利息**的能力。

## 水平参考

| 水平 | 评价 | 说明 |
|------|------|------|
| > 5x | 安全 | 利息负担轻 |
| 3-5x | 适中 | 利息负担可控 |
| 2-3x | 警惕 | 利息负担较重 |
| < 2x | 危险 | 可能无法支付利息 |
| < 1x | 极度危险 | 经营利润不够付利息 |

## 相关指标

### 现金利息覆盖倍数

$$\text{现金利息覆盖倍数} = \frac{\text{CFO} + \text{利息支出} + \text{税款支出}}{\text{利息支出}}$$

用现金流而非利润来衡量，更保守。

### 固定费用覆盖倍数

$$\text{固定费用覆盖倍数} = \frac{\text{EBIT} + \text{租赁费用}}{\text{利息费用} + \text{租赁费用}}$$

考虑了租赁费用，更全面。

## 分析要点

### 趋势分析

| 趋势 | 含义 |
|------|------|
| 覆盖倍数上升 | 偿债能力增强 |
| 覆盖倍数下降 | 偿债能力减弱 |
| 覆盖倍数 < 2x | 需要高度关注 |

### 行业差异

| 行业 | 典型覆盖倍数 | 原因 |
|------|-------------|------|
| 科技 | 10-20x | 低负债、高利润 |
| 消费品 | 5-10x | 稳定现金流 |
| 地产 | 2-4x | 高杠杆 |
| 公用事业 | 3-5x | 稳定但杠杆高 |

## 红旗信号

| 信号 | 可能含义 |
|------|----------|
| 覆盖倍数 < 2x | 可能无法支付利息 |
| 覆盖倍数持续下降 | 偿债能力恶化 |
| 覆盖倍数波动大 | 经营不稳定 |
| 利息费用增速 > 收入增速 | 杠杆成本上升 |

## 案例分析

### 情况一：健康的利息覆盖

```
EBIT：100亿
利息费用：15亿
覆盖倍数：6.7x（安全）
```

### 情况二：危险的利息覆盖

```
EBIT：20亿
利息费用：18亿
覆盖倍数：1.1x（极度危险）
```

**风险**：经营利润几乎全部用于支付利息。

2. 一家公司利息覆盖倍数从5x降到2x，可能发生了什么？
3. 如何用利息覆盖倍数判断一家公司是否应该减少负债？
