# 毛利率

## 公式

$$\text{毛利率} = \frac{\text{营业收入} - \text{营业成本}}{\text{营业收入}} \times 100\%$$

## 含义

衡量**产品或服务本身**的盈利能力，不考虑运营费用。

## 行业参考

| 行业 | 毛利率 | 特点 |
|------|--------|------|
| 软件/SaaS | 70-90% | 边际成本极低 |
| 奢侈品 | 60-70% | 品牌溢价 |
| 医药 | 60-80% | 研发驱动 |
| 消费品 | 40-50% | 品牌+规模 |
| 零售 | 25-35% | 薄利多销 |
| 制造 | 20-30% | 成本竞争 |

## 毛利率变化的原因

| 原因 | 影响 |
|------|------|
| 原材料涨价 | 毛利率下降 |
| 产品提价 | 毛利率上升 |
| 竞争加剧 | 毛利率下降 |
| 产品结构变化 | 取决于高毛利产品占比 |
| 规模效应 | 毛利率上升 |

## 分析要点

1. **趋势** — 3-5年毛利率变化
2. **同行对比** — 是否高于行业平均
3. **结构** — 不同产品/地区的毛利率差异
4. **可持续性** — 毛利率上升是否可持续

## 思考题

1. 为什么软件公司的毛利率远高于制造公司？
2. 毛利率从50%下降到45%，可能是什么原因？
3. 一家公司毛利率80%但净利率5%，钱花到哪去了？
