# 净利率

## 公式

$$\text{净利率} = \frac{\text{净利润}}{\text{营业收入}} \times 100\%$$

## 含义

衡量公司**最终**每1元收入能赚多少利润。

## 行业参考

| 行业 | 净利率 | 特点 |
|------|--------|------|
| 软件/SaaS | 20-35% | 高毛利+低运营费用 |
| 奢侈品 | 15-25% | 品牌溢价 |
| 消费品 | 10-15% | 规模效应 |
| 零售 | 3-5% | 薄利多销 |
| 制造 | 5-10% | 成本压力 |
| 银行 | 25-30% | 高杠杆 |

## 毛利率 vs 净利率的差距

$$\text{差距} = \text{毛利率} - \text{净利率}$$

| 差距 | 含义 |
|------|------|
| 小（<20%） | 运营效率高 |
| 大（>30%） | 运营费用高或非经营费用多 |

## 净利率的陷阱

| 陷阱 | 说明 |
|------|------|
| 削减研发 | 短期提高净利率，长期损害竞争力 |
| 一次性收益 | 净利率虚高 |
| 高杠杆 | 风险高 |
| 低税率 | 税务优惠可能不可持续 |

## 分析框架

```
1. 净利率趋势如何？
   └── 稳定或上升为好

2. 与同行对比如何？
   └── 高于同行说明有优势

3. 毛利率与净利率差距大吗？
   └── 找出差距来源

4. 是否有非经常性项目？
   └── 扣非净利率更可靠
```

## 思考题

1. Costco净利率只有2.5%，为什么还能投资？
2. 净利率持续下降但收入增长，可能是什么原因？
3. 如何判断一家公司的净利率是否可持续？
