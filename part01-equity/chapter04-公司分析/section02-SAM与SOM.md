# SAM与SOM

## 定义

### SAM（Serviceable Addressable Market，可服务市场）

> 公司的产品或服务**实际能够触达**的市场规模。它是TAM的一个子集，考虑了地理、产品类型、客户群体等限制。

### SOM（Serviceable Obtainable Market，可获得市场）

> 公司在现有资源和能力下，**实际能够获得**的市场份额。它是SAM的一个子集，考虑了竞争、执行能力等因素。

## TAM vs SAM vs SOM

| 层级      | 问题            | 例子（Netflix）         |
| ------- | ------------- | ------------------- |
| **TAM** | 全球娱乐市场有多大？    | ~$2.5万亿（含电影、电视、游戏等） |
| **SAM** | 流媒体市场有多大？     | ~$5000亿             |
| **SOM** | Netflix能获得多少？ | ~$350亿（当前收入）        |

## 如何估算SAM

### 步骤

1. **从TAM出发**
2. **按地理范围缩小** — 公司实际运营的市场
3. **按产品类型缩小** — 公司实际提供的产品
4. **按客户群体缩小** — 公司实际服务的客户

### 例子：Tesla的SAM

```
TAM：全球汽车市场 ~$3万亿
  → 排除商用车          ~$2.5万亿
    → 排除两轮/三轮车    ~$2.3万亿
      → 排除二手车       ~$1.5万亿（新车市场）
        → 聚焦电动车     ~$5000亿（SAM）
```

## 如何估算SOM

### 步骤

1. **从SAM出发**
2. **考虑市场份额** — 当前和可达到的
3. **考虑竞争格局** — 有多少竞争对手
4. **考虑执行能力** — 公司的资源和能力

### 例子：Tesla的SOM

```
SAM：全球电动车市场 ~$5000亿
  → 当前市场份额 ~15%
  → SOM ≈ $750亿（当前）
  → 未来SOM可能增长到 $1000亿+
```

## SOM的关键驱动因素

| 因素 | 说明 |
|------|------|
| **竞争格局** | 竞争对手越少，SOM越大 |
| **产品竞争力** | 产品越好，市场份额越高 |
| **品牌力** | 品牌越强，获客越容易 |
| **渠道能力** | 渠道越广，覆盖越广 |
| **资本实力** | 资本越充足，扩张越快 |

## SOM的增长路径

```
SOM增长
├── 市场份额提升（抢竞争对手的份额）
├── SAM扩大（进入新市场/新品类）
├── TAM扩大（市场整体增长）
└── 价格提升（高端化）
```

## 典型案例

### Amazon的TAM/SAM/SOM演变

| 时期 | TAM | SAM | SOM |
|------|-----|-----|-----|
| 1990s | 全球零售 | 美国图书市场 | 在线图书 |
| 2000s | 全球零售 | 美国电商 | 综合电商 |
| 2010s | 全球零售+云 | 全球电商+云 | 电商+AWS |
| 2020s | 全球零售+云+AI | 全球数字经济 | 多元化平台 |

**关键洞察**：Amazon的SAM不断扩大

### NVIDIA的TAM/SAM/SOM

| 时期 | TAM | SAM | SOM |
|------|-----|-----|-----|
| 2010s | 半导体 | GPU市场 | 游戏GPU |
| 2020s | 半导体 | AI芯片 | AI训练芯片 |

**关键洞察**：AI浪潮大幅扩大了NVIDIA的SAM和SOM

## TAM/SAM/SOM的局限性

| 局限性 | 说明 |
|--------|------|
| **估算不确定性** | 不同方法得出不同结果 |
| **动态变化** | TAM/SAM/SOM会随时间变化 |
| **竞争变化** | 新竞争者可能改变SOM |
| **技术变革** | 新技术可能重新定义TAM |

## 分析框架

```
1. TAM有多大？在增长吗？
   └── 驱动力是什么？

2. SAM有多合理？
   └── 地理、产品、客户限制

3. SOM有多大？
   └── 当前份额和增长潜力

4. 公司能扩大SOM吗？
   └── 竞争优势、执行能力
```

## 思考题

1. 为什么说"大TAM不等于好投资"？
2. 如何判断一家公司的SAM是在扩大还是在缩小？
3. 一家公司在小TAM中份额很高，和在大TAM中份额很低，哪种情况更有投资价值？
