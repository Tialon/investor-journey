# Common Stocks and Uncommon Profits（怎样选择成长股）

## 基本信息

| 项目       | 内容                      |     |
| -------- | ----------------------- | --- |
| **作者**   | Philip Fisher（菲利普·费雪）   |     |
| **首次出版** | 1958年                   |     |
| **地位**   | 成长股投资的经典之作，巴菲特称费雪对他影响巨大 |     |
| **风格**   | 注重定性分析、深入研究公司质量         |     |

## 核心思想

### 1. 费雪的投资哲学

> **"投资的目标是找到少数几家优秀的公司，然后长期持有。"**

| 维度 | 费雪 | 格雷厄姆 |
|------|------|----------|
| 重点 | 公司质量 | 价格便宜 |
| 方法 | 定性分析 | 定量分析 |
| 持有期 | 非常长期 | 价值回归后卖出 |
| 集中度 | 集中投资 | 适度分散 |

### 2. 15条选股原则

费雪提出了15条选股原则，分为**管理层**、**产品/服务**、**财务**三大类：

#### 管理层原则

| 原则            | 说明          |
| ------------- | ----------- |
| **1. 持续增长潜力** | 公司是否有足够市场空间 |
| **2. 管理层决心**  | 是否有决心开发新产品  |
| **3. 研发投入**   | 研发是否有效      |
| **4. 销售组织**   | 销售能力是否强大    |
| **5. 利润率**    | 利润率是否足够高    |
| **6. 维持利润率**  | 是否在改善利润率    |
| **7. 劳资关系**   | 员工关系是否良好    |
| **8. 管理层深度**  | 是否有优秀的中层管理  |
| **9. 薪酬体系**   | 高管薪酬是否合理    |

#### 产品/服务原则

| 原则 | 说明 |
|------|------|
| **10. 竞争优势** | 是否有显著的竞争优势 |
| **11. 产品前景** | 未来是否有新产品 |
| **12. 规模效应** | 规模是否带来成本优势 |

#### 财务原则

| 原则 | 说明 |
|------|------|
| **13. 融资需求** | 是否需要频繁融资 |
| **14. 信息透明** | 管理层是否坦诚 |
| **15. 会计诚信** | 财务是否诚实 |

### 3. "闲聊法"（Scuttlebutt Method）

费雪独创的研究方法：

> **"通过与公司的客户、供应商、竞争对手、前员工交谈，了解公司的真实情况。"**

```
闲聊法的信息来源：
├── 客户 — 产品好不好用？
├── 供应商 — 付款及时吗？
├── 竞争对手 — 它厉害在哪里？
├── 前员工 — 公司文化怎么样？
├── 行业专家 — 未来趋势如何？
└── 分析师 — 市场怎么看？
```

### 4. 何时卖出

费雪认为只有三种情况应该卖出：

| 情况          | 说明       |
| ----------- | -------- |
| **买入时犯了错误** | 基本面判断失误  |
| **公司基本面恶化** | 竞争优势消失   |
| **发现更好的机会** | 资金需要重新配置 |

**费雪反对因为以下原因卖出**：
- 股价下跌
- 市场整体下跌
- 短期业绩波动

### 5. 不要过度分散

> **"过度分散是无知的避风港。"**

费雪建议：
- 持有5-10只股票
- 每只股票占比10-20%
- 深入研究每只股票

## 费雪 vs 格雷厄姆

| 维度   | 费雪    | 格雷厄姆  |
| ---- | ----- | ----- |
| 重点   | 公司质量  | 价格便宜  |
| 分析   | 定性为主  | 定量为主  |
| 持有期  | 非常长期  | 价值回归  |
| 集中度  | 集中    | 分散    |
| 代表   | 成长股   | 价值股   |
| 理想公司 | 伟大的公司 | 便宜的公司 |

### 巴菲特的融合

巴菲特说：
> **"我是85%的格雷厄姆 + 15%的费雪。"**

后来演变为：
> **"以合理的价格买入伟大的公司，比以便宜的价格买入平庸的公司好。"**

## 名言摘录

> "投资的目标是找到少数几家优秀的公司，然后长期持有。"

> "过度分散是无知的避风港。"

> "不要因为股价下跌就卖出。"

> "最好的投资机会往往出现在大多数人看不到的地方。"

> "了解一家公司的管理层比了解它的产品更重要。"

## 对现代投资者的启示

| 原则 | 现代应用 |
|------|----------|
| 闲聊法 | 社交媒体、论坛、客户评价 |
| 深入研究 | 利用AI和数据分析 |
| 长期持有 | 避免频繁交易 |
| 集中投资 | 精选少数优质公司 |

## 与其他书籍的关系

- **The Intelligent Investor** — 费雪的补充，注重质量而非价格
- **One Up On Wall Street** — 林奇继承了费雪的"从生活中发现牛股"思想
- **Trading in the Zone** — 更注重交易心理

## 思考题

2. 为什么费雪认为只有三种情况应该卖出？
3. 如何平衡"集中投资"和"风险分散"？
