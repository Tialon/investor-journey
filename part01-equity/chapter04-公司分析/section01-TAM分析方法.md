# TAM分析方法

## 定义

> **TAM（Total Addressable Market，总可寻址市场）**是指一个产品或服务在假设100%市场份额的情况下，**理论上能获得的最大市场规模**。

它回答的问题是：**这个市场有多大？**

## TAM的三层结构

```
┌─────────────────────────────────┐
│           TAM（总市场）          │
│  ┌───────────────────────────┐  │
│  │      SAM（可服务市场）     │  │
│  │  ┌─────────────────────┐  │  │
│  │  │   SOM（可获得市场）  │  │  │
│  │  └─────────────────────┘  │  │
│  └───────────────────────────┘  │
└─────────────────────────────────┘
```

| 层级 | 定义 | 例子（电动车） |
|------|------|--------------|
| **TAM** | 整个汽车市场 | 全球汽车年销量 ~8000万辆 |
| **SAM** | 电动车市场 | 全球电动车年销量 ~1500万辆 |
| **SOM** | 特斯拉能获得的市场 | 特斯拉年销量 ~200万辆 |

详见 [SAM与SOM](SAM与SOM.md)

## TAM的计算方法

### 方法一：自上而下（Top-Down）

从宏观数据出发，逐步缩小范围。

```
全球汽车市场 TAM = $3万亿
  → 新能源汽车 SAM = $5000亿
    → 中国市场 SOM = $1500亿
```

**优点**：快速获得大图
**缺点**：可能过于粗略

### 方法二：自下而上（Bottom-Up）

从具体需求出发，逐步扩大。

```
中国年销量 2500万辆
  × 电动车渗透率 30%
  × 平均售价 20万元
  = 1.5万亿元
```

**优点**：更精确
**缺点**：需要更多数据

### 方法三：价值理论（Value Theory）

根据产品创造的价值来估算。

```
如果AI工具能为每个程序员节省10小时/月
  × 全球程序员 3000万人
  × 时薪 $50
  × 12个月
  = TAM $1800亿/年
```

## TAM分析的关键问题

### 1. TAM在增长还是萎缩？

| 趋势 | 含义 | 例子 |
|------|------|------|
| 快速增长 | 大机会 | AI、电动车 |
| 稳定增长 | 成熟市场 | 消费品 |
| 萎缩 | 需要谨慎 | 传统零售 |

### 2. TAM的驱动力是什么？

| 驱动力 | 例子 |
|--------|------|
| 技术变革 | AI创造新市场 |
| 政策推动 | 碳中和推动新能源 |
| 人口变化 | 老龄化推动医疗 |
| 消费升级 | 高端消费增长 |

### 3. 公司能获得多少TAM？

取决于：
- 竞争格局
- 产品竞争力
- 执行能力
- 护城河深度

## 典型案例

### NVIDIA的TAM

| 层级 | 市场 | 规模 |
|------|------|------|
| TAM | 全球半导体市场 | ~$6000亿 |
| SAM | AI芯片市场 | ~$2000亿 |
| SOM | NVIDIA AI芯片 | ~$1000亿+ |

**关键洞察**：AI浪潮扩大了NVIDIA的TAM

### Tesla的TAM

| 层级 | 市场 | 规模 |
|------|------|------|
| TAM | 全球汽车市场 | ~$3万亿 |
| SAM | 电动车市场 | ~$5000亿 |
| SOM | Tesla电动车 | ~$1000亿 |

**关键洞察**：电动车渗透率提升扩大了SAM

## TAM分析的陷阱

| 陷阱 | 说明 |
|------|------|
| **TAM过大** | "这是一个万亿市场"——但你能拿到多少？ |
| **TAM不增长** | 市场规模不变，只能靠抢份额 |
| **TAM被替代** | 新技术可能让旧TAM消失 |
| **TAM定义过窄** | 低估了市场机会 |

## TAM与投资决策

| TAM特征 | 投资含义 |
|---------|----------|
| 大TAM + 高增长 | 最佳机会（如AI） |
| 大TAM + 低增长 | 竞争激烈，需要护城河 |
| 小TAM + 高增长 | 细分龙头机会 |
| 小TAM + 低增长 | 机会有限 |

## 思考题

1. 为什么说"TAM只是一个起点"，还需要关注什么？
2. 如何判断一个TAM是在增长还是在萎缩？
3. 一家公司在小TAM中占70%份额，和在大TAM中占5%份额，哪个更值得投资？
