# One Up On Wall Street（彼得·林奇的成功投资）

## 基本信息

| 项目       | 内容                             |
| -------- | ------------------------------ |
| **作者**   | Peter Lynch（彼得·林奇）             |
| **首次出版** | 1989年                          |
| **地位**   | 最受欢迎的投资书籍之一，林奇是史上最成功的基金经理之一    |
| **业绩**   | 1977-1990年管理富达麦哲伦基金，年化回报率29.2% |

## 核心思想

### 1. 业余投资者的优势

> **"业余投资者有优势，只要你愿意花时间研究。"**

| 业余投资者的优势 | 机构投资者的劣势 |
| -------- | -------- |
| 不受业绩基准约束 | 必须跟踪指数   |
| 可以长期持有   | 季度业绩压力   |
| 可以投资小公司  | 资金规模限制   |
| 没有合规限制   | 严格的合规要求  |

### 2. 从生活中发现牛股

> **"你可以在超市、商场、餐厅发现下一个10倍股。"**

林奇认为，普通投资者在日常生活中比华尔街分析师更早发现好公司。

**例子**：
- Dunkin' Donuts — 林奇的妻子经常去
- The Gap — 林奇的孩子喜欢
- Walmart — 在小镇看到它的扩张

### 3. 六种股票分类

| 类型        | 特征              | 策略     |
| --------- | --------------- | ------ |
| **慢速增长型** | 大型成熟公司，增速<5%    | 关注分红   |
| **稳定增长型** | 大型优质公司，增速10-12% | 核心持仓   |
| **快速增长型** | 小型高增长公司，增速20%+  | 寻找10倍股 |
| **周期型**   | 随经济周期波动         | 把握时机   |
| **困境反转型** | 陷入困境但可能恢复       | 高风险高回报 |
| **隐蔽资产型** | 拥有被低估的资产        | 深入研究   |
|           |                 |        |

### 4. 选股13条原则

1. **名字越无聊，公司越好**（如Automatic Data Processing）
2. **业务越无聊越好**（如瓶盖制造商）
3. **业务令人厌恶更好**（如垃圾处理）
4. **分拆出来的公司值得关注**
5. **机构不持股、分析师不跟踪的公司**
6. **有谣言困扰的公司**
7. **令人沮丧的行业中的好公司**
8. **处于零增长行业的增长公司**
9. **有护城河的公司**
10. **内部人士在买入**
11. **公司在回购股票**
12. **资产负债表良好**
13. **自由现金流充裕**

## 重要概念

### PEG比率

$$\text{PEG} = \frac{\text{PE}}{\text{盈利增长率}}$$

| PEG | 评价 |
|-----|------|
| < 1 | 可能被低估 |
| 1-2 | 合理 |
| > 2 | 可能被高估 |

### 两分钟独白

林奇要求每个基金经理能用两分钟解释清楚：
1. 为什么买这只股票
2. 公司的业务是什么
3. 为什么它是好投资

### 不要买什么

| 避免 | 原因 |
|------|------|
| 热门行业的热门股 | 估值过高 |
| 故事动听但无业绩的公司 | 泡沫 |
| 多元化恶化的公司 | 管理层不专注 |
| 下一个XX公司 | 通常不是 |

## 投资流程

```
1. 从生活中发现机会
   └── 超市、商场、日常体验

2. 基本面研究
   └── 财务报表、业务模式、竞争优势

3. 估值分析
   └── PEG、PE、现金流

4. 买入并持有
   └── 长期持有，不频繁交易

5. 持续跟踪
   └── 季报、年报、行业变化
```

## 名言摘录

> "不研究公司就像打牌不看牌一样。"

> "股票投资最重要的是你能否找到10倍股。"

> "你有足够的时间，不要试图预测市场时机。"

> "如果你花13分钟分析经济和市场趋势，那你就浪费了10分钟。"

## 对现代投资者的启示

| 原则 | 现代应用 |
|------|----------|
| 从生活中发现牛股 | 关注消费趋势、科技趋势 |
| PEG估值 | 成长股估值的重要工具 |
| 不要追热点 | 避免泡沫 |
| 长期持有 | 减少交易成本 |

## 与其他书籍的关系

- **The Intelligent Investor** — 更注重安全边际和价值投资
- **Common Stocks and Uncommon Profits** — 更注重定性分析
- **Trading in the Zone** — 更注重交易心理

3. PEG比率在什么情况下不适用？
