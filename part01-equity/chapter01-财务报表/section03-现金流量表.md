# 现金流量表

## 定义

> **现金流量表（Cash Flow Statement）**是反映公司在**某一期间**现金流入和流出情况的报表。它揭示公司**真实的现金状况**——利润表告诉你赚了多少"纸面利润"，现金流量表告诉你实际收到了多少真金白银。

> **"净利润只是一个意见，现金流才是事实。"**

## 三大活动分类

现金流量表将公司的所有现金流分为三类：

### 1. 经营活动现金流（CFO - Cash Flow from Operations）

公司**主营业务**产生的现金。

| 流入 | 流出 |
|------|------|
| 收到客户的货款 | 支付供应商货款 |
| 收到利息/股息 | 支付员工工资 |
| | 支付税费 |

**核心问题：公司的主业能不能产生现金？**

### 2. 投资活动现金流（CFI - Cash Flow from Investing）

公司**投资和资产处置**产生的现金。

| 流入 | 流出 |
|------|------|
| 出售资产 | 购买固定资产（CapEx） |
| 出售投资 | 收购公司 |
| 收回贷款 | 购买投资 |

**核心问题：公司在扩张还是收缩？**

### 3. 筹资活动现金流（CFF - Cash Flow from Financing）

公司**与股东和债权人**之间的现金流。

| 流入 | 流出 |
|------|------|
| 发行股票 | 回购股票 |
| 借款 | 偿还债务 |
| | 支付股息 |

**核心问题：公司在借钱还是还钱？**

## 经营现金流 vs 净利润

这是分析的关键：

$$\text{经营现金流} vs \text{净利润}$$

| 情况 | 含义 | 信号 |
|------|------|------|
| CFO > 净利润 | 利润质量高，真金白银 | ✅ 健康 |
| CFO ≈ 净利润 | 正常 | ✅ 正常 |
| CFO < 净利润 | 利润质量存疑 | ⚠️ 警惕 |
| CFO < 0 | 经营在烧钱 | ❌ 危险 |

### 为什么两者会不一致？

| 原因 | 说明 |
|------|------|
| 应收账款增加 | 卖了货但没收到钱 |
| 存货增加 | 花钱买了原材料但没卖出去 |
| 折旧摊销 | 利润表扣了费用但没实际花钱 |
| 预收账款增加 | 收了钱但还没确认收入 |

## 自由现金流（FCF）

$$\text{FCF} = \text{经营现金流} - \text{资本支出（CapEx）}$$

这是公司在维持正常运营后，**真正可以自由支配的现金**。是估值（DCF）的核心输入。

## 实例：苹果公司（AAPL）2024年简化现金流表

| 项目 | 金额（亿美元） |
|------|--------------|
| 经营活动现金流 | ~1,100 |
| - 资本支出 | ~110 |
| = **自由现金流** | ~990 |
| 投资活动现金流 | ~-300 |
| 筹资活动现金流 | ~-800 |
| （回购股票+分红） | |

**解读**：
- 经营现金流 > 净利润 → 利润质量高
- 自由现金流充裕 → 有钱回购和分红
- 筹资活动大额流出 → 在回馈股东

## 现金流量表的八种组合模式

| 经营 | 投资 | 筹资 | 典型阶段 |
|------|------|------|----------|
| + | - | + | **成长期**：主业赚钱，大量投资，借债扩张 |
| + | - | - | **成熟期**：主业赚钱，适度投资，回馈股东 |
| + | + | - | **收缩期**：主业赚钱，卖资产，还债 |
| + | + | + | **异常**：赚钱+卖资产+借钱，需警惕 |
| - | - | + | **初创期**：主业烧钱，大量投资，靠融资支撑 |
| - | + | + | **危险**：主业烧钱，卖资产，借债维持 |
| - | - | - | **衰退期**：全面失血 |
| - | + | - | **清算期**：主业烧钱，卖资产，还债 |

**最佳信号**：+ - -（成熟期，现金牛）

## 如何阅读现金流量表

### 关注点

1. **经营现金流是否持续为正？** — 主业能否养活自己
2. **经营现金流 vs 净利润** — 利润质量如何
3. **自由现金流** — 真正可支配的现金
4. **资本支出趋势** — 公司是否在扩张
5. **回购和分红** — 管理层对未来的信心

### 红旗信号

| 信号 | 可能含义 |
|------|----------|
| 连续多年经营现金流为负 | 主业不赚钱，靠融资续命 |
| 净利润高但经营现金流低 | 利润质量差，可能有应收账款堆积 |
| 大额投资活动但收入不增长 | 投资效率低 |
| 频繁融资（发行股票/借款） | 内部造血能力不足 |

## 与其他报表的关系

- **利润表** → 净利润是现金流量表的起点
- **资产负债表** → 现金变动解释了资产负债表中"现金"的期末余额

## 三大报表的联动

```
         利润表
         净利润
           │
           ▼
    ┌──────┴──────┐
    │             │
    ▼             ▼
资产负债表     现金流量表
留存收益      经营/投资/筹资
```

## 思考题

1. 为什么说"现金为王"？一家公司可以没有利润但有现金吗？
2. 亚马逊早期经营现金流为正但净利润为负，这是怎么做到的？
3. 一家公司经营现金流持续为负，但股价一直涨，可能是什么原因？
