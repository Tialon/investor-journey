# 资产负债表

## 定义

> **资产负债表（Balance Sheet）**是反映公司在**某一特定时点**的财务状况的报表，展示公司拥有的资产、欠下的债务以及股东权益。

它是公司的"家底"——告诉你公司**有什么**、**欠什么**、**值多少**。

## 核心公式

$$\text{资产} = \text{负债} + \text{股东权益}$$

这个等式永远成立，所以叫"平衡"表（Balance Sheet）。

## 结构拆解

### 左边（或上方）：资产

公司拥有的资源，按流动性排列：

| 分类 | 包含项目 | 说明 |
|------|----------|------|
| **流动资产** | 现金、应收账款、存货、短期投资 | 一年内可变现 |
| **非流动资产** | 固产、无形资产、商誉、长期投资 | 长期使用 |

### 右边（或下方）：负债 + 股东权益

| 分类 | 包含项目 | 说明 |
|------|----------|------|
| **流动负债** | 应付账款、短期借款、预收款项 | 一年内需偿还 |
| **非流动负债** | 长期借款、债券、递延税款 | 长期偿还 |
| **股东权益** | 股本、资本公积、留存收益 | 属于股东的部分 |

## 关键指标

### 1. 流动比率

$$\text{流动比率} = \frac{\text{流动资产}}{\text{流动负债}}$$

- 一般 **> 1.5** 为健康
- 过高可能说明资产利用效率低

### 2. 资产负债率

$$\text{资产负债率} = \frac{\text{总负债}}{\text{总资产}}$$

- 一般 **< 60%** 为合理（行业差异大）
- 银行业天然高负债率，不能用此标准

### 3. 净资产（Book Value）

$$\text{净资产} = \text{总资产} - \text{总负债} = \text{股东权益}$$

### 4. 有形净资产

$$\text{有形净资产} = \text{股东权益} - \text{商誉} - \text{无形资产}$$

排除了"虚"的资产，更保守。

## 实例：苹果公司（AAPL）2024年简化资产负债表

```
资产                              负债 + 股东权益
┌──────────────────────┐         ┌──────────────────────┐
│ 流动资产              │         │ 流动负债              │
│   现金    $29B        │         │   应付账款  $68B      │
│   应收    $30B        │         │   短期借款  $15B      │
│   存货    $7B         │         │ 其他流动    $58B      │
│ 其他流动  $49B        │         ├──────────────────────┤
├──────────────────────┤         │ 非流动负债            │
│ 非流动资产            │         │   长期借款  $109B     │
│   固定资产 $44B       │         │   其他非流动 $83B     │
│   无形资产 $0         │         ├──────────────────────┤
│   商誉    $0          │         │ 股东权益              │
│   其他    $56B        │         │   留存收益  等        │
│                       │         │   共计      $57B      │
└──────────────────────┘         └──────────────────────┘
         总资产 ≈ $315B                   总计 ≈ $315B
```

## 如何阅读资产负债表

### 关注点

1. **现金多不多？** — 公司抗风险能力
2. **应收账款增速是否超过收入增速？** — 可能在"放水"拉收入
3. **存货是否异常增加？** — 产品可能卖不动
4. **商誉是否过高？** — 收购溢价大，未来可能减值
5. **负债结构如何？** — 短期负债多 → 还款压力大

### 红旗信号

| 信号 | 可能含义 |
|------|----------|
| 应收账款暴增 | 激进确认收入，可能有坏账风险 |
| 存货暴增 | 产品滞销，未来可能减值 |
| 商誉占比过高 | 收购溢价大，商誉减值风险 |
| 短期借款远超现金 | 流动性危机风险 |
| 股东权益持续减少 | 公司在"吃老本" |

## 与其他报表的关系

- **利润表** → 净利润流入资产负债表的"留存收益"
- **现金流量表** → 解释资产负债表中"现金"的变动原因

## 思考题

1. 为什么说资产负债表是一个"时点"概念，而利润表是"期间"概念？
2. 如果一家公司的资产负债率只有10%，这是好事还是坏事？
3. 为什么银行的资产负债率可以高达90%以上而不出问题？
