# 利润表

## 定义

> **利润表（Income Statement）**是反映公司在**某一期间**（季度或年度）经营成果的报表，展示公司赚了多少钱、花了多少钱。

它是公司的"成绩单"——告诉你公司**赚不赚钱**。

## 核心结构

```
营业收入（Revenue）
  - 营业成本（COGS）
━━━━━━━━━━━━━━━━━━━━━
= 毛利（Gross Profit）
  - 销售费用（SG&A）
  - 研发费用（R&D）
  - 折旧摊销（D&A）
━━━━━━━━━━━━━━━━━━━━━
= 营业利润（Operating Income / EBIT）
  ± 利息收支
  ± 其他收入/支出
━━━━━━━━━━━━━━━━━━━━━
= 税前利润（EBT）
  - 所得税
━━━━━━━━━━━━━━━━━━━━━
= 净利润（Net Income）
  - 优先股股利
━━━━━━━━━━━━━━━━━━━━━
= 归属于普通股东的净利润
```

## 关键利润层级

| 利润层级 | 公式 | 含义 |
|----------|------|------|
| **毛利** | 收入 - 营业成本 | 产品的基础盈利能力 |
| **营业利润** | 毛利 - 运营费用 | 核心业务的盈利能力 |
| **税前利润** | 营业利润 ± 利息/其他 | 考虑财务成本后的盈利 |
| **净利润** | 税前利润 - 税 | 最终归属股东的利润 |

## 关键指标

### 1. 毛利率

$$\text{毛利率} = \frac{\text{毛利}}{\text{营业收入}} \times 100\%$$

- **> 50%**：高毛利（软件、奢侈品、医药）
- **30-50%**：中等毛利（消费品、工业）
- **< 30%**：低毛利（零售、制造）

### 2. 营业利润率

$$\text{营业利润率} = \frac{\text{营业利润}}{\text{营业收入}} \times 100\%$$

### 3. 净利率

$$\text{净利率} = \frac{\text{净利润}}{\text{营业收入}} \times 100\%$$

### 4. 每股收益（EPS）

$$\text{EPS} = \frac{\text{归属于普通股东的净利润}}{\text{加权平均股数}}$$

## 实例：苹果公司（AAPL）2024年简化利润表

| 项目 | 金额（亿美元） |
|------|--------------|
| 营业收入 | ~3,900 |
| - 营业成本 | ~2,100 |
| = **毛利** | ~1,800 |
| - 运营费用 | ~570 |
| = **营业利润** | ~1,230 |
| ± 利息/其他 | ~50 |
| = **税前利润** | ~1,280 |
| - 所得税 | ~200 |
| = **净利润** | ~1,080 |
| **毛利率** | **~46%** |
| **净利率** | **~28%** |

## 如何阅读利润表

### 关注点

1. **收入增速** — 公司是否在成长？
2. **毛利率趋势** — 产品竞争力是否在变化？
3. **费用控制** — 运营效率如何？
4. **一次性项目** — 是否有非经常性损益扭曲利润？
5. **EPS vs 净利润** — 股数变化是否稀释了每股收益？

### 红旗信号

| 信号 | 可能含义 |
|------|----------|
| 收入增长但利润不增长 | 可能在"增收不增利"，烧钱换规模 |
| 毛利率持续下降 | 产品竞争力下降或成本上升 |
| 研发费用大幅削减 | 短期美化利润，长期损害竞争力 |
| 频繁出现"其他收入"大额 | 可能在用非经常性项目粉饰利润 |
| 净利润远高于经营现金流 | 利润质量差（详见现金流量表） |

## 利润表的局限性

利润表用的是**权责发生制**（Accrual Basis），意味着：
- 收入确认时不一定收到钱
- 费用确认时不一定付了钱
- 折旧摊销是非现金费用

所以**净利润 ≠ 现金**。要看真实现金流，必须结合**现金流量表**。

## 与其他报表的关系

- **→ 资产负债表**：净利润流入"留存收益"
- **→ 现金流量表**：净利润是现金流量表的起点（间接法）

## 思考题

1. 为什么亚马逊长期净利润很低甚至亏损，但股价一直涨？
2. 一家公司毛利率80%但净利率只有5%，可能发生了什么？
3. 为什么说"净利润只是一个意见，现金流才是事实"？
