# 自由现金流（FCF）

## 定义

> **自由现金流（Free Cash Flow，FCF）**是公司在维持正常运营和必要投资后，**真正可以自由支配的现金**。它是衡量公司"造血能力"的终极指标。

$$\text{FCF} = \text{经营现金流（CFO）} - \text{资本支出（CapEx）}$$

## 两种FCF

| 类型 | 公式 | 用途 |
|------|------|------|
| **FCFF（公司自由现金流）** | NOPAT + D&A - CapEx - ΔNWC | 估值（企业价值） |
| **FCFE（股权自由现金流）** | FCFF - 利息×(1-t) + 净借款 | 估值（股权价值） |

实际分析中，简化版最常用：

$$\text{FCF} \approx \text{CFO} - \text{CapEx}$$

## 为什么FCF比净利润更重要？

| 维度   | 净利润          | FCF      |
| ---- | ------------ | -------- |
| 会计准则 | 权责发生制        | 收付实现制    |
| 可操纵性 | 高（收入确认、折旧政策） | 低        |
| 含义   | 纸面利润         | 真金白银     |
| 用途   | 会计报表         | 估值、分红、回购 |

> **"净利润是意见，现金流是事实。"**

## FCF的用途

公司产生的FCF可以用于：

```
自由现金流
├── 回购股票（Buyback）
├── 分红（Dividend）
├── 偿还债务（Debt Repayment）
├── 收购（Acquisition）
└── 保留现金（Cash Reserve）
```

**一家持续产生高FCF的公司，有更多选择权。**

## 关键指标

### 1. FCF Yield（自由现金流收益率）

$$\text{FCF Yield} = \frac{\text{FCF}}{\text{市值}} \times 100\%$$

| 水平 | 评价 |
|------|------|
| > 8% | 高，可能被低估 |
| 5-8% | 合理 |
| < 3% | 低，可能被高估 |

### 2. FCF Margin（自由现金流利润率）

$$\text{FCF Margin} = \frac{\text{FCF}}{\text{收入}} \times 100\%$$

反映每1元收入能产生多少自由现金。

### 3. FCF vs 净利润

$$\text{FCF / 净利润} > 1 \text{（优秀）}$$

长期来看，FCF应该接近甚至超过净利润（因为折旧是非现金费用）。

## 不同类型公司的FCF特征

| 公司类型 | FCF特征 | 例子 |
|----------|---------|------|
| **成熟科技** | 高FCF，低CapEx | Microsoft、Apple |
| **成长科技** | 低FCF，高CapEx | 早期Amazon |
| **重资产** | FCF受限于高CapEx | 钢铁、航空 |
| **轻资产** | FCF高，CapEx低 | 软件、咨询 |
| **周期性** | FCF随周期波动 | 半导体、能源 |

## 实例：苹果公司（AAPL）

| 指标         | 2024年     |
| ---------- | --------- |
| 经营现金流      | ~1,100亿   |
| 资本支出       | ~110亿     |
| **自由现金流**  | **~990亿** |
| FCF Margin | ~25%      |
| FCF Yield  | ~3.5%     |

苹果的FCF几乎全部用于回购股票和分红——这是成熟公司的典型特征。

## FCF估值基础

DCF模型的核心就是对FCF折现：

$$\text{企业价值} = \sum_{t=1}^{n} \frac{FCF_t}{(1 + WACC)^t} + \frac{终值}{(1 + WACC)^n}$$

**FCF是所有估值方法的"锚"。**

## 红旗信号

| 信号 | 可能含义 |
|------|----------|
| FCF持续为负 | 公司在烧钱，依赖融资 |
| 净利润高但FCF低 | 利润质量差，应收账款堆积 |
| CapEx远超折旧 | 可能在过度投资 |
| FCF全部用于收购 | 靠并购驱动增长，风险高 |

2. 一家公司FCF很高但不回购不分红，钱去哪了？
3. 如何用FCF判断一家公司是否被低估？
