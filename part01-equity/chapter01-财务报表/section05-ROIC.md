# ROIC（投入资本回报率）

## 定义

> **ROIC（Return on Invested Capital，投入资本回报率）**衡量的是公司用**所有投入的资本**（包括股东的钱和借来的钱）赚了多少利润。它比ROE更能反映公司**真实的投资回报能力**。

$$\text{ROIC} = \frac{\text{NOPAT}}{\text{投入资本}} \times 100\%$$

其中：
- **NOPAT** = 营业利润 ×（1 - 税率）= 税后营业利润
- **投入资本** = 股东权益 + 有息负债 - 现金

## ROE vs ROIC

| 维度 | ROE | ROIC |
|------|-----|------|
| 分子 | 净利润 | NOPAT |
| 分母 | 股东权益 | 全部投入资本 |
| 是否受杠杆影响 | **是** | **否** |
| 适合比较 | 同杠杆水平的公司 | 不同资本结构的公司 |

**核心区别**：ROE可以被杠杆"虚增"，ROIC不行。

## 为什么ROIC比ROE更重要？

> **一个公司的价值 = 投入资本 + 超额回报的现值**

$$\text{超额回报} = (\text{ROIC} - \text{WACC}) \times \text{投入资本}$$

| 情况 | 含义 |
|------|------|
| **ROIC > WACC** | 公司创造价值（经济利润为正） |
| **ROIC = WACC** | 公司刚好赚到资本成本 |
| **ROIC < WACC** | 公司毁灭价值（经济利润为负） |

**这是估值的底层逻辑**——长期来看，只有ROIC > WACC的公司才值得投资。

## ROIC的简化计算

实际分析中常用简化版本：

$$\text{ROIC} \approx \frac{\text{营业利润} \times (1 - \text{税率})}{\text{股东权益} + \text{有息负债} - \text{超额现金}}$$

或更简单：

$$\text{ROIC} \approx \frac{\text{EBIT} \times (1 - \text{税率})}{\text{总资产} - \text{无息负债}}$$

## 不同水平的ROIC

| ROIC水平 | 评价 | 含义 |
|----------|------|------|
| **> 25%** | 卓越 | 具有强大护城河 |
| **15-25%** | 优秀 | 竞争优势明显 |
| **10-15%** | 良好 | 投资回报合理 |
| **5-10%** | 一般 | 接近资本成本 |
| **< 5%** | 较差 | 可能在毁灭价值 |

## 实例

| 公司 | ROIC | WACC | 经济利润 | 说明 |
|------|------|------|----------|------|
| **Microsoft** | ~30% | ~9% | 正 | 强大护城河，持续创造价值 |
| **NVIDIA** | ~40% | ~10% | 正 | AI浪潮下的超额回报 |
| **航空公司** | ~5% | ~8% | 负 | 行业特性，资本毁灭 |
| **公用事业** | ~6% | ~6% | ≈0 | 只赚到资本成本 |

## ROIC与护城河

**高ROIC持续多年 = 公司有护城河**

因为如果没有护城河，竞争对手会涌入，把ROIC拉低到资本成本附近。

```
无护城河公司：  ROIC → WACC → 0%（竞争侵蚀利润）
有护城河公司：  ROIC >> WACC（长期维持超额回报）
```

## ROIC的局限性

1. **计算依赖估计** — NOPAT和投入资本的定义有多种
2. **资本化vs费用化** — 不同会计处理影响投入资本
3. **轻资产公司** — 品牌、IP等无形资产可能被低估
4. **周期性行业** — 单年ROIC可能失真，需要看多年平均
