# 资本配置能力

## 定义

> **资本配置（Capital Allocation）**是指管理层如何决定使用公司赚来的钱——是再投资、收购、回购股票、分红还是偿还债务。这是管理层最重要的能力之一。

> **"管理层最重要的工作就是配置资本。"** —— 沃伦·巴菲特

## 资本配置的五种选择

```
公司产生的现金
├── 1. 再投资（CapEx）         → 扩大产能、研发
├── 2. 收购（M&A）             → 买其他公司
├── 3. 回购股票（Buyback）     → 减少股本
├── 4. 分红（Dividend）        → 直接回馈股东
└── 5. 偿还债务（Debt Paydown）→ 降低杠杆
```

### 如何选择？

| 情况 | 最优选择 |
|------|----------|
| ROIC >> WACC | 再投资（回报高于资本成本） |
| ROIC ≈ WACC | 回购或分红（再投资不创造价值） |
| 股价被严重低估 | 回购股票 |
| 股价被高估 | 发行股票（稀释） |
| 负债过高 | 偿还债务 |
| 有好的收购标的 | 收购 |

## 资本配置的评估标准

### 1. 再投资回报率

$$\text{再投资回报率} = \frac{\text{新增利润}}{\text{再投资金额}}$$

| 水平 | 评价 |
|------|------|
| > 20% | 优秀 |
| 10-20% | 良好 |
| < 10% | 一般（可能不如回购） |

### 2. 收购记录

| 维度 | 好收购 | 差收购 |
|------|--------|--------|
| 价格 | 合理或低估 | 高溢价 |
| 整合 | 成功整合 | 整合失败 |
| 协同 | 实现协同效应 | 协同未实现 |
| 战略 | 符合长期战略 | 盲目多元化 |

### 3. 回购时机

| 时机 | 评价 |
|------|------|
| 股价低估时回购 | ✅ 优秀 |
| 股价合理时回购 | ✅ 良好 |
| 股价高估时回购 | ❌ 毁灭价值 |

### 4. 分红政策

| 维度 | 好分红 | 差分红 |
|------|--------|--------|
| 稳定性 | 持续稳定增长 | 不稳定 |
| 派息率 | 合理（30-60%） | 过高（> 80%） |
| 增长 | 随盈利增长 | 不增长 |

## 典型案例

### 优秀资本配置：Apple

| 年份 | 动作 | 说明 |
|------|------|------|
| 2012-至今 | 大量回购 | 股本从26B→15B，大幅减少 |
| 持续 | 分红 | 稳定增长的分红 |
| 持续 | 研发投入 | 保持技术领先 |

**结果**：EPS增速远高于净利润增速

### 优秀资本配置：Berkshire Hathaway

| 策略 | 说明 |
|------|------|
| 不分红 | 利润全部再投资 |
| 收购优质公司 | 长期持有 |
| 股票投资 | 低价买入优质公司 |
| 保留现金 | 等待机会 |

**结果**：长期复合回报率远超市场

### 问题资本配置：部分公司

| 问题 | 后果 |
|------|------|
| 高价收购 | 商誉减值 |
| 股价高估时回购 | 毁灭价值 |
| 盲目扩张 | ROIC下降 |
| 过度分红 | 缺乏再投资资金 |

## 资本配置分析框架

```
1. 公司的ROIC vs WACC是多少？
   └── ROIC > WACC → 应该再投资
   └── ROIC < WACC → 应该回购或分红

2. 过去5年的资本配置记录如何？
   ├── 再投资回报率
   ├── 收购成功率
   ├── 回购时机
   └── 分红稳定性

3. 管理层是否有清晰的资本配置理念？
   └── 致股东信中的阐述

4. 是否存在毁灭价值的行为？
   ├── 高价收购
   ├── 盲目扩张
   └── 过度激励
```

## 红旗信号

| 信号 | 可能含义 |
|------|----------|
| 频繁高价收购 | 管理层贪大求全 |
| ROIC持续低于WACC但仍大量投资 | 资本配置效率低 |
| 股价高估时大量回购 | 毁灭股东价值 |
| 负债率持续上升但仍分红 | 财务纪律差 |
| 管理层薪酬与资本配置效率脱钩 | 治理问题 |

## 思考题

1. 为什么说"回购是分红的替代品"？在什么情况下回购比分红更好？
2. 一家公司ROIC只有5%，但仍在大量再投资，这是好是坏？
3. 如何判断管理层的收购是在"创造价值"还是"毁灭价值"？
