# 价差策略（Credit Spread）

## 1. 策略概述

价差策略（Credit Spread）是卖方策略中最重要的一步进阶。它解决了裸卖出期权风险无限的问题，同时保留了卖方赚取时间价值的优势。

> **Credit Spread = 卖出一个期权 + 买入一个同类但更远虚值的期权**

通过买入一个更便宜的保护性期权，你**限制了最大亏损**，代价是**降低了最大盈利**（权利金收入减少）。

### 两种基本类型

| 类型 | 构成 | 适用观点 |
|------|------|---------|
| **Bull Put Spread** | 卖 Put（高行权价） + 买 Put（低行权价） | 温和看多 |
| **Bear Call Spread** | 卖 Call（低行权价） + 买 Call（高行权价） | 温和看空 |

## 2. Bull Put Spread（牛市看跌价差）

### 构建方式

- **卖出** 1 张较高行权价的 Put（收取较多权利金）
- **买入** 1 张较低行权价的 Put（支付较少权利金）
- 两者到期日相同，数量相同

**净收入** = 卖出 Put 收取的权利金 - 买入 Put 支付的权利金

### 盈亏分析

| 指标 | 公式 |
|------|------|
| 最大盈利 | 净权利金收入（Net Premium） |
| 最大亏损 | (行权价间距 - 净权利金) × 100 |
| 盈亏平衡点 | 卖出行权价 - 净权利金 |
| 盈利条件 | 到期股价 > 卖出的行权价 |

### ASCII 盈亏图

```
收益
  ^
  |  ‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾  ← 最大盈利 = 净权利金
  |  |              |
─ ─ ─|─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ → 股价
  |                 \
  |                  \
  |                   ‾‾‾‾‾‾‾‾‾‾‾  ← 最大亏损（有限）
  |
  ↓
  卖出Put行权价  买入Put行权价
```

### 实例分析

**场景**：
- AAPL 当前价格：$155
- 卖出 1 张 $150 Put，收取权利金 $4.00
- 买入 1 张 $140 Put，支付权利金 $1.50
- 净收入 = $4.00 - $1.50 = $2.50/股 = $250/张

**情景分析**：

| 到期股价 | 卖出$150 Put | 买入$140 Put | 净盈亏 |
|---------|-------------|-------------|--------|
| $160 | +$400（保留全部权利金） | -$150（期权归零） | +$250 |
| $155 | +$400 | -$150 | +$250 |
| $150 | +$400 | -$150 | +$250 |
| $148 | +$200 | -$150 | +$50 |
| $147.50 | $0 | -$150 | -$150（盈亏平衡） |
| $145 | -$200 | -$150 | -$350 |
| $140 | -$700 | -$150 | -$850 |
| $130 | -$1,700 | +$850 | -$850（最大亏损） |

**关键数据**：
- 最大盈利：$250（股价 > $150）
- 最大亏损：($150 - $140 - $2.50) × 100 = $750（股价 < $140）
- 盈亏平衡点：$150 - $2.50 = $147.50
- 风险回报比：$750 / $250 = 3:1

## 3. Bear Call Spread（熊市看涨价差）

### 构建方式

- **卖出** 1 张较低行权价的 Call（收取较多权利金）
- **买入** 1 张较高行权价的 Call（支付较少权利金）
- 两者到期日相同，数量相同

### 盈亏分析

| 指标 | 公式 |
|------|------|
| 最大盈利 | 净权利金收入 |
| 最大亏损 | (行权价间距 - 净权利金) × 100 |
| 盈亏平衡点 | 卖出行权价 + 净权利金 |
| 盈利条件 | 到期股价 < 卖出的行权价 |

### ASCII 盈亏图

```
收益
  ^
  |  ‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾  ← 最大盈利 = 净权利金
  |                 |
─ ─ ─|─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ → 股价
  |                /
  |               /
  |              /
  ‾‾‾‾‾‾‾‾‾‾‾‾‾  ← 最大亏损（有限）
  |
  ↓
  卖出Call行权价  买入Call行权价
```

### 实例分析

**场景**：
- AAPL 当前价格：$155
- 卖出 1 张 $160 Call，收取权利金 $3.50
- 买入 1 张 $170 Call，支付权利金 $1.20
- 净收入 = $3.50 - $1.20 = $2.30/股 = $230/张

**关键数据**：
- 最大盈利：$230（股价 < $160）
- 最大亏损：($170 - $160 - $2.30) × 100 = $770（股价 > $170）
- 盈亏平衡点：$160 + $2.30 = $162.30

## 4. 两种 Spread 的对比

| 维度 | Bull Put Spread | Bear Call Spread |
|------|----------------|-----------------|
| 市场观点 | 温和看多 | 温和看空 |
| 构建 | 卖高Put + 买低Put | 卖低Call + 买高Call |
| 盈利条件 | 股价 > 卖出Put行权价 | 股价 < 卖出Call行权价 |
| Greeks Delta | 正（净多头Delta） | 负（净空头Delta） |
| 最佳入场 | 股价跌到支撑位时 | 股价涨到阻力位时 |

## 5. 行权价选择的艺术

### 风险回报比的权衡

| 卖出行权价（Delta） | 买入行权价 | 净权利金 | 最大亏损 | 风险回报比 | 胜率（近似） |
|-------------------|----------|---------|---------|----------|------------|
| 0.40 Delta | 更远 OTM | 较多 | 较大 | 较差 | ~60% |
| 0.30 Delta | 更远 OTM | 中等 | 中等 | 中等 | ~70% |
| 0.20 Delta | 更远 OTM | 较少 | 较小 | 较好 | ~80% |
| 0.10 Delta | 更远 OTM | 很少 | 很小 | 最好 | ~90% |

**经验法则**：
- 卖出 0.30 Delta 的期权是常见的起点
- 买入的行权价间距不要太窄（风险回报比差），也不要太宽（最大亏损太大）
- 常见间距：$5 或 $10（取决于标的股价）

### 到期日选择

- **30-45天**：最常用，Theta衰减速度最快
- **60-90天**：权利金更多，但Theta衰减较慢
- **低于21天**：Gamma风险增大，不建议新手使用

## 6. Greeks 分析

### Bull Put Spread 的 Greeks

| Greeks | 值 | 说明 |
|--------|---|------|
| Delta | 正（小） | 温和看多 |
| Gamma | 接近0 | 买卖期权的Gamma部分抵消 |
| Theta | 正 | 时间衰减有利 |
| Vega | 接近0 | 买卖期权的Vega大部分抵消 |

### Bear Call Spread 的 Greeks

| Greeks | 值 | 说明 |
|--------|---|------|
| Delta | 负（小） | 温和看空 |
| Gamma | 接近0 | 买卖期权的Gamma部分抵消 |
| Theta | 正 | 时间衰减有利 |
| Vega | 接近0 | 买卖期权的Vega大部分抵消 |

**关键洞察**：Credit Spread 的 Greeks 都比较小，这意味着它是一个**方向性不强、主要靠时间衰减获利**的策略。

## 7. 与裸卖出的对比

| 维度 | Credit Spread | 裸卖出（Naked） |
|------|--------------|----------------|
| 最大亏损 | 有限（已知） | 理论上无限 |
| 保证金要求 | 低（行权价间距 - 净权利金） | 高 |
| 权利金收入 | 较少（买保护期权消耗部分） | 较多 |
| 风险控制 | 内置 | 需要主动管理 |
| 适合人群 | 所有交易者 | 经验丰富的交易者 |

## 8. 管理和调整

### 8.1 止损规则

- **亏损达到最大亏损的 50% 时平仓**：如果最大亏损是 $750，当亏损达到 $375 时考虑止损
- **股价突破卖出的行权价时评估**：不一定要立即止损，但要提高警惕

### 8.2 止盈规则

- **盈利达到最大盈利的 50-75% 时平仓**：不值得为剩余的 25% 继续承担风险
- 经验法则：盈利 50% 时平仓，重新寻找机会

### 8.3 Rolling 操作

- **Roll Out**：到期前如果股价在危险区域，可以将到期日延后，收取更多权利金
- **Roll Up/Down**：调整行权价，但要注意不要扩大风险

## 9. 常见错误

### ❌ 错误一：间距太窄

卖出 $150 Put + 买入 $148 Put，间距只有 $2。净权利金可能只有 $0.30，但最大亏损是 $1.70，风险回报比 5.7:1，非常不划算。

### ❌ 错误二：间距太宽

卖出 $150 Put + 买入 $100 Put，间距 $50。最大亏损 $5,000，和裸卖 Put 差不多。

### ❌ 错误三：选择流动性差的期权

流动性差的期权 Bid-Ask Spread 很大，进出成本高，会严重影响策略收益。

### ❌ 错误四：财报前建仓

财报可能导致股价大幅跳空，Credit Spread 的保护作用有限。

### ❌ 错误五：过度交易

不要因为 Credit Spread 风险有限就频繁交易。每次交易都有成本，过度交易会侵蚀收益。

## 10. IBKR 操作说明

### 下单步骤

1. 在 TWS 中打开期权链
2. 选择"策略"（Strategy）模式
3. 选择 Vertical Spread（垂直价差）
4. 设置买卖方向和行权价
5. 确认净收入和最大亏损
6. 使用 Limit Order 设置净收入价格

### 注意事项

- Credit Spread 作为组合下单，确保使用"价差"订单类型
- 不要分两笔单独下单，否则会变成裸卖出
- IBKR 会自动计算组合的保证金要求

## 11. 小结

| 要点 | 内容 |
|------|------|
| 策略构成 | 卖出一个期权 + 买入更远虚值的同类期权 |
| 两种类型 | Bull Put Spread（看多）、Bear Call Spread（看空） |
| 最大盈利 | 净权利金 |
| 最大亏损 | 行权价间距 - 净权利金（有限） |
| 核心优势 | 有限风险、低保证金、赚取时间价值 |
| 核心劣势 | 盈利有限、风险回报比可能不佳 |
| 最佳入场 | 温和看多/看空、高IV环境 |
