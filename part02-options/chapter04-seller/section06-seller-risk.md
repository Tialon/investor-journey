# 卖方风险管理

## 1. 卖方的核心风险

卖方的本质是**收取权利金，承担风险**。这就像保险公司——大部分时间保费是白赚的，但偶尔一次大灾难可能赔得倾家荡灭。

> **卖方的风险不是"可能亏钱"，而是"可能亏大钱"。**

### 卖方面临的主要风险

| 风险类型 | 说明 | 例子 |
|---------|------|------|
| **方向风险** | 股价大幅反向运动 | 卖 Call 后股价暴涨 |
| **波动率风险** | IV 大幅上升 | 卖期权后 IV 翻倍 |
| **Gamma 风险** | 临近到期股价剧烈波动 | 最后一天股价跳空 |
| **事件风险** | 黑天鹅事件 | 财报暴雷、政策突变 |
| **流动性风险** | 无法平仓 | 极端行情下 Bid-Ask 失效 |

## 2. 仓位管理——最重要的防线

### 2.1 单笔交易仓位限制

**经验法则**：每笔卖方交易的最大亏损不超过总资金的 **2-5%**。

| 账户资金 | 单笔最大亏损（2%） | 单笔最大亏损（5%） |
|---------|------------------|------------------|
| $10,000 | $200 | $500 |
| $50,000 | $1,000 | $2,500 |
| $100,000 | $2,000 | $5,000 |
| $500,000 | $10,000 | $25,000 |

### 2.2 总仓位限制

**经验法则**：所有卖方头寸的最大亏损之和不超过总资金的 **10-20%**。

例如：
- 账户资金 $100,000
- 总最大亏损不超过 $20,000
- 如果每个 Iron Condor 最大亏损 $500，最多持有 40 个

### 2.3 分散化

| 分散维度 | 建议 |
|---------|------|
| 标的分散 | 不要只卖一个标的的期权 |
| 到期日分散 | 不要所有头寸同一时间到期 |
| 策略分散 | 不要只用一种策略 |
| 行业分散 | 不要集中在同一行业 |

## 3. 止损策略

### 3.1 百分比止损

最简单的方法：当亏损达到最大亏损的一定百分比时平仓。

| 止损百分比 | 特点 |
|----------|------|
| 50% | 最常用，平衡止损频率和保护效果 |
| 30% | 更保守，止损更频繁 |
| 70% | 更激进，给策略更多空间 |

**操作流程**：
1. 建仓时记录最大亏损金额
2. 设定止损百分比（如 50%）
3. 当浮亏达到止损线时，平仓
4. 不要犹豫，不要抱有"再等等"的心态

### 3.2 技术止损

基于标的的技术指标止损：
- 股价突破关键支撑/阻力位
- 股价突破卖出的行权价
- 股价突破盈亏平衡点

### 3.3 时间止损

- 到期前 7 天如果仍在亏损，考虑平仓
- 临近到期 Gamma 风险急剧增大，不值得继续持有

### 3.4 波动率止损

- IV 大幅上升（如增加 20%+）时考虑平仓
- IV 上升意味着市场预期更大波动，卖方风险增加

## 4. 止盈策略

### 4.1 固定百分比止盈

**经验法则**：盈利达到最大盈利的 **50-75%** 时平仓。

| 止盈百分比 | 特点 |
|----------|------|
| 50% | 最常用，不贪心 |
| 75% | 留一些空间，但锁定大部分利润 |
| 100% | 持有到期，但承担全部风险 |

**为什么要提前止盈？**

假设你卖出一个 Iron Condor，最大盈利 $300：
- 当盈利达到 $225（75%）时，继续持有只多赚 $75
- 但如果股价反转，可能亏损 $300+
- 风险回报比已经不划算

### 4.2 时间止盈

- 到期前 7 天如果盈利超过 50%，考虑平仓
- 最后一周 Gamma 风险大，收益已经不多

### 4.3 移动止盈

- 如果股价向有利方向移动，可以移动止损线
- 例如：初始止损 50%，当盈利达到 30% 时，将止损线移到 20%

## 5. 波动率风险管理

### 5.1 IV 环境评估

在建仓前评估当前的 IV 水平：

| IV 水平 | 建议 |
|---------|------|
| IV 很高（>历史 80 分位） | 卖方有利，可以积极建仓 |
| IV 较高（>历史 60 分位） | 卖方较有利 |
| IV 中等（历史 40-60 分位） | 中性，谨慎建仓 |
| IV 较低（<历史 40 分位） | 卖方不利，减少仓位 |
| IV 很低（<历史 20 分位） | 卖方非常不利，考虑买方策略 |

### 5.2 IV Rank 和 IV Percentile

- **IV Rank**：当前 IV 在过去一年 IV 范围中的位置（0-100%）
- **IV Percentile**：过去一年中有多少天的 IV 低于当前 IV

在 IBKR 中可以查看这些指标，用于评估 IV 环境。

### 5.3 Vega 风险控制

- 卖出长期期权的 Vega 风险更大（长期期权 Vega 更大）
- 优先卖出 30-45 天到期的期权
- 避免卖出 60+ 天到期的期权（除非有特殊理由）

## 6. Gamma 风险管理

### 6.1 什么是 Gamma 风险

Gamma 衡量 Delta 的变化速度。临近到期时，ATM 期权的 Gamma 急剧增大，意味着：
- 股价小幅变动 → Delta 大幅变化 → 亏损快速增加

### 6.2 控制 Gamma 风险

| 方法 | 说明 |
|------|------|
| 避免持有到期 | 到期前 7-14 天平仓 |
| 不卖太短期的期权 | 最少 21 天到期 |
| 使用价差策略 | 买卖期权抵消部分 Gamma |
| 监控 Delta 变化 | Delta 变化超过 0.10 时评估 |

### 6.3 Gamma Scalping（反向操作）

如果股价快速变动导致 Delta 偏移，可以通过买卖股票来重新平衡 Delta。但这种方法成本高，不适合频繁使用。

## 7. 事件风险管理

### 7.1 常见事件

| 事件 | 风险等级 | 建议 |
|------|---------|------|
| 财报 | ⭐⭐⭐⭐⭐ | 财报前平仓或大幅减仓 |
| 美联储决议 | ⭐⭐⭐⭐ | 减仓或使用价差保护 |
| FDA审批 | ⭐⭐⭐⭐⭐ | 绝对不要裸卖 |
| 重大诉讼 | ⭐⭐⭐ | 评估影响后决定 |
| 分红除权 | ⭐⭐ | 注意提前行权风险 |
| 指数调仓 | ⭐⭐ | 关注个股影响 |

### 7.2 财报季策略

- **财报前**：平仓或减仓所有裸卖头寸
- **财报后**：等 IV crush 后再建仓
- **使用价差**：如果一定要在财报前建仓，使用 Credit Spread 限制风险

### 7.3 黑天鹅应对

- 永远不要满仓卖方
- 保留至少 30-50% 的现金作为缓冲
- 设置好止损单，即使不盯盘也能自动执行

## 8. 保证金管理

### 8.1 保证金要求

| 策略 | 保证金要求 |
|------|----------|
| 裸卖 Call | 标的价格 × 100 - OTM 金额 + 权利金（很高） |
| 裸卖 Put | 行权价 × 100 的一定比例（高） |
| Credit Spread | 行权价间距 × 100 - 净权利金（低） |
| Iron Condor | 较宽翅膀的间距 × 100 - 净权利金（低） |
| Covered Call | 无额外保证金（已有股票） |
| CSP | 行权价 × 100（现金冻结） |

### 8.2 保证金使用率

**经验法则**：保证金使用率不超过 **50-70%**。

- 保证金使用率过高 → 没有缓冲空间 → 容易被强制平仓
- 保证金使用率适中 → 有足够缓冲 → 可以从容应对波动

### 8.3 保证金追缴（Margin Call）

当账户权益低于保证金要求时，IBKR 会发出保证金追缴：
- 你需要追加资金或平仓
- 如果不及时处理，IBKR 会强制平仓（通常先平亏损最大的头寸）
- 强制平仓的价格通常很差

## 9. 心理管理

### 9.1 卖方的心理挑战

- **高胜率的诱惑**：卖方大部分时间赚钱，容易产生"这次也会赚"的错觉
- **小赚大亏的模式**：连续赚了很多次小钱，一次大亏可能全部赔回去
- **止损的痛苦**：卖方止损意味着"放弃"已经赚到的权利金

### 9.2 心理管理建议

| 建议 | 说明 |
|------|------|
| 接受亏损是常态 | 没有100%胜率的策略 |
| 严格执行止损 | 不要抱有"再等等"的心态 |
| 不要报复性交易 | 亏损后不要急于"赚回来" |
| 记录每笔交易 | 分析成功和失败的原因 |
| 控制交易频率 | 不要因为"容易"就频繁交易 |

### 9.3 避免"卖方陷阱"

**卖方陷阱**：连续多次赚取小权利金后，开始加大仓位或放松风控，然后一次大亏把之前赚的全部吐出来。

**应对方法**：
- 始终遵守仓位和止损规则
- 不要因为"一直赚"就放松警惕
- 记住：卖方的风险是**尾部风险**——不常发生但后果严重

## 10. 风控检查清单

每次建仓前检查：

- [ ] 单笔最大亏损 ≤ 总资金 2-5%
- [ ] 总最大亏损 ≤ 总资金 10-20%
- [ ] 保证金使用率 ≤ 50-70%
- [ ] 标的分散（不集中在单一标的）
- [ ] 到期日分散（不同时到期）
- [ ] 无重大事件（财报、FDA等）
- [ ] IV 环境评估（不在低 IV 时卖方）
- [ ] 使用价差策略限制风险
- [ ] 设置止损规则
- [ ] 有足够现金缓冲

## 11. 实战案例：风控如何救命

### 案例：2020年3月新冠疫情暴跌

**背景**：
- 2020年2月，SPY 在 $330 左右
- 某交易者卖出 SPY $310 Put，收取 $5 权利金
- 最大盈利 $500，看起来很安全

**无风控**：
- 2020年3月，SPY 暴跌到 $220
- 被行权以 $310 买入市价 $220 的股票
- 亏损 = ($310 - $220 - $5) × 100 = **$8,500**

**有风控**：
- 设置止损：亏损达到最大盈利的 50%（$250）时平仓
- 当 SPY 跌到 $320 时，止损触发
- 亏损 = $250
- 虽然亏了 $250，但避免了 $8,500 的灾难

### 教训

> **风控不是为了不亏钱，而是为了不死。**

卖方的利润是有限的（权利金），但风险可能是无限的。风控确保你在"大灾难"中存活下来，继续赚取下一次的权利金。
