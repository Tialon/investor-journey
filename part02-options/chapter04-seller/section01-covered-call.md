# 备兑看涨期权（Covered Call）

## 1. 策略概述

备兑看涨期权（Covered Call）是期权交易中最经典、最广泛使用的策略之一。它的核心思想非常简单：

> **持有股票 + 卖出看涨期权 = Covered Call**

你已经持有 100 股 AAPL，现在你卖出 1 张 AAPL 的看涨期权（Call），收取权利金。如果股价在到期时没有涨过行权价，你白赚权利金；如果涨过了，你的股票会被按行权价买走，但你仍然赚了权利金 + 股票升值的部分。

### 为什么叫"备兑"？

因为你已经持有对应的股票（100股对应1张合约），所以你的卖出是有"保障"的——即使被行权，你只需要交出已持有的股票，不需要在市场上高价买入。这就是"备兑（Covered）"的含义。

与之相对的是**裸卖出看涨期权（Naked Call）**——没有持有股票就卖出 Call，风险理论上是无限的。

## 2. 建仓要素

| 要素 | 说明 |
|------|------|
| 持有股票 | 至少 100 股（1张合约对应100股） |
| 卖出 Call | 选择行权价和到期日 |
| 收取权利金 | 卖出时立即收到权利金 |

### 行权价选择

| 行权价选择 | 权利金收入 | 被行权概率 | 适用场景 |
|-----------|----------|----------|---------|
| OTM（虚值） | 较少 | 较低 | 看好股票继续涨，同时赚取额外收入 |
| ATM（平值） | 中等 | ~50% | 平衡收入和被行权风险 |
| ITM（实值） | 较多 | 较高 | 更注重权利金收入，愿意卖出股票 |

### 到期日选择

- **短期（1-2周）**：权利金少但频率高，Theta衰减快
- **中期（1个月）**：最常见的选择，平衡收入和管理频率
- **长期（2-3个月）**：权利金多但灵活性低

经验法则：**卖出30-45天到期的OTM Call** 是最常用的设置。

## 3. 盈亏分析

### 收益公式

$$\text{总收益} = \text{股票收益} + \text{期权收益}$$

- **如果到期股价 ≤ 行权价**（未被行权）：
  $$\text{收益} = (S_T - S_0) + \text{Premium}$$

- **如果到期股价 > 行权价**（被行权）：
  $$\text{收益} = (K - S_0) + \text{Premium}$$

其中：
- $S_T$ = 到期时股价
- $S_0$ = 买入股票的价格
- $K$ = 行权价
- $\text{Premium}$ = 收取的权利金

### 关键数据

| 指标 | 公式 |
|------|------|
| 最大盈利 | $(K - S_0) + \text{Premium}$ |
| 盈亏平衡点 | $S_0 - \text{Premium}$ |
| 最大亏损 | $S_0 - \text{Premium}$（股价跌到0） |

### ASCII 盈亏图

```
收益
  ^
  |              /‾‾‾‾‾‾‾‾‾  ← 最大盈利上限
  |             /
  |            /
  |           /
  |          /
  |         /
─ ─ ─ ─ ─/─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ → 股价
  |      /|
  |     / |
  |    /  |
  |   /   |
  |  /    |
  | /     ↓
  |/  盈亏平衡点 = 买入价 - 权利金
  |
  ↓ 亏损
```

**注意**：与单纯持有股票相比，Covered Call 的盈利有上限（因为股票被行权价封顶），但下行保护增加了（权利金降低了盈亏平衡点）。

## 4. 实例分析

### 场景设定

- AAPL 当前价格：$155
- 你持有 100 股 AAPL，买入价 $150
- 你卖出 1 张 AAPL Call，行权价 $160，30天后到期
- 收取权利金：$3.50/股 = $350/张

### 情景分析

| 到期股价 | 股票盈亏 | 期权盈亏 | 总盈亏 | 说明 |
|---------|---------|---------|--------|------|
| $140 | -$1,000 | +$350 | -$650 | 股票大跌，权利金部分补偿 |
| $145 | -$500 | +$350 | -$150 | 股票下跌，权利金大部分补偿 |
| $150 | $0 | +$350 | +$350 | 股价不变，纯赚权利金 |
| $155 | +$500 | +$350 | +$850 | 股价小涨，股票+权利金双赚 |
| $160 | +$1,000 | +$350 | +$1,350 | 股价涨到行权价，最大收益 |
| $165 | +$1,000 | +$350 | +$1,350 | 股价超过行权价，收益封顶 |
| $170 | +$1,000 | +$350 | +$1,350 | 继续涨也不多赚 |

**关键观察**：
- 盈亏平衡点 = $150 - $3.50 = $146.50（比原来持有股票的 $150 更低）
- 最大盈利 = ($160 - $150) × 100 + $350 = $1,350
- 如果 AAPL 涨到 $170，单纯持有股票赚 $2,000，Covered Call 只赚 $1,350，少赚了 $650

## 5. Greeks 分析

| Greeks | 值 | 说明 |
|--------|---|------|
| Delta | 正（但小于100股） | 仍看多，但上涨动能被削弱 |
| Gamma | 负 | 卖出Call的Gamma为负，Delta增长变慢 |
| Theta | 正 | 时间衰减对卖方有利，每天赚取时间价值 |
| Vega | 负 | 波动率下降对卖方有利 |

### 与持有股票对比

| 维度 | 持有100股 | Covered Call |
|------|----------|-------------|
| Delta | +100 | +100 - Call Delta ≈ +55~+80 |
| Theta | 0 | 正（赚时间价值） |
| Vega | 0 | 负（怕波动率上升） |
| 最大盈利 | 无限 | 有限（封顶） |
| 下行保护 | 无 | 有（权利金缓冲） |

## 6. 适用场景

### ✅ 最适合

- **温和看多**：认为股价会小幅上涨或横盘
- **持股不动期**：长期持有股票，想增强收益
- **高波动率环境**：IV高时权利金更丰厚
- **降低持仓成本**：用权利金逐步降低持股成本

### ❌ 不适合

- **强烈看多**：大涨行情中 Covered Call 会限制你的收益
- **股价即将大跌**：权利金不足以弥补大跌损失
- **波动率很低**：权利金太少，不值得操作

## 7. 变体策略

### 7.1 Buy-Write

同时买入股票和卖出 Call，一步完成建仓。与先持有股票再卖 Call 效果相同。

### 7.2 Rolling

当股价接近或超过行权价时，可以通过 Rolling 操作延长或调整：

- **Roll Up**：将 Call 的行权价上移（如从 160 调到 170），给股票更多上涨空间
- **Roll Out**：将 Call 的到期日延长（如从本月延到下月），继续收取时间价值
- **Roll Up & Out**：同时上移行权价和延长到期日

### 7.3 覆盖比例调整

不一定要 1:1 覆盖（100股配1张Call）。可以：
- 半覆盖：200股配1张Call（保留更多上涨空间）
- 超覆盖：100股配2张Call（实际上是部分裸卖，风险增大）

## 8. 常见错误

### ❌ 错误一：在强烈看多时使用

如果你认为股票要大涨，不要用 Covered Call。它的本质是**用部分上涨空间换取权利金收入**。

### ❌ 错误二：卖出太短期的虚值 Call

一周到期的深度 OTM Call 权利金极少（可能只有几美元），但交易成本不变，得不偿失。

### ❌ 错误三：不设止损

Covered Call 不是"无风险"策略。如果股票大跌，权利金远远不够弥补。需要在股票层面设置止损。

### ❌ 错误四：财报前卖出 Covered Call

财报可能导致股价大幅波动。如果股价暴涨，你被行权的价格远低于财报后的市场价。

### ❌ 错误五：恐慌性回购

股价涨到行权价附近时，不要恐慌性地高价回购 Call。接受被行权的结果，或者使用 Rolling 策略。

## 9. IBKR 操作说明

### 下单步骤

1. 在 TWS 或网页版中找到你持有的股票
2. 右键选择"交易期权"（Trade Options）
3. 选择"卖出"（Sell）方向
4. 选择行权价和到期日
5. 确认订单类型（通常用 Limit Order）
6. 提交订单

### 注意事项

- 确保账户中已持有对应股票
- IBKR 会自动识别为 Covered Call，不需要额外保证金
- 可以使用 OptionTrader 工具查看策略分析

## 10. 小结

| 要点 | 内容 |
|------|------|
| 策略构成 | 持有股票 + 卖出 Call |
| 适用观点 | 温和看多 / 横盘 |
| 最大盈利 | (行权价 - 买入价) + 权利金 |
| 最大亏损 | 买入价 - 权利金（股价跌到0） |
| 核心优势 | 增强持股收益、降低盈亏平衡点 |
| 核心劣势 | 限制上涨空间 |
| Greeks 特征 | 正Delta、正Theta、负Vega |
