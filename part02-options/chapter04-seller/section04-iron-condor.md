# 铁鹰策略（Iron Condor）

## 1. 策略概述

铁鹰策略（Iron Condor）是期权交易中最受欢迎的中性策略之一。它的核心思想：

> **同时卖出一个 Bull Put Spread 和一个 Bear Call Spread = Iron Condor**

你预期股价会在一个区间内横盘震荡，不会大幅上涨也不会大幅下跌。通过同时卖出上方的 Call Spread 和下方的 Put Spread，你从两边收取权利金。

### 策略构成

```
买入 OTM Put（保护） ← 卖出 OTM Put ← [当前股价] → 卖出 OTM Call → 买入 OTM Call（保护）
    K1                    K2                              K3              K4
```

- **卖出 Put Spread**：卖出 $K_2$ Put + 买入 $K_1$ Put
- **卖出 Call Spread**：卖出 $K_3$ Call + 买入 $K_4$ Call
- 条件：$K_1 < K_2 < \text{当前股价} < K_3 < K_4$

## 2. 建仓要素

| 要素 | 说明 |
|------|------|
| 卖出 OTM Put | 低于当前股价 |
| 买入更低 OTM Put | 保护下方风险 |
| 卖出 OTM Call | 高于当前股价 |
| 买入更高 OTM Call | 保护上方风险 |
| 到期日 | 四个期权相同 |

### 翅膀宽度选择

| 设置 | 特点 |
|------|------|
| 窄翅膀（$5间距） | 最大亏损小，但权利金也少 |
| 宽翅膀（$10间距） | 权利金多，但最大亏损也大 |
| 不对称翅膀 | 根据市场偏向调整上下间距 |

### 常见设置

- **对称 Iron Condor**：上下翅膀宽度相同
- **Delta 选择**：卖出 Delta 0.15-0.30 的期权
- **到期日**：30-45 天

## 3. 盈亏分析

### 收益公式

- **最大盈利** = 净权利金收入（Put Spread 权利金 + Call Spread 权利金）
- **最大亏损** = (翅膀宽度 - 净权利金) × 100（取上下翅膀中较大的那个）
- **上方盈亏平衡点** = 卖出 Call 行权价 + 净权利金
- **下方盈亏平衡点** = 卖出 Put 行权价 - 净权利金
- **盈利条件**：到期股价在两个盈亏平衡点之间

### ASCII 盈亏图

```
收益
  ^
  |               ‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾  ← 最大盈利
  |               |              |
─ ─ ─|─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ → 股价
  |              /                \
  |             /                  \
  |            /                    \
  ‾‾‾‾‾‾‾‾‾‾‾                      ‾‾‾‾‾‾‾‾‾‾‾  ← 最大亏损（有限）
  |
  ↓
  K1(买P)  K2(卖P)         K3(卖C)  K4(买C)
         ↓盈亏平衡              盈亏平衡↓
```

## 4. 实例分析

### 场景设定

- SPY 当前价格：$450
- 卖出 1 张 $435 Put，收取 $3.00
- 买入 1 张 $425 Put，支付 $1.20
- 卖出 1 张 $465 Call，收取 $2.80
- 买入 1 张 $475 Call，支付 $1.00

### 计算

| 指标 | 计算 | 结果 |
|------|------|------|
| Put Spread 净收入 | $3.00 - $1.20 | $1.80 |
| Call Spread 净收入 | $2.80 - $1.00 | $1.80 |
| 总净权利金 | $1.80 + $1.80 | $3.60/股 = $360 |
| 最大盈利 | $360 | $360 |
| 最大亏损（上方） | ($10 - $3.60) × 100 | $640 |
| 最大亏损（下方） | ($10 - $3.60) × 100 | $640 |
| 上方盈亏平衡 | $465 + $3.60 | $468.60 |
| 下方盈亏平衡 | $435 - $3.60 | $431.40 |
| 盈利区间 | $431.40 ~ $468.60 | $37.20 宽度 |

### 情景分析

| 到期 SPY | 盈亏 | 说明 |
|---------|------|------|
| $420 | -$640 | 最大亏损（下方） |
| $425 | -$640 | 开始保护 |
| $431.40 | $0 | 下方盈亏平衡 |
| $435 | +$360 | 开始最大盈利区 |
| $440 | +$360 | 最大盈利 |
| $450 | +$360 | 最大盈利 |
| $465 | +$360 | 结束最大盈利区 |
| $468.60 | $0 | 上方盈亏平衡 |
| $475 | -$640 | 最大亏损（上方） |
| $480 | -$640 | 最大亏损 |

## 5. Greeks 分析

| Greeks | 值 | 说明 |
|--------|---|------|
| Delta | ≈ 0 | 中性策略，方向性很弱 |
| Gamma | 负 | 股价大幅变动对卖方不利 |
| Theta | 正 | 时间衰减是主要利润来源 |
| Vega | 负 | 波动率下降对卖方有利 |

### 关键权衡：Theta vs Gamma

- **Theta（正）**：每天都在赚钱（时间衰减）
- **Gamma（负）**：如果股价大幅变动，Delta 会快速变化，导致亏损

这就是 Iron Condor 的核心博弈：**你赌的是股价不会大幅波动，用时间换空间**。

## 6. 适用场景

### ✅ 最适合

- **横盘市场**：预期股价在区间内震荡
- **高波动率环境**：IV 高时权利金丰厚，Iron Condor 的盈利区间更宽
- **波动率预期下降**：IV 下降时 Iron Condor 价值减少（对卖方有利）
- **财报后**：不确定性消除，IV crush 对卖方有利

### ❌ 不适合

- **趋势市场**：强趋势中 Iron Condor 容易突破盈亏平衡点
- **低波动率环境**：权利金太少，盈利区间太窄
- **重大事件前**：财报、美联储决议等可能导致大幅波动

## 7. 管理和调整

### 7.1 止损规则

- **亏损达到最大亏损的 50% 时平仓**
- **股价突破卖出的行权价时提高警惕**
- **不要等到到期日才止损**

### 7.2 止盈规则

- **盈利达到最大盈利的 50% 时平仓**
- 重新寻找新的 Iron Condor 机会
- 不要贪心，赚到该赚的就走

### 7.3 Rolling 调整

- **Roll Up/Down**：如果股价向一个方向移动，可以调整未受威胁一侧的行权价
- **Roll Out**：延长到期日，收取更多权利金
- **调整翅膀宽度**：收窄或扩大间距

### 7.4 单腿止损

如果股价突破一侧的卖出行权价，可以：
- 平仓整个 Iron Condor
- 只平仓被威胁的那条腿
- 将被威胁的腿 Roll 到更远的位置

## 8. 变体策略

### 8.1 不对称 Iron Condor

上下翅膀宽度不同，或上下 Delta 不同。例如：
- 如果偏向看多：下方 Put Spread 用更宽的间距（更多保护），上方 Call Spread 用更窄的间距
- 如果偏向看空：反之

### 8.2 Iron Condor with Adjustment

预留调整空间：
- 初始只建立一侧（如 Bull Put Spread）
- 等股价移动后再建立另一侧
- 这样可以避免一开始就面临双侧风险

### 8.3 Broken Wing Iron Condor

一侧翅膀比另一侧宽。例如：
- 买入 $420 Put + 卖出 $430 Put + 卖出 $460 Call + 买入 $475 Call
- 上方翅膀 $15 宽，下方翅膀 $10 宽
- 可能产生净收入或净支出

## 9. 常见错误

### ❌ 错误一：盈利区间太窄

卖出 Delta 0.40 的期权，盈利区间可能只有 5%，很容易被突破。建议使用 Delta 0.15-0.30。

### ❌ 错误二：不设止损

Iron Condor 的最大亏损可能远大于最大盈利。如果股价大幅突破，亏损可以非常惊人。

### ❌ 错误三：财报前建仓

财报可能导致股价跳空 10%+，Iron Condor 的盈利区间可能完全被突破。

### ❌ 错误四：频繁调整

过度调整会增加交易成本，也可能在调整过程中错失盈利机会。

### ❌ 错误五：忽视流动性

选择流动性差的期权，Bid-Ask Spread 大，进出成本高。

## 10. IBKR 操作说明

### 下单步骤

1. 在 TWS 中打开期权链
2. 选择"策略"模式
3. 选择 Iron Condor 策略
4. 设置四个行权价和到期日
5. 确认净收入和最大亏损
6. 使用 Limit Order 提交

### 注意事项

- 确保使用组合订单，不要分四笔单独下单
- 检查四个期权的流动性
- IBKR 会自动计算组合的保证金要求

## 11. 统计数据

基于历史数据的 Iron Condor 统计（SPY，30天到期，Delta 0.20）：

| 指标 | 大致范围 |
|------|---------|
| 月度胜率 | 60-75% |
| 平均月收益 | 2-5% |
| 最大单月亏损 | 10-20% |
| 风险回报比 | 1:2 到 1:3 |

> ⚠️ 以上数据仅供参考，实际表现取决于市场环境和具体参数设置。

## 12. 小结

| 要点 | 内容 |
|------|------|
| 策略构成 | Bull Put Spread + Bear Call Spread |
| 适用观点 | 中性/横盘 |
| 最大盈利 | 净权利金 |
| 最大亏损 | 翅膀宽度 - 净权利金（有限） |
| 盈利条件 | 股价在盈亏平衡区间内 |
| 核心优势 | 有限风险、双边收权利金、高胜率 |
| 核心劣势 | 风险回报比不佳、大幅波动时亏损 |
| Greeks | Delta≈0、正Theta、负Vega、负Gamma |
