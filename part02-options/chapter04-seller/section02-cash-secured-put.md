# 现金担保看跌期权（Cash-Secured Put）

## 1. 策略概述

现金担保看跌期权（Cash-Secured Put，简称 CSP）是另一个经典的卖方策略。它的核心思想：

> **卖出看跌期权 + 准备足够的现金买入股票 = Cash-Secured Put**

你卖出 1 张 AAPL 的看跌期权（Put），收取权利金。同时在账户中准备足够的现金（行权价 × 100），以备被行权时买入股票。

### 策略本质

CSP 的本质是**"愿意以某个价格买入股票，同时赚取等待的权利金"**。

换个角度理解：
- 你看好 AAPL，但觉得当前价格 $155 偏高
- 你愿意在 $145 买入
- 你卖出 $145 行权价的 Put，收取 $3 权利金
- 如果股价跌到 $145 以下，你被行权，以 $145 买入（实际成本 $145 - $3 = $142）
- 如果股价没跌到 $145，你白赚 $3 权利金

## 2. 建仓要素

| 要素 | 说明 |
|------|------|
| 卖出 Put | 选择行权价和到期日 |
| 准备现金 | 行权价 × 100 × 合约数 |
| 收取权利金 | 卖出时立即收到 |

### 行权价选择

| 行权价 | 权利金 | 被行权概率 | 思路 |
|--------|-------|----------|------|
| 深度 OTM（远低于现价） | 很少 | 很低 | 纯赚权利金，但收入微薄 |
| OTM（低于现价） | 较少 | 较低 | 主要赚权利金，附带低价买入机会 |
| ATM（接近现价） | 中等 | ~50% | 平衡权利金和买入意愿 |
| ITM（高于现价） | 较多 | 较高 | 更倾向于被行权买入股票 |

**最常用**：卖出 OTM Put，行权价在你愿意买入股票的价格附近。

## 3. 盈亏分析

### 收益公式

- **如果到期股价 ≥ 行权价**（未被行权）：
  $$\text{收益} = \text{Premium}$$

- **如果到期股价 < 行权价**（被行权）：
  $$\text{收益} = \text{Premium} - (K - S_T) \times 100$$

即以行权价买入了市价更低的股票，但权利金降低了成本。

### 关键数据

| 指标 | 公式 |
|------|------|
| 最大盈利 | Premium（权利金） |
| 盈亏平衡点 | $K - \text{Premium}$ |
| 最大亏损 | $(K - \text{Premium}) \times 100$（股价跌到0） |

### ASCII 盈亏图

```
收益
  ^
  |  ‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾  ← 最大盈利 = 权利金
  |  |                |
─ ─ ─|─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ → 股价
  |                  \
  |                   \
  |                    \
  |                     \
  |                      \
  |                       \
  ↓                        ↓ 亏损
                        盈亏平衡点 = K - Premium
```

**与 Covered Call 的盈亏图对比**：CSP 的盈亏图与 Covered Call 几乎完全对称！这是因为两者在数学上是等价的（Put-Call Parity 的体现）。

## 4. 实例分析

### 场景设定

- AAPL 当前价格：$155
- 你卖出 1 张 AAPL Put，行权价 $145，30天后到期
- 收取权利金：$2.80/股 = $280/张
- 准备现金：$145 × 100 = $14,500

### 情景分析

| 到期股价 | 是否行权 | 期权盈亏 | 说明 |
|---------|---------|---------|------|
| $170 | 否 | +$280 | 白赚权利金 |
| $160 | 否 | +$280 | 白赚权利金 |
| $155 | 否 | +$280 | 白赚权利金 |
| $145 | 否 | +$280 | 刚好未被行权 |
| $142.20 | 是 | $0 | 盈亏平衡点（$145 - $2.80） |
| $140 | 是 | -$220 | 以$145买入市价$140的股票，净亏$220 |
| $130 | 是 | -$1,220 | 以$145买入市价$130的股票 |
| $120 | 是 | -$2,220 | 以$145买入市价$120的股票 |

**关键观察**：
- 如果被行权，你以 $145 买入 AAPL，但实际成本是 $145 - $2.80 = $142.20
- 如果 AAPL 没跌到 $145，你白赚 $280，相当于年化约 22% 的回报（$280/$14,500 × 12/1）

## 5. Greeks 分析

| Greeks | 值 | 说明 |
|--------|---|------|
| Delta | 负（卖出Put的Delta为负） | 股价上涨对你有利 |
| Gamma | 负 | Delta变化对卖方不利 |
| Theta | 正 | 时间衰减对卖方有利 |
| Vega | 负 | 波动率下降对卖方有利 |

## 6. CSP vs Covered Call

| 维度 | CSP | Covered Call |
|------|-----|-------------|
| 前提 | 有现金，想买股票 | 有股票，想增强收益 |
| 收益结构 | 等价（Put-Call Parity） | 等价 |
| 最大盈利 | 权利金 | 行权价差 + 权利金 |
| 资金占用 | 需要冻结现金 | 需要持有股票 |
| 税务处理 | 被行权后成本基础=行权价-权利金 | 卖出股票有资本利得 |

### 等价性证明

根据 Put-Call Parity：
$$C - P = S - K \cdot e^{-rT}$$

卖出 Put + 持有现金 ≈ 卖出 Covered Call（在不考虑利率的情况下）

## 7. 适用场景

### ✅ 最适合

- **想买入某只股票但觉得当前价格偏高**：卖出 Put 等待价格回调，同时赚取权利金
- **看好市场但想低成本建仓**：用 CSP 代替直接买入
- **高波动率环境**：IV 高时权利金丰厚
- **长期投资者**：不急于买入，愿意等待

### ❌ 不适合

- **强烈看空**：如果股价大跌，你会以远高于市场的价格买入
- **资金不足**：必须有足够的现金覆盖行权义务
- **波动率很低**：权利金太少，资金利用效率低

## 8. 等价关系：CSP = 低价限价买单

CSP 本质上是一种**"带收入的限价买单"**：

- 你想在 $145 买入 AAPL → 挂 $145 的限价买单
- 或者：卖出 $145 Put，收取 $2.80 权利金

两种方式的区别：
- 限价单：免费，但可能永远不成交
- CSP：立即收到 $2.80，但如果股价跌到 $145 以下你必须以 $145 买入（即使市价更低）

## 9. 常见错误

### ❌ 错误一：在强烈看空时卖出 Put

如果你认为股价会大跌，不要卖出 Put。被行权买入后股价继续下跌的损失可能远超权利金。

### ❌ 错误二：卖出太多 Put

CSP 的风险是被行权后股价继续大跌。控制仓位，不要把所有资金都用于担保 Put。

### ❌ 错误三：选择太深虚值的 Put

Delta 0.05 的 Put 权利金极少，资金利用效率极低。你的现金被冻结，回报率可能还不如货币基金。

### ❌ 错误四：忽视财报等事件

财报后股价可能大幅跳空。如果你卖出的 Put 在财报后变为深度实值，亏损可能很大。

### ❌ 错误五：被行权后恐慌卖出

如果你被行权以 $145 买入了 AAPL，而市价是 $140，不要恐慌卖出。如果你当初是看好 AAPL 的，$142.20（扣除权利金后）的成本可能正是你想要的买入价。

## 10. 高级技巧：Wheel Strategy

Wheel Strategy 是 CSP 和 Covered Call 的组合循环：

```
卖出 Put → 被行权 → 买入股票 → 卖出 Covered Call → 被行权 → 卖出股票 → 回到卖出 Put
```

### 流程

1. **第一阶段**：持续卖出 OTM Put，收取权利金
2. **如果被行权**：以行权价买入股票，进入第二阶段
3. **第二阶段**：对持有的股票卖出 Covered Call
4. **如果被行权**：股票被以行权价卖出，回到第一阶段

**优点**：持续产生权利金收入，无论股价涨跌
**风险**：股票大跌时亏损可能很大

## 11. IBKR 操作说明

### 下单步骤

1. 在 TWS 中搜索标的股票
2. 选择"交易期权"→"卖出"→"Put"
3. 选择行权价（你愿意买入的价格）和到期日
4. 确认保证金要求（CSP 需要冻结 行权价×100 的现金）
5. 使用 Limit Order 提交

### 保证金要求

- CSP 不需要额外保证金，因为你已经准备了足够的现金
- 但如果账户开启了 Portfolio Margin，实际资金占用可能更低

## 12. 小结

| 要点 | 内容 |
|------|------|
| 策略构成 | 卖出 Put + 准备足够的现金 |
| 适用观点 | 温和看多 / 想低价买入 |
| 最大盈利 | 权利金 |
| 最大亏损 | 行权价 - 权利金（股价跌到0） |
| 核心优势 | 赚取权利金 + 有机会低价买入 |
| 核心劣势 | 被行权后股价可能继续跌 |
| 与 Covered Call | 数学上近似等价 |

1. 你想买入 AAPL，当前价格 $155。你卖出 $145 Put 收取 $3 权利金。到期时 AAPL 跌到 $135，你的实际买入成本是多少？比直接在 $155 买入亏了多少？
2. 为什么说 CSP 和 Covered Call 是"镜像策略"？它们在什么情况下表现不同？
3. Wheel Strategy 的核心逻辑是什么？它在什么市场环境下表现最好？
