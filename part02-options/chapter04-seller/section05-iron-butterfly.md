# 铁蝶策略（Iron Butterfly）

## 1. 策略概述

铁蝶策略（Iron Butterfly）是 Iron Condor 的"收紧版"。它的核心思想：

> **卖出 ATM Straddle + 买入 OTM Strangle = Iron Butterfly**

你预期股价在到期时会非常接近当前价格，不会大幅变动。通过卖出平值的跨式组合（Straddle），你收取最大的权利金；同时买入虚值的宽跨式组合（Strangle）来限制风险。

### 与 Iron Condor 的区别

| 维度 | Iron Condor | Iron Butterfly |
|------|------------|----------------|
| 卖出行权价 | OTM（两侧都虚值） | ATM（中间平值） |
| 最大盈利 | 较少 | 较多 |
| 盈利区间 | 较宽 | 较窄 |
| 胜率 | 较高 | 较低 |
| 风险回报比 | 较好 | 较差 |

## 2. 策略构成

```
买入 OTM Put ← 卖出 ATM Put / ATM Call → 买入 OTM Call
    K1              K2 (=当前股价)  K3              K4
```

- **卖出 ATM Put**（行权价 $K_2$） + **卖出 ATM Call**（行权价 $K_3$，通常 $K_2 = K_3$）
- **买入 OTM Put**（行权价 $K_1$） + **买入 OTM Call**（行权价 $K_4$）

### 简化理解

Iron Butterfly = **卖出 ATM Straddle** + **买入 OTM Strangle**

## 3. 盈亏分析

### 收益公式

- **最大盈利** = 净权利金收入（在 ATM 行权价处）
- **最大亏损** = (翅膀宽度 - 净权利金) × 100
- **上方盈亏平衡** = ATM 行权价 + 净权利金
- **下方盈亏平衡** = ATM 行权价 - 净权利金
- **盈利条件**：到期股价在盈亏平衡点之间

### ASCII 盈亏图

```
收益
  ^
  |                    *
  |                   / \
  |                  /   \
  |                 /     \
─ ─ ─ ─ ─ ─ ─ ─ ─/─ ─ ─ ─\─ ─ ─ ─ ─ ─ → 股价
  |              /         \
  |             /           \
  ‾‾‾‾‾‾‾‾‾‾‾‾             ‾‾‾‾‾‾‾‾‾‾‾‾  ← 最大亏损
  |
  ↓
  K1(买P)    K2/K3(卖P/C)    K4(买C)
         ↓盈亏平衡    盈亏平衡↓
```

**注意**：Iron Butterfly 的盈亏图看起来像一只蝴蝶，因此得名。

## 4. 实例分析

### 场景设定

- AAPL 当前价格：$155
- 卖出 1 张 $155 Put，收取 $5.50
- 卖出 1 张 $155 Call，收取 $5.00
- 买入 1 张 $145 Put，支付 $2.00
- 买入 1 张 $165 Call，支付 $1.80

### 计算

| 指标 | 计算 | 结果 |
|------|------|------|
| 卖出 Straddle 收入 | $5.50 + $5.00 | $10.50 |
| 买入 Strangle 支出 | $2.00 + $1.80 | $3.80 |
| 净权利金 | $10.50 - $3.80 | $6.70/股 = $670 |
| 最大盈利 | $670 | $670 |
| 最大亏损 | ($10 - $6.70) × 100 | $330 |
| 上方盈亏平衡 | $155 + $6.70 | $161.70 |
| 下方盈亏平衡 | $155 - $6.70 | $148.30 |
| 盈利区间 | $148.30 ~ $161.70 | $13.40 宽度 |

### 情景分析

| 到期 AAPL | 盈亏 | 说明 |
|----------|------|------|
| $140 | -$330 | 最大亏损（下方） |
| $145 | -$330 | 买入Put保护 |
| $148.30 | $0 | 下方盈亏平衡 |
| $150 | +$170 | 盈利区 |
| $155 | +$670 | 最大盈利（股价=行权价） |
| $160 | +$170 | 盈利区 |
| $161.70 | $0 | 上方盈亏平衡 |
| $165 | -$330 | 买入Call保护 |
| $170 | -$330 | 最大亏损（上方） |

## 5. Greeks 分析

| Greeks | 值 | 说明 |
|--------|---|------|
| Delta | ≈ 0 | 中性策略 |
| Gamma | 负（较大） | ATM Straddle 的 Gamma 很大 |
| Theta | 正（较大） | ATM Straddle 的 Theta 很大 |
| Vega | 负（较大） | ATM Straddle 的 Vega 很大 |

### 关键特征

Iron Butterfly 的 Greeks 绝对值比 Iron Condor 更大：
- **更高的 Theta**：每天赚取更多时间价值
- **更高的 Gamma**：股价变动时 Delta 变化更快，风险更大
- **更高的 Vega**：波动率变动的影响更大

## 6. 与 Iron Condor 的详细对比

| 维度 | Iron Condor | Iron Butterfly |
|------|------------|----------------|
| 卖出行权价 | OTM | ATM |
| 最大盈利位置 | 一个区间 | 一个点（行权价） |
| 最大盈利金额 | 较少 | 较多 |
| 盈利区间宽度 | 较宽 | 较窄 |
| 胜率 | 较高（60-75%） | 较低（40-55%） |
| 风险回报比 | 较好（1:2~1:3） | 较差（1:1~1:2） |
| Gamma 风险 | 较小 | 较大 |
| 适用环境 | 横盘震荡 | 预期股价非常稳定 |

### 选择建议

- **新手**：推荐 Iron Condor，胜率更高，管理更容易
- **经验丰富**：可以在特定情况下使用 Iron Butterfly
- **高 IV 环境**：Iron Condor 更安全
- **低 IV 环境**：Iron Butterfly 可能提供更好的权利金

## 7. 管理和调整

### 7.1 止损规则

- **亏损达到最大亏损的 30-50% 时平仓**
- Iron Butterfly 的 Gamma 风险更大，需要更积极的止损
- 股价偏离 ATM 行权价超过 1-2% 时提高警惕

### 7.2 止盈规则

- **盈利达到最大盈利的 50% 时平仓**
- 由于最大盈利只在一个点，实际操作中很难获得全部最大盈利
- 经验法则：赚取最大盈利的 30-50% 就可以考虑平仓

### 7.3 调整策略

- **向盈利方向 Roll**：如果股价上涨，可以将整个策略上移
- **收窄翅膀**：如果股价在盈利区间内稳定，可以收窄保护期权以增加权利金
- **转为 Iron Condor**：如果股价偏离 ATM，可以调整为 Iron Condor

## 8. 变体策略

### 8.1 Broken Wing Iron Butterfly

翅膀宽度不对称。例如：
- 买入 $140 Put + 卖出 $155 Put/Call + 买入 $170 Call
- 下方翅膀 $15，上方翅膀 $15，但行权价不是完全对称

### 8.2 Jade Lizard

Iron Butterfly 的变体，只有一侧有保护：
- 卖出 ATM Put + 卖出 OTM Call Spread
- 没有下方保护，但总权利金可能超过上方翅膀宽度
- 风险：下方无限（如果没有买 Put 保护）

### 8.3 Twisted Sister

Jade Lizard 的反向：
- 卖出 ATM Call + 卖出 OTM Put Spread
- 风险：上方无限（如果没有买 Call 保护）

## 9. 常见错误

### ❌ 错误一：期望获得全部最大盈利

Iron Butterfly 的最大盈利只在一个点（到期股价 = 行权价），实际上很难精确命中。期望获得全部最大盈利是不现实的。

### ❌ 错误二：止损太慢

Iron Butterfly 的 Gamma 风险比 Iron Condor 大得多。股价一旦偏离 ATM，亏损会快速增加。止损要更果断。

### ❌ 错误三：在低 IV 环境使用

低 IV 时 ATM Straddle 的权利金不够丰厚，Iron Butterfly 的盈利空间太小。

### ❌ 错误四：选择太短期的到期日

Iron Butterfly 的 Gamma 风险在临近到期时更大。建议选择 30-45 天到期，不要选择最后一周。

### ❌ 错误五：不考虑流动性

ATM 期权通常流动性较好，但保护期权（OTM）的流动性可能较差。确保所有四条腿都有合理的 Bid-Ask Spread。

## 10. IBKR 操作说明

### 下单步骤

1. 在 TWS 中打开期权链
2. 选择"策略"模式
3. 选择 Iron Butterfly 策略
4. 设置 ATM 行权价和翅膀宽度
5. 确认净权利金和最大亏损
6. 使用 Limit Order 提交

### 注意事项

- Iron Butterfly 的 ATM 行权价通常选择最接近当前股价的行权价
- 确保四条腿的流动性都足够
- 使用组合订单，不要分四笔单独下单

## 11. 实战技巧

### 11.1 选择入场时机

- **IV 高于历史均值时**：权利金更丰厚
- **预期波动率下降时**：IV crush 对卖方有利
- **财报后**：不确定性消除，股价趋于稳定

### 11.2 翅膀宽度选择

- **$5 间距**：适合股价较低的标的（<$100）
- **$10 间距**：适合股价中等的标的（$100-$300）
- **$15-20 间距**：适合股价较高的标的（>$300）

### 11.3 资金管理

- 每个 Iron Butterfly 的最大亏损不应超过总资金的 2-3%
- 同时持有的 Iron Butterfly 数量不应超过 3-5 个
- 分散在不同标的和到期日

## 12. 小结

| 要点 | 内容 |
|------|------|
| 策略构成 | 卖出 ATM Straddle + 买入 OTM Strangle |
| 适用观点 | 中性/股价非常稳定 |
| 最大盈利 | 净权利金（在 ATM 行权价处） |
| 最大亏损 | 翅膀宽度 - 净权利金（有限） |
| 盈利条件 | 股价接近 ATM 行权价 |
| 核心优势 | 高权利金收入、有限风险 |
| 核心劣势 | 盈利区间窄、Gamma 风险大 |
| Greeks | Delta≈0、高Theta、高Vega、高Gamma |
