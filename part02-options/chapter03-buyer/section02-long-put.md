# 买入看跌期权（Long Put）

## 1. 策略概述

买入看跌期权（Long Put）是期权交易中最直接的看空策略。它的核心思想：

> **你预测股价会下跌，买入 Put 赚取下跌的收益。**

Long Put 的魅力在于：
- **有限风险**：最大亏损 = 支付的权利金
- **高杠杆**：用少量资金获得做空的收益
- **不需要融券**：比直接做空股票更方便

## 2. 建仓要素

| 要素 | 说明 |
|------|------|
| 选择标的 | 看空的股票/ETF |
| 选择行权价 | 决定杠杆和胜率 |
| 选择到期日 | 决定时间价值和有效期 |
| 支付权利金 | 买入成本 |

### 行权价选择

| 行权价 | 权利金 | 杠杆 | 胜率 | 适用场景 |
|--------|-------|------|------|---------|
| ITM（实值） | 贵 | 低 | 高 | 保守看空 |
| ATM（平值） | 中等 | 中等 | ~50% | 标准看空 |
| OTM（虚值） | 便宜 | 高 | 低 | 投机性看空 |

## 3. 盈亏分析

### 收益公式

- **最大亏损** = 支付的权利金（股价涨到行权价以上）
- **最大盈利** = 行权价 - 权利金（股价跌到 0）
- **盈亏平衡点** = 行权价 - 权利金

### ASCII 盈亏图

```
收益
  ^
  |
  |  \                    最大盈利 = K - Premium
  |   \                   （股价跌到0）
  |    \
  |     \
  |      \
─ ─ ─ ─ ─\─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ → 股价
  |        |
  |        |
  |        ↓
  |    盈亏平衡点 = K - Premium
  |
  |  ‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾  ← 最大亏损 = 权利金
  ↓
```

## 4. 实例分析

### 场景设定

- AAPL 当前价格：$155
- 你买入 1 张 AAPL Put，行权价 $150，30天后到期
- 支付权利金：$4.50/股 = $450/张

### 情景分析

| 到期股价 | 期权盈亏 | 总盈亏 | 说明 |
|---------|---------|--------|------|
| $170 | -$450 | -$450 | 股价大涨，期权归零 |
| $160 | -$450 | -$450 | 股价上涨，期权归零 |
| $155 | -$450 | -$450 | 股价不变，期权归零 |
| $150 | -$450 | -$450 | 股价跌到行权价，期权仍归零 |
| $145.50 | $0 | $0 | 盈亏平衡点（$150 - $4.50） |
| $140 | +$550 | +$550 | 股价下跌，盈利 |
| $130 | +$1,550 | +$1,550 | 股价大跌，大赚 |
| $120 | +$2,550 | +$2,550 | 股价暴跌，暴赚 |

**关键数据**：
- 最大亏损：$450（权利金）
- 盈亏平衡点：$150 - $4.50 = $145.50
- 如果 AAPL 跌到 $140，盈利 = ($150 - $140 - $4.50) × 100 = $550
- 收益率：$550 / $450 = 122%

### 与做空股票对比

| 维度 | Long Put（$150 Put） | 做空100股AAPL |
|------|---------------------|--------------|
| 资金占用 | $450（权利金） | $15,500（保证金） |
| 最大亏损 | $450 | 无限 |
| 如果跌到$140 | +$550（122%收益） | +$1,500（9.7%收益） |
| 杠杆 | 高 | 低 |
| 时间限制 | 有（30天） | 无 |

## 5. Greeks 分析

| Greeks | 值 | 说明 |
|--------|---|------|
| Delta | 负（-0.30 到 -0.50） | 股价下跌时期权涨 |
| Gamma | 正 | Delta 变化对买方有利 |
| Theta | 负 | 时间衰减对买方不利 |
| Vega | 正 | 波动率上升对买方有利 |

### 与 Long Call 的 Greeks 对比

| Greeks | Long Call | Long Put |
|--------|----------|----------|
| Delta | 正 | 负 |
| Gamma | 正 | 正 |
| Theta | 负 | 负 |
| Vega | 正 | 正 |

**关键洞察**：Long Put 和 Long Call 的 Greeks 在绝对值上相似，只是 Delta 符号相反。两者的 Theta 和 Vega 特征相同——都是时间衰减的受害者，都受益于波动率上升。

## 6. 作为保险（Protective Put）

Long Put 最经典的用途之一是**为股票持仓提供保险**。

### 策略构成

> **持有股票 + 买入 Put = Protective Put**

这就像为你的股票买了一份"保险"：
- 如果股价上涨，你享受上涨收益（减去保险费）
- 如果股价下跌，Put 的盈利弥补股票的损失

### 实例

- 你持有 100 股 AAPL，成本 $155
- 你买入 1 张 $150 Put，权利金 $4.50
- 保险成本 = $450 / $15,500 = 2.9%

| 到期 AAPL | 股票盈亏 | Put 盈亏 | 总盈亏 |
|----------|---------|---------|--------|
| $170 | +$1,500 | -$450 | +$1,050 |
| $160 | +$500 | -$450 | +$50 |
| $155 | $0 | -$450 | -$450 |
| $150 | -$500 | -$450 | -$950 |
| $140 | -$1,500 | +$550 | -$950 |
| $130 | -$2,500 | +$1,550 | -$950 |

**关键观察**：
- 最大亏损被锁定在 -$950（$500 股票亏损 + $450 权利金）
- 如果没有买 Put，股价跌到 $130 时亏损 $2,500
- Protective Put 限制了下行风险，但代价是减少了上行收益

### Collar 策略

如果 Protective Put 的成本太高，可以同时卖出 Covered Call 来补贴：

> **持有股票 + 买入 Put + 卖出 Call = Collar**

- 卖出 Call 收取的权利金可以部分或全部抵消买入 Put 的成本
- 代价是限制了上涨空间

## 7. 与做空股票的对比

| 维度 | Long Put | Short Stock |
|------|----------|-------------|
| 资金占用 | 少（权利金） | 多（保证金） |
| 最大亏损 | 有限（权利金） | 无限 |
| 杠杆 | 高 | 低 |
| 时间限制 | 有（到期日） | 无 |
| 融券成本 | 无 | 可能有 |
| 分红影响 | 无 | 需要支付分红 |
| 适用场景 | 短期看空 | 长期看空 |

### 选择建议

- **短期看空（1个月内）**：Long Put 更合适
- **长期看空（3个月+）**：Short Stock 或 LEAPS Put
- **资金有限**：Long Put（杠杆高）
- **风险厌恶**：Long Put（有限风险）

## 8. 常见错误

### ❌ 错误一：买太虚值的 Put

Delta 0.10 的 Put 虽然便宜，但需要股价大幅下跌才能盈利。大部分时间这些期权会归零。

### ❌ 错误二：买太短期的 Put

一周到期的 Put 时间衰减极快，即使方向对了，也可能因为时间不够而亏损。

### ❌ 错误三：忽视隐含波动率

IV 过高时买入 Put 非常贵。即使股价下跌，IV 下降（IV Crush）可能抵消大部分盈利。

### ❌ 错误四：不止损

Long Put 的最大亏损是权利金，但如果股价持续上涨，不要继续持有归零的期权。

### ❌ 错误五：财报后买入 Put

财报后 IV 通常会大幅下降（IV Crush），此时买入 Put 成本高且容易被 IV 下降侵蚀。

## 9. IBKR 操作说明

### 下单步骤

1. 在 TWS 中搜索标的股票
2. 选择"交易期权"→"买入"→"Put"
3. 选择行权价和到期日
4. 使用 Limit Order 设置买入价格
5. 提交订单

### 注意事项

- Long Put 只需支付权利金，不需要额外保证金
- 确保账户有足够资金支付权利金
- 可以使用"止损单"自动止损

## 10. 小结

| 要点 | 内容 |
|------|------|
| 策略构成 | 买入 Put |
| 适用观点 | 看空 / 对冲 |
| 最大亏损 | 权利金 |
| 最大盈利 | 行权价 - 权利金（股价跌到0） |
| 盈亏平衡 | 行权价 - 权利金 |
| 核心优势 | 有限风险、高杠杆、不需融券 |
| 核心劣势 | 时间衰减、需要股价"快速"下跌 |
| Greeks | 负Delta、正Gamma、负Theta、正Vega |
