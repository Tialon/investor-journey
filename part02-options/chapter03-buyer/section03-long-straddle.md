# 买入跨式策略（Long Straddle & Long Strangle）

> 当你确信市场即将剧烈波动，却又无法判断方向时，买入跨式策略是期权交易者手中最优雅的武器之一。

## 一、什么是买入跨式策略 (Long Straddle)

### 1.1 策略定义

**买入跨式策略 (Long Straddle)** 是指同时买入相同标的、相同到期日、相同行权价的一个看涨期权 (Call) 和一个看跌期权 (Put)。通常选择平值期权 (At-the-Money, ATM) 作为构建基础。

**构建方式：**
- 买入 1 份 ATM Call（行权价 = 当前标的价格）
- 买入 1 份 ATM Put（行权价 = 当前标的价格）

例如，假设 AAPL 当前价格为 $180：
- 买入 1 份 AAPL 180 Call，支付权利金 $5.00
- 买入 1 份 AAPL 180 Put，支付权利金 $4.80
- **总成本（净权利金支出）= $5.00 + $4.80 = $9.80**

这个 $9.80 就是你在这笔交易中可能损失的最大金额（每份合约对应 100 股，则最大亏损为 $980）。

### 1.2 核心逻辑

Long Straddle 的核心逻辑非常直白：**你不需要知道方向，只需要知道会有大动作。** 无论标的价格大涨还是大跌，只要波动幅度足够大，策略就能盈利。这使得它成为事件驱动型交易的首选工具。

从期权定价的角度来看，买入跨式本质上是在做多波动率 (Long Volatility)。你支付的权利金中包含了时间价值 (Time Value)，而你期望在期权到期前，标的的实际波动能够超越市场已经隐含的预期波动。

## 二、什么是买入宽跨式策略 (Long Strangle)

### 2.1 策略定义

**买入宽跨式策略 (Long Strangle)** 是指同时买入相同标的、相同到期日、但不同行权价的一个虚值看涨期权 (OTM Call) 和一个虚值看跌期权 (OTM Put)。

**构建方式：**
- 买入 1 份 OTM Call（行权价高于当前标的价格）
- 买入 1 份 OTM Put（行权价低于当前标的价格）

继续用 AAPL 的例子，假设当前价格 $180：
- 买入 1 份 AAPL 185 Call（行权价比现价高 $5），支付权利金 $2.50
- 买入 1 份 AAPL 175 Put（行权价比现价低 $5），支付权利金 $2.30
- **总成本 = $2.50 + $2.30 = $4.80**

### 2.2 Straddle 与 Strangle 的关键区别

| 对比维度 | Long Straddle | Long Strangle |
|---------|---------------|---------------|
| 行权价选择 | 相同（通常 ATM） | 不同（两个 OTM） |
| 建仓成本 | 较高 | 较低 |
| 盈亏平衡间距 | 较窄 | 较宽 |
| 需要的波动幅度 | 较小即可盈利 | 需要更大的波动才能盈利 |
| 最大亏损 | 较大（成本更高） | 较小（成本更低） |
| 最大盈利 | 理论上无限 | 理论上无限 |

简单来说：**Strangle 更便宜，但需要更大幅度的波动才能跨过盈亏平衡点；Straddle 更贵，但相对温和的波动就能开始盈利。**

## 三、盈亏分析

### 3.1 Long Straddle 盈亏结构

**最大亏损 (Maximum Loss)：** 等于支付的总权利金。当标的到期价格恰好等于行权价时，Call 和 Put 同时归零，你损失全部权利金。

**最大盈利 (Maximum Profit)：** 理论上无限。标的价格上涨没有上限，下跌最多到零。

**盈亏平衡点 (Break-Even Points)：** 两个盈亏平衡点分别是：
- **上方盈亏平衡点 = 行权价 + 总权利金**
- **下方盈亏平衡点 = 行权价 - 总权利金**

以上述 AAPL Straddle 为例：
- 上方 B-E-P = $180 + $9.80 = **$189.80**
- 下方 B-E-P = $180 - $9.80 = **$170.20**

也就是说，AAPL 必须涨到 $189.80 以上，或者跌到 $170.20 以下，这个策略才开始盈利。盈亏平衡点之间的距离为 $19.60，这大约是标的当前价格的 10.9%。

**Long Straddle 盈亏图（到期日）：**

```
  盈利(P/L)
    |        \           /
    |         \         /
    |          \       /
    |           \     /
    |            \   /
    |             \ /
  0 |------/------*------\------
    |     /   (180)      \
    |    /                \
    |   /                  \
-9.80|--*--------------------*---
    | 170.20              189.80
    |
    +---------------------------------> 标的价格
        下方BEP   行权价   上方BEP
```

图中可以清楚看到：
- 当价格在 $170.20 到 $189.80 之间时，策略处于亏损状态
- 当价格跌破 $170.20 或涨破 $189.80 时，策略开始盈利
- 最大亏损在 $180 处，为 $9.80
- 向上和向下的盈利空间在理论上都是无限的

### 3.2 Long Strangle 盈亏结构

**最大亏损：** 等于支付的总权利金。当标的到期价格落在两个行权价之间时，Call 和 Put 同时归零。

**最大盈利：** 理论上无限。

**盈亏平衡点：**
- **上方 B-E-P = OTM Call 行权价 + 总权利金**
- **下方 B-E-P = OTM Put 行权价 - 总权利金**

以上述 AAPL Strangle 为例：
- 上方 B-E-P = $185 + $4.80 = **$189.80**
- 下方 B-E-P = $175 - $4.80 = **$170.20**

有趣的是，在这个例子中两个策略的盈亏平衡点完全相同，但 Strangle 的最大亏损只有 $4.80，远低于 Straddle 的 $9.80。当然，这是理想化的例子——实际交易中两者盈亏平衡点通常不同，Strangle 的盈亏平衡间距一般更大。

**Long Strangle 盈亏图（到期日）：**

```
  盈利(P/L)
    |         \       /
    |          \     /
    |           \   /
    |            \ /
  0 |---/--------*--------\----
    |  /     (175-185)     \
    | /                     \
    |/                       \
-4.80*------------------------*--
    |170.20                 189.80
    |
    +---------------------------------> 标的价格
      下方BEP  Put行权  Call行权  上方BEP
```

对比两张图可以看到：
- Strangle 的"亏损区间"（两个行权价之间的平台区域）更宽
- 但 Strangle 每亏损一单位的金额更少
- Strangle 的"盈利起点"距离当前价格更远

## 四、适用场景

### 4.1 最佳使用场景

买入跨式策略在以下场景中表现最为出色：

**（1）财报发布 (Earnings Release)**

公司季度财报是使用 Long Straddle/Strangle 最经典的场景。财报前，市场通常会对公司业绩存在较大不确定性，而一旦财报超出或低于预期，股价往往会出现剧烈波动。

例如：2024 年某季度，META 财报后股价单日涨幅超过 20%，如果你在财报前建立了 Long Straddle，无论你是否猜到了方向，都能获得可观的利润。

**（2）FDA 药物审批决定**

生物医药公司的新药审批是另一个典型场景。FDA 的批准或拒绝决定往往在一天之内让股价翻倍或腰斩。

**（3）重大政策事件**

美联储利率决议 (FOMC)、大选结果公布、重要经济数据发布（如非农就业数据）等，都可能引发市场的大幅波动。

**（4）并购传闻**

当市场传闻某公司可能被收购时，股价可能大涨（如果传闻属实）或大跌（如果传闻被否认），Long Straddle 可以捕捉这种不确定性。

**（5）技术面关键突破**

当标的在关键技术位（如长期支撑/阻力位）附近盘整时，突破方向不明但突破幅度可能很大。

### 4.2 不适合的场景

**（1）横盘震荡市**

如果你认为标的将在窄幅区间内波动，不要使用此策略。时间价值的衰减会持续侵蚀你的头寸。

**（2）隐含波动率过高时**

当 IV 处于历史高位时，期权权利金非常昂贵，即使标的出现了大幅波动，你也可能因为支付了过高的权利金而无法盈利。

**（3）临近到期日**

Theta 衰减在到期前最后 30 天加速，如果购买太短期的跨式，时间衰减的影响会非常剧烈。

## 五、成本分析

### 5.1 Straddle vs Strangle 的成本比较

Long Straddle 的成本几乎总是高于 Long Strangle，原因很简单：ATM 期权的时间价值最大，而 OTM 期权的时间价值较小。

以 AAPL 当前价格 $180 为例：

| 策略组合 | Call 成本 | Put 成本 | 总成本 | 上方BEP | 下方BEP | 盈利区间宽度 |
|---------|-----------|----------|--------|---------|---------|------------|
| Straddle (180/180) | $5.00 | $4.80 | $9.80 | $189.80 | $170.20 | $19.60 (10.9%) |
| Strangle (175/185) | $2.50 | $2.30 | $4.80 | $189.80 | $170.20 | $19.60 (10.9%) |
| Strangle (170/190) | $1.20 | $1.10 | $2.30 | $192.30 | $167.70 | $24.60 (13.7%) |

### 5.2 成本与盈利概率的关系

这是一个非常关键的概念：**你支付的权利金越多，需要的波动越小，但最大亏损越大；你支付的权利金越少，需要的波动越大，但最大亏损越小。**

换句话说：
- **Straddle**：高成本、窄盈利区间、较低的盈亏平衡门槛
- **Strangle**：低成本、宽盈利区间、较高的盈亏平衡门槛

实际交易中，你需要根据对波动幅度的预估来选择合适的策略。如果你预期波动在 8-10% 左右，Straddle 可能更合适；如果你预期波动会达到 15% 以上，Strangle 的性价比更高。

### 5.3 价差宽度的选择

对于 Strangle，选择多宽的行权价是一个关键决策：

- **窄 Strangle（如 ±$5）**：成本较高，接近 Straddle 的效果，但仍然比 Straddle 便宜
- **中等 Strangle（如 ±$10）**：平衡成本和盈亏平衡点，通常是较好的折中选择
- **宽 Strangle（如 ±$20）**：成本很低，但需要非常大的波动才能盈利

一般建议选择 **Delta 绝对值在 0.15-0.25 之间** 的行权价作为 Strangle 的构建基础，这样可以在成本和盈利概率之间取得较好的平衡。

## 六、Greeks 分析

理解 Long Straddle/Strangle 的 Greeks 特征对于有效管理这类头寸至关重要。

### 6.1 Delta

**Long Straddle 在建立时的 Delta 接近于零。** 因为你同时持有一个 Call（Delta 为正）和一个 Put（Delta 为负），两者大小相近、方向相反，相互对冲。

- Long Straddle（ATM）：Delta ≈ 0（因为 ATM Call Delta ≈ +0.50，ATM Put Delta ≈ -0.50）
- Long Strangle（OTM）：Delta ≈ 0（但 OTM Call Delta 绝对值和 OTM Put Delta 绝对值均小于 0.50）

随着标的价格变动，Delta 会动态变化：
- 标的价格上涨时，Delta 变为正值（Call 的 Delta 增大，Put 的 Delta 减小），头寸整体呈现多头倾向
- 标的价格下跌时，Delta 变为负值（Put 的 Delta 绝对值增大，Call 的 Delta 减小），头寸整体呈现空头倾向

这种"方向自适应"的特性是 Long Straddle 的魅力所在——它自动跟随趋势方向。

### 6.2 Gamma

**Long Straddle/Strangle 拥有很高的正 Gamma (Positive Gamma)。** 这是策略最重要的 Greeks 特征之一。

高 Gamma 意味着：
- 当标的上涨时，Delta 会快速增加（变得更正），让你更多地参与上涨
- 当标的下跌时，Delta 会快速减少（变得更负），让你更多地参与下跌
- 标的价格离行权价越近（ATM 附近），Gamma 最大

对于 Straddle 而言，最大 Gamma 出现在标的恰好等于行权价时。这意味着一旦标的价格开始偏离行权价，你的头寸就会迅速获得方向性暴露，速度越来越快。

### 6.3 Vega

**Long Straddle/Strangle 拥有很高的正 Vega (Positive Vega)。** 这是策略的另一个核心特征。

高 Vega 意味着：
- **隐含波动率 (Implied Volatility, IV) 上升时，策略价值增加**
- **IV 下降时，策略价值减少**

这一点在实际交易中极为重要。很多初学者只关注标的涨跌方向，却忽略了 IV 变化对跨式策略的巨大影响。

Vega 在 ATM 期权上最大，因此：
- Straddle（ATM + ATM）的 Vega 总量 > Strangle（OTM + OTM）的 Vega 总量
- 这也解释了为什么 Straddle 更贵——它对 IV 变动更敏感

### 6.4 Theta

**Long Straddle/Strangle 拥有负 Theta (Negative Theta)。** 这是策略的主要成本。

负 Theta 意味着：
- **每一天过去，如果没有足够的价格波动，头寸价值就会减少**
- Theta 衰减在期权临近到期时加速（尤其在最后 30 天）

对于 Straddle 而言，由于两个期权都是 ATM，Theta 衰减较快。对于 Strangle 而然，由于 OTM 期权的 Theta 相对较小，衰减速度略慢。

**核心权衡关系：** 你用 Theta（时间衰减）的成本去购买 Gamma（方向跟随）和 Vega（波动率暴露）的收益。如果标的波动足够大，Gamma 的收益超过 Theta 的损耗；如果标的不动，Theta 就会慢慢把你的权利金吃掉。

### 6.5 Greeks 汇总表

| Greek | 符号 | Long Straddle 的特征 | 含义 |
|-------|------|---------------------|------|
| Delta | Δ | ≈ 0（初始） | 初始方向中性 |
| Gamma | Γ | 高正值 | 价格变动时 Delta 变化快 |
| Vega | ν | 高正值 | 波动率上升有利 |
| Theta | θ | 高负值 | 时间衰减是主要成本 |

## 七、波动率的角色

波动率 (Volatility) 在买入跨式策略中扮演着决定性的角色。可以说，**理解波动率比理解方向更重要。**

### 7.1 隐含波动率 (IV) 的影响

当你买入一个 Straddle 时，你支付的权利金中很大一部分是时间价值，而时间价值主要由隐含波动率决定。

**核心问题：你买入时的 IV 是高还是低？**

- **低 IV 环境下买入**：权利金便宜，你需要的波动幅度较小就能盈利，策略期望值更高
- **高 IV 环境下买入**：权利金昂贵，即使标的出现了大幅波动，你也可能因为 IV crush（波动率骤降）而亏损

### 7.2 IV Crush（波动率骤降）

IV Crush 是 Long Straddle/Strangle 交易者最大的敌人之一。它的典型场景是：

**财报前 IV 很高**（因为市场预期会有大波动）→ **财报发布后 IV 急剧下降**（因为不确定性消除）→ **即使股价大幅波动，期权价值也可能因为 IV 下降而缩水**

这就是为什么有些交易者在财报前买入 Straddle，即使财报后股价确实大幅波动了，却依然亏损——因为 IV crush 吃掉了本应属于你的利润。

### 7.3 如何判断 IV 是否合理

**（1）历史波动率对比 (HV vs IV)**

将当前的 IV 与标的过去 30 天、60 天、90 天的实际波动率 (Historical Volatility, HV) 进行对比。如果 IV 远高于 HV，说明期权相对昂贵。

**（2）IV 百分位 (IV Percentile) 或 IV 排名 (IV Rank)**

查看当前 IV 在过去一年中的百分位排名：
- IV 百分位 < 30%：相对便宜，有利于买入策略
- IV 百分位 > 70%：相对昂贵，不利于买入策略

**（3）预期波动幅度 (Expected Move)**

利用期权的定价反推市场预期的波动幅度。例如，如果 ATM Straddle 的总权利金是 $10，对应标的价格 $100，市场隐含的预期波动就是 10%。如果你认为实际波动会超过这个幅度，策略值得建立。

### 7.4 实际操作建议

**买入跨式策略的黄金法则：在 IV 低的时候买入，在 IV 高的时候卖出。** 这听起来简单，但实际操作中需要纪律。

具体的筛选标准：
- IV 百分位低于 30% 时考虑建立多头跨式
- IV 百分位高于 70% 时避免建立多头跨式
- 特别关注 IV 相对 HV 的折价/溢价情况

## 八、实例分析：AAPL 财报前的 Straddle 交易

### 8.1 场景设定

假设现在是 2025 年 1 月下旬，AAPL 将在 1 月 30 日盘后发布 2025 财年第一季度财报。

**当前市场数据：**
- AAPL 当前股价：$180.00
- 财报日期：1 月 30 日（距离到期约 3 周）
- 当前 AAPL 的 30 天 IV：28%
- AAPL 的 30 天 HV：22%
- IV 百分位：35%（中偏低）
- 2 月 21 日到期的期权链数据如下：

| 行权价 | Call 权利金 | Put 权利金 | Straddle 成本 | Delta (Call/Put) |
|--------|------------|------------|--------------|-----------------|
| $170 (OTM Put) | — | $1.80 | — | — / -0.15 |
| $175 (OTM Put) | — | $3.20 | — | — / -0.28 |
| $180 (ATM) | $4.50 | $4.20 | $8.70 | +0.52 / -0.48 |
| $185 (OTM Call) | $2.40 | — | — | +0.25 / — |
| $190 (OTM Call) | $1.10 | — | — | +0.12 / — |

### 8.2 策略构建

**方案 A：Long Straddle (180/180)**
- 买入 1 份 2/21 180 Call @ $4.50
- 买入 1 份 2/21 180 Put @ $4.20
- 总成本 = $8.70（每份合约 $870）
- 上方 B-E-P = $180 + $8.70 = **$188.70**
- 下方 B-E-P = $180 - $8.70 = **$171.30**
- 需要 AAPL 波动约 **4.8%** 才能到盈亏平衡

**方案 B：Long Strangle (175/185)**
- 买入 1 份 2/21 185 Call @ $2.40
- 买入 1 份 2/21 175 Put @ $3.20
- 总成本 = $5.60（每份合约 $560）
- 上方 B-E-P = $185 + $5.60 = **$190.60**
- 下方 B-E-P = $175 - $5.60 = **$169.40**
- 需要 AAPL 波动约 **5.9%** 才能到盈亏平衡

**方案 C：宽 Strangle (170/190)**
- 买入 1 份 2/21 190 Call @ $1.10
- 买入 1 份 2/21 170 Put @ $1.80
- 总成本 = $2.90（每份合约 $290）
- 上方 B-E-P = $190 + $2.90 = **$192.90**
- 下方 B-E-P = $170 - $2.90 = **$167.10**
- 需要 AAPL 波动约 **7.2%** 才能到盈亏平衡

### 8.3 盈利情景分析

假设你在 1 月 27 日建仓，持有到财报发布后（1 月 31 日）。

**情景 1：AAPL 财报大超预期，股价涨至 $195**

| 策略 | Call 价值 | Put 价值 | 头寸总价值 | 盈亏 |
|------|----------|----------|-----------|------|
| Straddle (180/180) | $15.00 | $0.50 | $15.50 | +$6.80 (+78%) |
| Strangle (175/185) | $10.00 | $0.20 | $10.20 | +$4.60 (+82%) |
| 宽 Strangle (170/190) | $5.00 | $0.05 | $5.05 | +$2.15 (+74%) |

注意：Strangle 的百分比收益更高，虽然绝对收益低于 Straddle。

**情景 2：AAPL 财报不及预期，股价跌至 $165**

| 策略 | Call 价值 | Put 价值 | 头寸总价值 | 盈亏 |
|------|----------|----------|-----------|------|
| Straddle (180/180) | $0.30 | $15.00 | $15.30 | +$6.60 (+76%) |
| Strangle (175/185) | $0.10 | $10.00 | $10.10 | +$4.50 (+80%) |
| 宽 Strangle (170/190) | $0.02 | $5.00 | $5.02 | +$2.12 (+73%) |

**情景 3：AAPL 财报符合预期，股价小幅波动至 $182**

| 策略 | Call 价值 | Put 价值 | 头寸总价值 | 盈亏 |
|------|----------|----------|-----------|------|
| Straddle (180/180) | $5.50 | $2.50 | $8.00 | -$0.70 (-8%) |
| Strangle (175/185) | $3.00 | $1.20 | $4.20 | -$1.40 (-25%) |
| 宽 Strangle (170/190) | $1.50 | $0.50 | $2.00 | -$0.90 (-31%) |

**情景 4：最坏情况——IV Crush 叠加小幅波动**

这是最需要注意的场景。假设财报后 IV 从 28% 降至 18%：
- 即使 AAPL 移动到 $185，由于 IV crush，期权价值可能不升反降
- Straddle 可能亏损 $2-3，Strangle 可能亏损 $1-2

### 8.4 实际操作建议

基于以上分析：
1. **IV 百分位 35% 是可以接受的买入环境**，不算特别便宜但也不贵
2. 如果你认为 AAPL 财报后波动可能在 5-8%，选择 **Straddle** 更安全
3. 如果你认为波动会超过 10%，选择 **Strangle** 性价比更高
4. **注意仓位管理**：不要把所有资金都放在一次财报交易上，建议单笔交易不超过总资金的 3-5%
5. **考虑提前建仓**：在财报前 5-7 天建仓，可以在 IV 还未大幅上升时获得更便宜的权利金

## 九、变体策略

### 9.1 Strap 策略（偏多跨式）

**Strap** 是 Long Straddle 的变体，通过购买不同数量的 Call 和 Put 来表达方向性偏好。

**构建方式：**
- 买入 2 份 ATM Call
- 买入 1 份 ATM Put

**特点：**
- 初始 Delta 为正值（偏多头方向）
- 上涨时盈利更快、更大（因为有 2 份 Call）
- 下跌时也需要更大的跌幅才能盈利
- 总成本比 Straddle 更高

**适用场景：** 你认为标的会有大幅波动，但稍微倾向于看多。例如，你认为财报可能超预期，但又不想放弃看空的保护。

**盈亏平衡点：**
- 上方 B-E-P = 行权价 + 总权利金 / 2（因为上涨时有 2 份 Call 发挥作用）
- 下方 B-E-P = 行权价 - 总权利金

### 9.2 Strip 策略（偏空跨式）

**Strip** 是 Strap 的镜像，通过购买更多 Put 来表达看空偏好。

**构建方式：**
- 买入 1 份 ATM Call
- 买入 2 份 ATM Put

**特点：**
- 初始 Delta 为负值（偏空头方向）
- 下跌时盈利更快、更大（因为有 2 份 Put）
- 上涨时也需要更大的涨幅才能盈利
- 总成本比 Straddle 更高

**适用场景：** 你认为标的会有大幅波动，但稍微倾向于看空。例如，在市场恐慌情绪升温时，下跌的幅度和速度往往超过上涨。

**盈亏平衡点：**
- 上方 B-E-P = 行权价 + 总权利金
- 下方 B-E-P = 行权价 - 总权利金 / 2（因为下跌时有 2 份 Put 发挥作用）

### 9.3 Strap 与 Strip 的比较

```
  Strap (偏多)              Strip (偏空)
  盈利(P/L)                 盈利(P/L)
    |      /  \               |      \  /
    |     /    \              |       \/
    |    /      \             |       /
  0 |---/--------\--        0 |------/\------
    |  /          \           |     /  \
    | /            \          |    /    \
-成本|*              *     -成本|---*------*---
    +-> 标的价格              +-> 标的价格
   上涨盈利更快更大          下跌盈利更快更大
```

选择 Strap 还是 Strip，取决于你对波动方向的倾向程度。如果完全不确定方向，就用标准的 Straddle。

## 十、风险管理与注意事项

### 10.1 仓位管理

- 单笔 Straddle/Strangle 交易的总成本不应超过总交易资金的 **3-5%**
- 同时持有的多头跨式头寸不应超过总资金的 **10-15%**
- 避免在同一板块或高度相关的标的上同时建立多个跨式头寸

### 10.2 止损与止盈

虽然 Long Straddle/Strangle 的最大亏损有限（等于总权利金），但并不意味着你应该持有到到期。

**止损建议：**
- 当总亏损达到初始权利金的 **50%** 时，考虑止损退出
- 特别是在事件发生前如果预期发生变化（如事件被推迟或取消），及时退出

**止盈建议：**
- 当盈利达到初始权利金的 **100%**（翻倍）时，可以考虑平掉一半头寸锁定利润
- 保留另一半头寸继续持有，以捕捉可能的更大波动
- 事件发生后（如财报公布），无论盈亏，考虑立即平仓——此时 IV crush 通常会开始侵蚀剩余价值

### 10.3 常见错误

1. **在 IV 最高时买入**：财报前几天 IV 通常快速上升，此时买入是最贵的。提前建仓或在 IV 相对较低时入场。
2. **忽略 IV Crush**：很多交易者只看方向不看 IV，导致财报后即使股价大幅波动也亏钱。
3. **持有到期**：期权的时间价值在最后几天衰减最快，事件发生后应该及时获利了结或止损。
4. **过度交易**：不是每份财报都适合用 Straddle，只有当 IV 合理且预期波动幅度足够大时才值得交易。
5. **忽略交易成本**：Straddle/Strangle 涉及同时买入两个期权，交易佣金和买卖价差 (Bid-Ask Spread) 会增加实际成本。

## 十一、小结

买入跨式策略是期权交易者工具箱中最强大的波动率交易工具之一。让我们回顾核心要点：

1. **Long Straddle**：买入相同行权价的 Call 和 Put，成本较高，需要的波动较小
2. **Long Strangle**：买入不同行权价的 OTM Call 和 Put，成本较低，需要的波动较大
3. **核心特征**：Delta 中性、高 Gamma、高 Vega、负 Theta
4. **盈利条件**：标的出现足够大幅度的波动（超过隐含的预期波动）
5. **核心风险**：时间衰减 (Theta) 和波动率骤降 (IV Crush)
6. **最佳时机**：IV 百分位较低时，配合高不确定性事件（财报、FDA 审批等）
7. **变体策略**：Strap（偏多）和 Strip（偏空）可以表达方向性偏好

**记住一个公式：**

> Long Straddle/Strangle 的盈利 = 实际波动带来的收益 - 时间衰减的成本 - 波动率下降的损失

只有当"实际波动带来的收益"足够大，能够覆盖后两者的损失时，策略才盈利。
