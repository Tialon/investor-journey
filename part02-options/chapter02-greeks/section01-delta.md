# Delta (δ)

## 1. 什么是 Delta

你买入一份 AAPL 的看涨期权，行权价 150，AAPL 当前价格 155。如果 AAPL 股价从 155 涨到 156（涨了 1 美元），你的期权价格会涨多少？

答案是：**大约涨 0.60 美元**。

这个"大约 0.60"就是 Delta——它衡量的是**期权价格对标的资产价格变动的敏感度**。

> **Delta (δ)**：标的资产价格变动 1 美元时，期权价格的变动量。

## 2. Delta 的数学定义

$$\Delta = \frac{\partial V}{\partial S}$$

其中：
- $V$ = 期权价格
- $S$ = 标的资产价格

用差分近似：

$$\Delta \approx \frac{\Delta V}{\Delta S} = \frac{\text{期权价格变动}}{\text{标的资产价格变动}}$$

## 3. 不同头寸的 Delta 值

### 3.1 基本规则

| 头寸 | Delta 范围 | 含义 |
|------|-----------|------|
| Long Call (买入看涨) | 0 到 +1.0 | 股价涨，期权涨 |
| Short Call (卖出看涨) | 0 到 -1.0 | 股价涨，期权跌（卖方亏） |
| Long Put (买入看跌) | -1.0 到 0 | 股价涨，期权跌 |
| Short Put (卖出看跌) | +1.0 到 0 | 股价涨，期权涨（卖方赚） |

### 3.2 直觉理解

- **Long Call**：你赌股价涨，所以 Delta 为正——股价涨你赚钱
- **Long Put**：你赌股价跌，所以 Delta 为负——股价涨你亏钱
- **Short Call**：你是 Long Call 的对手方，Delta 符号相反
- **Short Put**：你是 Long Put 的对手方，Delta 符号相反

## 4. Delta 与 Moneyness 的关系

Delta 的大小与期权的实虚值程度密切相关：

| Moneyness | Call Delta | Put Delta | 说明 |
|-----------|-----------|----------|------|
| 深度实值 (Deep ITM) | ≈ +1.0 | ≈ -1.0 | 几乎等同于持有/做空股票 |
| 实值 (ITM) | 0.5 ~ 1.0 | -1.0 ~ -0.5 | 内在价值占主导 |
| 平值 (ATM) | ≈ 0.5 | ≈ -0.5 | 50/50 概率 |
| 虚值 (OTM) | 0 ~ 0.5 | -0.5 ~ 0 | 主要是时间价值 |
| 深度虚值 (Deep OTM) | ≈ 0 | ≈ 0 | 几乎不可能变为实值 |

### 关键洞察

**ATM 期权的 Delta 约为 ±0.5**，这意味着：
- 股价涨 1 美元，ATM Call 大约涨 0.50 美元
- 这个 0.50 也可以近似理解为"该期权到期时变为实值的概率约为 50%"

> ⚠️ 注意：Delta 并非严格的概率，但在实际交易中可以这样近似理解。

## 5. Delta 的三种含义

### 5.1 价格敏感度

Delta = 0.60 意味着：标的涨 $1，期权大约涨 $0.60。

### 5.2 到期时为实值的近似概率

Delta ≈ 0.30 的 Call，到期时大约有 30% 的概率会变为实值（ITM）。

这个等价关系在实际交易中非常有用：
- 卖 Delta 0.30 的 Call，大约有 70% 的概率到期时为虚值（OTM），卖方获利
- 卖 Delta 0.10 的 Call，大约有 90% 的概率到期时为虚值，但权利金很少

### 5.3 等效股票头寸

Delta 可以用来计算期权头寸等效于持有多少股股票：

**等效股票数 = Delta × 合约乘数 × 合约数量**

例如：
- 持有 1 张 AAPL Call，Delta = 0.60，合约乘数 = 100
- 等效股票 = 0.60 × 100 × 1 = **60 股**

这意味着 1 张 Delta = 0.60 的 Call，其价格变动等同于持有 60 股 AAPL。

## 6. Delta 对冲 (Delta Hedging)

Delta 对冲是专业交易员的核心技术之一。目标是让整个投资组合的 Delta 为零（Delta Neutral），这样组合不受标的价格小幅变动的影响。

### 例子

你卖了 10 张 AAPL Call，每张 Delta = 0.40。
- 期权头寸 Delta = -0.40 × 100 × 10 = **-400**
- 你需要买入 **400 股 AAPL** 来对冲（400 × Delta 1.0 = +400）
- 组合总 Delta = -400 + 400 = **0**（Delta Neutral）

当股价变动时，Delta 也会变化（这由 Gamma 衡量），所以需要动态调整对冲头寸——这就是**动态对冲（Dynamic Hedging）**。

## 7. 实例分析

假设 AAPL 当前价格 = $155

| 行权价 | 类型 | 期权价格 | Delta | 到期ITM概率(近似) |
|--------|------|---------|-------|-----------------|
| 140 | Call | $17.50 | 0.85 | 85% |
| 145 | Call | $13.80 | 0.78 | 78% |
| 150 | Call | $10.50 | 0.68 | 68% |
| 155 | Call | $7.60 | 0.55 | 55% |
| 160 | Call | $5.20 | 0.42 | 42% |
| 165 | Call | $3.40 | 0.30 | 30% |
| 170 | Call | $2.10 | 0.20 | 20% |
| 175 | Call | $1.30 | 0.12 | 12% |

观察规律：
- 行权价越低（越深度实值），Delta 越接近 1
- ATM（155行权价）Delta ≈ 0.55
- 行权价越高（越深度虚值），Delta 越接近 0

## 8. Delta 的变化规律

### 8.1 随股价变化

- 股价上涨 → Call Delta 增大（趋向 +1），Put Delta 增大（趋向 0）
- 股价下跌 → Call Delta 减小（趋向 0），Put Delta 减小（趋向 -1）

### 8.2 随时间变化

- 随着到期日临近：
  - ITM 期权 Delta 趋向 ±1.0（越来越像股票）
  - OTM 期权 Delta 趋向 0（越来越可能归零）
  - ATM 期权 Delta 保持在 ±0.5 附近

### 8.3 随波动率变化

- 波动率增大 → Delta 趋向 ±0.5（ATM化效应）
- 波动率减小 → ITM Delta 趋向 ±1.0，OTM Delta 趋向 0

## 9. 交易中的 Delta 应用

### 9.1 选择入场 Delta

| 策略 | 常用Delta范围 | 特点 |
|------|-------------|------|
| 保守买 Call | 0.60-0.70 | 胜率较高，杠杆较低 |
| 标准买 Call | 0.40-0.60 | 平衡胜率和杠杆 |
| 投机买 Call | 0.10-0.30 | 高杠杆，低胜率 |
| 卖 Covered Call | 0.20-0.40 | 平衡权利金收入和被行权风险 |

### 9.2 组合 Delta 管理

- **看多市场**：组合 Delta > 0
- **看空市场**：组合 Delta < 0
- **中性/不确定**：组合 Delta ≈ 0

### 9.3 方向性交易简化

用 Delta 可以快速估算：
- 买入 1 张 Delta 0.50 的 Call ≈ 做多 50 股
- 买入 1 张 Delta -0.50 的 Put ≈ 做空 50 股

## 10. Delta 的局限性

1. **Delta 不是常数**：它随股价、时间、波动率变化（由 Gamma 和其他 Greeks 衡量）
2. **Delta 对冲需要频繁调整**：股价大幅变动后，原来的对冲不再有效
3. **概率只是近似**：Delta ≠ 真实概率，尤其在极端情况下
4. **不考虑跳跃风险**：Delta 假设价格连续变动，但现实中可能出现跳空（如财报后）

## 11. 小结

| 要点 | 内容 |
|------|------|
| 定义 | 标的价格变动 $1，期权价格的变动量 |
| Long Call Delta | 0 到 +1.0 |
| Long Put Delta | -1.0 到 0 |
| ATM Delta | ≈ ±0.5 |
| 三种含义 | 价格敏感度、ITM概率近似、等效股票头寸 |
| Delta 中性 | 组合 Delta = 0，不受标的小幅变动影响 |

### 思考题

1. 你持有 5 张 AAPL Call，Delta = 0.40。如果 AAPL 涨了 $3，你的期权头寸大约盈亏多少？
2. 你想建立一个 Delta 中性的组合：你卖了 20 张 Put，每张 Delta = -0.35。你需要买入多少股 AAPL 来对冲？
3. 为什么深度虚值期权的 Delta 接近 0？这对卖方意味着什么？
