# 看涨期权与看跌期权

上一节我们了解了期权的基本概念——它是一种赋予持有者"权利而非义务"的金融合约。那么，这个"权利"具体是什么？答案很简单：要么是**买入**的权利，要么是**卖出**的权利。前者叫看涨期权，后者叫看跌期权。这两个品种，是整个期权世界的两块基石。

---

## 1. 看涨期权（Call Option）详解

### 1.1 定义

> **看涨期权（Call Option）** 赋予持有者在到期日或之前，以约定的行权价（Strike Price）**买入**标的资产的权利，但不是义务。

"看涨"二字已经说明了一切——你买 Call，是因为你**看涨**标的资产的价格。如果股价真的涨上去了，你就能以低于市场价的行权价买入，赚取差价。

### 1.2 买方（Long Call）

**买方的角色：权利的拥有者。**

当你买入一份看涨期权时：
- **权利**：你有权在到期日或之前，以行权价买入 100 股标的股票
- **义务**：无。你没有任何义务必须买入
- **成本**：你需要支付权利金（Premium）给卖方
- **最大亏损**：限于你支付的权利金。如果股价跌到行权价以下，你放弃行权，损失权利金
- **最大盈利**：理论上无限。股价涨得越高，你的盈利越大

**买方收益公式：**

```
Long Call 收益 = max(标的股价 - 行权价, 0) - 权利金
```

- 如果 到期时股价 > 行权价：你行权，收益 = 股价 - 行权价 - 权利金
- 如果 到期时股价 <= 行权价：你放弃行权，收益 = -权利金（全部亏损）

### 1.3 卖方（Short Call）

**卖方的角色：义务的承担者。**

当你卖出一份看涨期权时：
- **权利**：你收取买方支付的权利金
- **义务**：如果买方选择行权，你必须以行权价卖出 100 股标的股票给买方
- **最大盈利**：限于你收取的权利金。买方不行权，你就白赚权利金
- **最大亏损**：理论上无限。股价涨得越高，你的亏损越大

**卖方收益公式：**

```
Short Call 收益 = 权利金 - max(标的股价 - 行权价, 0)
```

注意：卖方的收益结构恰好是买方的镜像。买方赚多少，卖方就亏多少；反之亦然。期权交易是**零和博弈（Zero-Sum Game）**。

---

## 2. 看跌期权（Put Option）详解

### 2.1 定义

> **看跌期权（Put Option）** 赋予持有者在到期日或之前，以约定的行权价（Strike Price）**卖出**标的资产的权利，但不是义务。

"看跌"意味着你看空标的资产的价格。如果股价真的跌下去，你就能以高于市场价的行权价卖出，赚取差价。

### 2.2 买方（Long Put）

**买方的角色：权利的拥有者。**

当你买入一份看跌期权时：
- **权利**：你有权在到期日或之前，以行权价卖出 100 股标的股票
- **义务**：无。你没有任何义务必须卖出
- **成本**：你需要支付权利金给卖方
- **最大亏损**：限于你支付的权利金
- **最大盈利**：理论上有限但非常大——股价最多跌到 0，所以最大盈利 = 行权价 - 权利金

**买方收益公式：**

```
Long Put 收益 = max(行权价 - 标的股价, 0) - 权利金
```

- 如果 到期时股价 < 行权价：你行权，收益 = 行权价 - 股价 - 权利金
- 如果 到期时股价 >= 行权价：你放弃行权，收益 = -权利金

### 2.3 卖方（Short Put）

**卖方的角色：义务的承担者。**

当你卖出一份看跌期权时：
- **权利**：你收取买方支付的权利金
- **义务**：如果买方选择行权，你必须以行权价买入 100 股标的股票
- **最大盈利**：限于你收取的权利金
- **最大亏损**：理论上很大——股价跌到 0 时，你必须以行权价买入一文不值的股票

**卖方收益公式：**

```
Short Put 收益 = 权利金 - max(行权价 - 标的股价, 0)
```

## 3. 四种基本头寸对比

期权交易有且仅有四种基本头寸（Basic Positions）：

| 头寸 | 英文 | 预期方向 | 最大盈利 | 最大亏损 | 到期日盈亏公式 |
|------|------|---------|---------|---------|--------------|
| 买入看涨 | Long Call | 看涨（Bullish） | 无限 | 权利金 | max(S-K, 0) - P |
| 卖出看涨 | Short Call | 看跌/横盘 | 权利金 | 无限 | P - max(S-K, 0) |
| 买入看跌 | Long Put | 看跌（Bearish） | K - 权利金 | 权利金 | max(K-S, 0) - P |
| 卖出看跌 | Short Put | 看涨/横盘 | 权利金 | K - 权利金 | P - max(K-S, 0) |

> **符号说明**：S = 到期时标的股价，K = 行权价（Strike Price），P = 权利金（Premium）

### 四种头寸的核心差异

**从方向性角度看：**
- **Long Call**：最直接的看涨工具。股价涨得越多，赚得越多
- **Short Call**：看跌或认为股价不会涨过行权价。赚的是权利金，赌的是股价不涨
- **Long Put**：最直接的看跌工具。股价跌得越多，赚得越多
- **Short Put**：看涨或认为股价不会跌破行权价。赚的是权利金，赌的是股价不跌

**从风险收益结构看：**
- **买方（Long）**：付出权利金，风险有限，收益潜力大。类似买保险
- **卖方（Short）**：收取权利金，收益有限，风险潜力大。类似卖保险

## 4. 盈亏分析与图示

理解期权最直观的方式就是画出盈亏图（Payoff Diagram）。下面逐一分析四种头寸。

### 4.1 Long Call（买入看涨期权）

假设：买入 AAPL Call，行权价 K = 200，权利金 P = 5

```
  盈亏($)
    |
 80 +                                              /
    |                                            /
 60 +                                          /
    |                                        /
 40 +                                      /
    |                                    /
 20 +                                  /
    |                                /
   0 +----+----+----+----+----+----/--+----+----+-----> 股价($)
    |  160  170  180  190  200  205  210  220  230
 -5 +********
    |        ********
-10 +
```

**分析要点：**
- 盈亏平衡点（Break-Even）= 行权价 + 权利金 = 200 + 5 = **205**
- 股价 < 200：放弃行权，最大亏损 = **-5**（权利金）
- 股价 = 205：刚好回本，不赚不亏
- 股价 > 205：开始盈利，股价越高赚得越多
- 股价 = 220：收益 = 220 - 200 - 5 = **15**

### 4.2 Short Call（卖出看涨期权）

假设：卖出 AAPL Call，行权价 K = 200，权利金 P = 5

```
  盈亏($)
    |
 10 +
    |
  5 +********
    |        ********
   0 +----+----+----+----+----+----/--+----+----+-----> 股价($)
    |  160  170  180  190  200  205  210  220  230
-20 +                                \
    |                                  \
-40 +                                    \
    |                                      \
-60 +                                        \
    |                                          \
-80 +                                            \
```

**分析要点：**
- 盈亏平衡点 = 200 + 5 = **205**（与 Long Call 相同）
- 股价 < 200：买方不行权，你白赚权利金 **+5**
- 股价 = 205：刚好不赚不亏
- 股价 > 205：开始亏损，股价越高亏得越多
- 最大盈利 = **+5**（权利金），最大亏损 = **理论上无限**

### 4.3 Long Put（买入看跌期权）

假设：买入 AAPL Put，行权价 K = 200，权利金 P = 6

```
  盈亏($)
    |
194 +\
    |  \
170 +    \
    |      \
150 +        \
    |          \
130 +            \
    |              \
 94 +                \
    |                  \
   0 +----+----+----+----+----+----+----+----+-----> 股价($)
    |  140  150  160  170  180  190  194  200  210
 -6 +                                    ********
    |                                            ********
```

**分析要点：**
- 盈亏平衡点 = 行权价 - 权利金 = 200 - 6 = **194**
- 股价 > 200：放弃行权，最大亏损 = **-6**（权利金）
- 股价 = 194：刚好回本，不赚不亏
- 股价 < 194：开始盈利，股价越低赚得越多
- 股价跌到 0：最大盈利 = 200 - 0 - 6 = **194**

假设：卖出 AAPL Put，行权价 K = 200，权利金 P = 6

```
    |          /
-100+        /
    |      /

    |  /
-194+/
```

**分析要点：**
- 盈亏平衡点 = 200 - 6 = **194**
- 股价 > 200：买方不行权，你白赚权利金 **+6**
- 股价 = 194：刚好不赚不亏
- 股价 < 194：开始亏损，股价越低亏得越多
- 股价跌到 0：最大亏损 = 200 - 6 = **-194**

把四种头寸放在同一张图中对比：

```
    |             /                  \
    |           /                    \
    |         /                        \
    |       /                            \
```

## 5. 实际案例分析

下面用真实市场的数据来举例说明。以下价格为示例价格，用于教学演示。

### 案例一：Long Call — 看涨 AAPL

**背景**：2026 年 6 月，AAPL 当前股价为 $195。你看好苹果在下季度的表现，预计财报后股价会突破 $210。

**交易**：
- 买入 1 张 AAPL 2026 年 9 月到期、行权价 $200 的 Call
- 权利金：$5.50/股，即 $550/张（1张 = 100股）

**情景分析：**

| 到期时股价 | 期权价值 | 收益 | 收益率 |
|-----------|---------|------|--------|
| $180（大跌） | $0（虚值作废） | -$550 | -100% |
| $195（不变） | $0（虚值作废） | -$550 | -100% |
| $200（刚好到行权价） | $0 | -$550 | -100% |
| $205（小涨） | $500 | -$50 | -9.1% |
| $210（预期目标） | $1,000 | +$450 | +81.8% |
| $220（大涨） | $2,000 | +$1,450 | +263.6% |
| $250（暴涨） | $5,000 | +$4,450 | +809.1% |

**关键洞察：**
- 盈亏平衡点 = $200 + $5.50 = **$205.50**
- 如果 AAPL 涨到 $210，你用 $550 的成本赚了 $450，收益率 81.8%
- 如果用同样的 $550 买股票，只能买 2.8 股，涨到 $210 只赚 $42
- **这就是期权的杠杆效应（Leverage）**
- 风险：股价不涨或下跌，你损失全部 $550

### 案例二：Long Put — 对冲 AAPL 下跌风险

**背景**：你持有 100 股 AAPL，成本价 $190，当前股价 $195。下个月有财报，你担心业绩不及预期导致股价大跌，但又不想卖出股票（怕错过上涨）。

**交易**：
- 买入 1 张 AAPL 2026 年 7 月到期、行权价 $190 的 Put
- 权利金：$4.00/股，即 $400/张

**情景分析：**

| 到期时股价 | 股票盈亏 | Put 价值 | Put 收益 | 组合总盈亏 |
|-----------|---------|---------|---------|-----------|
| $170（暴跌） | -$2,500 | $2,000 | +$1,600 | -$900 |
| $180（大跌） | -$1,500 | $1,000 | +$600 | -$900 |
| $190（跌到行权价） | -$500 | $0 | -$400 | -$900 |
| $195（不变） | $0 | $0 | -$400 | -$400 |
| $210（上涨） | +$1,500 | $0 | -$400 | +$1,100 |
| $230（大涨） | +$3,500 | $0 | -$400 | +$3,100 |

**关键洞察：**
- 股价暴跌到 $170 时，如果没有 Put 保护，你会亏 $2,500
- 有了 Put 保护，你只亏 $900，**下跌风险被大幅限制**
- 代价是：如果股价上涨，你的盈利少 $400（权利金成本）
- **这就是期权的保险功能（Insurance）**——花一笔小钱，防止大损失

### 案例三：Short Put — 低价买入 AAPL 的策略

**背景**：AAPL 当前股价 $195，你一直想买入 AAPL，但觉得 $195 太贵了，希望在 $185 附近建仓。

**交易**：
- 卖出 1 张 AAPL 2026 年 8 月到期、行权价 $185 的 Put
- 收取权利金：$3.00/股，即 $300/张

**情景分析：**

| 到期时股价 | 是否被行权 | 收益/结果 |
|-----------|-----------|----------|
| $195（不变/上涨） | 否 | 白赚 $300 权利金 |
| $190（小跌） | 否 | 白赚 $300 权利金 |
| $185（跌到行权价） | 刚好 | 以 $185 买入 100 股，实际成本 $182 |
| $180（跌破行权价） | 是 | 以 $185 买入，但市场价只有 $180，净亏 $200 |
| $170（大跌） | 是 | 以 $185 买入，但市场价只有 $170，净亏 $1,200 |

**关键洞察：**
- 如果股价没跌到 $185，你白赚 $300——比银行存款强多了
- 如果股价跌到 $185 以下，你被迫以 $185 买入——但你本来就想在这个价位买，实际成本是 $185 - $3 = $182
- **这叫"有保护地抄底"**：你愿意买，就用 Short Put 设一个你满意的买入价
- 风险：股价暴跌时你会在高位被行权买入，所以只对你真心想持有的标的使用这个策略

### 案例四：Short Call — 备兑看涨（Covered Call）

**背景**：你持有 100 股 AAPL，成本 $180，当前股价 $195。你认为短期内股价不会突破 $210。

**交易**：
- 卖出 1 张 AAPL 2026 年 7 月到期、行权价 $210 的 Call
- 收取权利金：$2.50/股，即 $250/张

**情景分析：**

| 到期时股价 | 股票盈亏 | Call 结果 | 组合总盈亏 |
|-----------|---------|----------|-----------|
| $195（不变） | +$1,500 | 白赚 $250 | +$1,750 |
| $205（小涨） | +$2,500 | 白赚 $250 | +$2,750 |
| $210（涨到行权价） | +$3,000 | 白赚 $250 | +$3,250 |
| $220（突破行权价） | +$3,000* | -$750 | +$2,250 |
| $250（大涨） | +$3,000* | -$3,750 | -$750 |

> *股价超过 $210 后，你的股票会被以 $210 行权卖出，股票盈亏锁定在 $3,000

**关键洞察：**
- **Covered Call 是最经典的期权收入策略**——持有股票的同时卖 Call 赚权利金
- 适合"温和看涨"的场景：你认为股价会涨，但不会暴涨
- 代价：如果股价暴涨，你错过了 $210 以上的收益
- 本质：用"放弃暴涨收益"换取"确定的权利金收入"

## 6. 权利金的构成（简要引入）

我们一直在说"权利金"，那权利金到底是怎么定价的？一份期权的权利金由两部分构成：

```
权利金（Premium） = 内在价值（Intrinsic Value） + 时间价值（Time Value）
```

### 6.1 内在价值（Intrinsic Value）

内在价值是期权**立即行权**能获得的收益，它只取决于当前股价与行权价的关系。

**对于 Call：**
```
内在价值 = max(当前股价 - 行权价, 0)
```
- 股价 > 行权价：有内在价值，称为**实值（In-the-Money, ITM）**
- 股价 = 行权价：内在价值为零，称为**平值（At-the-Money, ATM）**
- 股价 < 行权价：内在价值为零，称为**虚值（Out-of-the-Money, OTM）**

**对于 Put：**
```
内在价值 = max(行权价 - 当前股价, 0)
```
- 股价 < 行权价：实值（ITM）
- 股价 = 行权价：平值（ATM）
- 股价 > 行权价：虚值（OTM）

**举例说明：AAPL 当前股价 $195**

| 期权类型 | 行权价 | 状态 | 内在价值 |
|---------|--------|------|---------|
| Call | $190 | ITM（实值） | $5 |
| Call | $195 | ATM（平值） | $0 |
| Call | $200 | OTM（虚值） | $0 |
| Put | $190 | OTM（虚值） | $0 |
| Put | $195 | ATM（平值） | $0 |
| Put | $200 | ITM（实值） | $5 |

### 6.2 时间价值（Time Value）

```
时间价值 = 权利金 - 内在价值
```

时间价值反映的是"在到期之前，股价有可能向有利方向变动"的价值。它受以下因素影响：

- **剩余时间**：到期时间越长，时间价值越高（更多时间等待股价变动）
- **波动率**：标的资产波动率越高，时间价值越高（股价更可能大幅变动）
- **利率**：利率越高，Call 的时间价值越高，Put 的时间价值越低

**举例：AAPL 当前股价 $195，行权价 $200 的 Call，权利金 $5.50**

```
内在价值 = max(195 - 200, 0) = $0（虚值）
时间价值 = $5.50 - $0 = $5.50
```

这份期权全部价值都是时间价值——买方赌的是到期前 AAPL 能涨过 $200。

**再举例：AAPL 当前股价 $195，行权价 $190 的 Call，权利金 $8**

```
内在价值 = max(195 - 190, 0) = $5（实值）
时间价值 = $8 - $5 = $3
```

这份期权有 $5 的"实打实"的内在价值，另外 $3 是时间价值。

### 6.3 时间衰减（Time Decay）

时间价值不是恒定的，它会随着到期日的临近而加速衰减。这个现象叫做**时间衰减（Time Decay）**，用希腊字母 **Theta（）** 衡量。

```
时间价值衰减示意：

时间价值
    |
    |*****
    |     ****
    |        ***
    |           **
    |             **
    |               *
    |                *
    |                 *
    +---+---+---+---+---+---> 时间
   60天 45天 30天 15天  0天（到期）
```

**关键规律：**
- 最后 30 天的时间衰减速度最快
- 平值（ATM）期权的时间价值最高
- 实值和虚值期权的时间价值相对较低

这对买方和卖方意味着：
- **买方的敌人是时间**：持有期权越久，时间价值流失越多
- **卖方的朋友是时间**：时间流逝，卖方收取的权利金越来越"安全"

---

## 7. 实用技巧与注意事项

### 7.1 选择 Call 还是 Put？

| 你的观点 | 推荐操作 | 说明 |
|---------|---------|------|
| 强烈看涨 | Long Call（买入看涨） | 杠杆做多，风险有限 |
| 温和看涨 | Covered Call 或 Bull Call Spread | 降低成本，但限制收益 |
| 强烈看跌 | Long Put（买入看跌） | 杠杆做空，风险有限 |
| 温和看跌 | Bear Put Spread | 降低成本，但限制收益 |
| 不确定方向但预期大波动 | Long Straddle（同时买 Call 和 Put） | 双向押注 |
| 认为不会波动 | Short Straddle（同时卖 Call 和 Put） | 赚取双份权利金 |

### 7.2 买方与卖方的心态差异

**作为买方：**
- 你需要标的资产**有足够大的波动**才能覆盖权利金成本
- 胜率通常较低（可能只有 30%-40%），但单次盈利可能很大
- 像买彩票——大多数时候亏小钱，偶尔赚大钱
- 时间是你的敌人

**作为卖方：**
- 你希望标的资产**不要大幅波动**，在行权价附近待着
- 胜率通常较高（可能有 60%-70%），但单次盈利有限
- 像开保险公司——大多数时候收保费，偶尔赔大钱
- 时间是你的朋友

### 7.3 新手常见错误

**错误一：只买深度虚值期权（Deep OTM）**
- 深度虚值期权虽然便宜，但需要股价大幅波动才能获利，胜率极低
- 长期大量买入深度虚值期权，大概率持续亏损

**错误二：忽视时间衰减**
- 买了一个月到期的 ATM Call，股价一周没动，你的期权可能已经贬值了 20-30%
- 时间衰减对买方是持续的"隐形税"

**错误三：裸卖期权不设止损（Naked Short without Stop Loss）**
- 裸卖 Call 的风险理论上是无限的——如果股价暴涨，你可能倾家荡产
- 新手绝对不要裸卖期权，至少用价差策略限制风险

**错误四：不理解合约乘数**
- 美股期权 1 张合约 = 100 股
- 权利金报价 $5 意味着每张合约 $500
- 不要把每股权利金和每张合约权利金搞混

### 7.4 如何快速判断期权状态

拿到一张期权合约，用这个快速判断法：

```
对于 Call：
  股价 > 行权价 → 实值（ITM）→ 有内在价值
  股价 = 行权价 → 平值（ATM）→ 内在价值为零
  股价 < 行权价 → 虚值（OTM）→ 内在价值为零

对于 Put：
  股价 < 行权价 → 实值（ITM）→ 有内在价值
  股价 = 行权价 → 平值（ATM）→ 内在价值为零
  股价 > 行权价 → 虚值（OTM）→ 内在价值为零
```

记忆口诀：**Call 比行权价高是实值，Put 比行权价低是实值**。"实值"就是"现在行权有得赚"。

## 8. 小结

本节我们系统学习了期权的两大基本类型：

| 要点 | 看涨期权（Call） | 看跌期权（Put） |
|------|-----------------|----------------|
| 买方权利 | 以行权价**买入**标的资产 | 以行权价**卖出**标的资产 |
| 买方预期 | 看涨 | 看跌 |
| 买方最大亏损 | 权利金 | 权利金 |
| 买方最大盈利 | 无限 | 行权价 - 权利金 |
| 卖方最大盈利 | 权利金 | 权利金 |
| 卖方最大亏损 | 无限 | 行权价 - 权利金 |
| 盈亏平衡点 | 行权价 + 权利金 | 行权价 - 权利金 |

**四种基本头寸的核心逻辑：**

```
┌─────────────────────────────────────────────────────┐
│                    期权交易四象限                      │
│                                                     │
│   看涨 + 买入 = Long Call（杠杆做多，风险有限）       │
│   看涨 + 卖出 = Short Call（赚权利金，风险无限）      │
│   看跌 + 买入 = Long Put（杠杆做空，风险有限）       │
│   看跌 + 卖出 = Short Put（赚权利金，风险较大）      │
│                                                     │
└─────────────────────────────────────────────────────┘
```

**权利金的构成：**
- 权利金 = 内在价值 + 时间价值
- 内在价值取决于股价与行权价的关系
- 时间价值随到期日临近加速衰减
- 实值、平值、虚值是描述期权状态的核心概念

### 思考题

1. AAPL 当前股价 $195，你买入一张行权价 $200 的 Call，权利金 $5。到期时 AAPL 涨到 $230，你的收益率是多少？如果用同样的 $500 直接买股票，收益率又是多少？两者差距说明了什么？

2. 为什么说"卖期权像开保险公司"？卖方的高胜率和有限收益之间有什么内在逻辑？

3. 你持有 100 股 TSLA，成本 $250，当前股价 $280。你决定卖出一张行权价 $300 的 Covered Call，收取权利金 $8。如果到期时 TSLA 涨到 $350，你的总收益是多少？你是否后悔卖了这个 Call？

4. AAPL 的行权价 $180 的 Call（当前股价 $195）和行权价 $210 的 Call，哪个的内在价值更高？哪个的时间价值可能更高？为什么？

5. 为什么深度虚值期权虽然便宜，但长期买入通常会亏损？请从"内在价值"和"时间价值"的角度分析。
