# 组合风险度量

在前面的章节中，我们学习了单个期权头寸的Greeks——Delta、Gamma、Theta、Vega。这些指标对于理解单笔交易的风险特征非常有用。然而，当你同时持有多个期权头寸时，仅仅看单个头寸的Greeks是远远不够的。你需要一个**组合层面（Portfolio Level）**的风险视角，才能真正理解你的投资组合面临什么样的风险。

本章将系统介绍组合风险管理的核心度量指标，帮助你从"看一棵树"进化到"看整片森林"。

---

## 一、组合风险度量概述——超越单个Greeks

### 1.1 为什么单个Greeks不够？

假设你同时持有以下三个头寸：

- 头寸A：买入SPY看涨期权，Delta = +500，Vega = +200
- 头寸B：卖出SPY看跌期权，Delta = +300，Vega = -150
- 头寸C：买入QQQ看涨期权，Delta = +400，Vega = +180

如果你只看每个头寸的Greeks，你会知道每个头寸单独的风险。但你真正需要知道的是：**如果市场上涨1%，你的整体会赚多少？如果波动率突然飙升，你的整体盈亏如何？**

这就是组合风险度量要回答的问题。

### 1.2 组合风险度量的层次

组合风险度量可以分为三个层次：

1. **Greeks汇总层**：将所有头寸的Greeks加总，得到Portfolio Greeks
2. **统计风险层**：使用VaR、Expected Shortfall等统计指标衡量整体风险
3. **极端风险层**：通过压力测试和情景分析评估极端市场条件下的潜在损失

本章主要覆盖前两个层次，第三个层次将在压力测试章节中详细讨论。

## 二、Portfolio Delta——组合方向风险

### 2.1 基本概念

**Portfolio Delta**（组合Delta）是所有头寸Delta的简单加总：

```
Portfolio Delta = Σ(每个头寸的Delta × 合约乘数 × 持仓数量)
```

对于美股期权，每张合约代表100股，因此：

```
Portfolio Delta = Σ(Delta_i × 100 × 持仓张数_i)
```

Portfolio Delta告诉我们：**当标的资产价格变动1美元时，整个组合的盈亏变化。**

### 2.2 计算实例

假设你的组合如下：

| 头寸 | 方向 | 张数 | 单张Delta | Delta贡献 |
|------|------|------|-----------|-----------|
| SPY 500 Call | 买入 | 2 | 0.55 | +110 |
| SPY 510 Call | 卖出 | 3 | 0.35 | -105 |
| SPY 490 Put | 买入 | 1 | -0.30 | -30 |
| QQQ 400 Call | 买入 | 1 | 0.60 | +60 |

Portfolio Delta = (+110 - 105 - 30 + 60) × 100 = +3,500

这意味着SPY每上涨1美元，组合大约盈利3,500美元。但注意，QQQ和SPY并非完全同步，实际盈亏还取决于两个标的的相关性。

### 2.3 Delta中性策略

当Portfolio Delta接近零时，我们称组合为**Delta中性（Delta Neutral）**。这意味着组合对标的资产的小幅价格变动不敏感。

Delta中性的好处：
- 消除方向性风险，专注于从时间衰减或波动率变化中获利
- 降低组合对短期市场噪音的敏感度
- 是做市商和专业交易员的基本策略

Delta中性的局限：
- 只对小幅价格变动有效，大幅变动时Delta本身会变化（这就是Gamma的作用）
- 需要定期再平衡（Rebalancing），因为Delta会随着价格和时间变化

### 2.4 IBKR中的Portfolio Delta

在盈透证券（Interactive Brokers）的TWS平台中，你可以在**Risk Navigator**工具中查看Portfolio Delta。具体路径为：

1. 打开TWS，点击"分析"菜单
2. 选择"Risk Navigator"
3. 在风险报告中查看"Portfolio Delta"列

你也可以在"期权分析"面板中查看每个标的的汇总Delta。

## 三、Portfolio Gamma——组合Delta变化速度

### 3.1 基本概念

**Portfolio Gamma**（组合Gamma）是所有头寸Gamma的加总：

```
Portfolio Gamma = Σ(每个头寸的Gamma × 合约乘数 × 持仓张数)
```

Gamma衡量的是：**当标的资产价格变动1美元时，Delta的变化量。**

- **正Gamma**：价格越涨Delta越大（越涨越买），价格越跌Delta越小（越跌越卖）。这是买入期权的特征。
- **负Gamma**：价格越涨Delta越小，价格越跌Delta越大。这是卖出期权的特征。

### 3.2 Gamma的风险含义

正Gamma组合在大幅波动时有利——无论涨跌，你都能"追上"趋势。但你需要付出时间价值（Theta）。

负Gamma组合在横盘时有利——你收取时间价值。但一旦出现大幅波动，你可能面临巨大亏损。

这就是期权交易中经典的**Gamma-Theta权衡**：

| 组合特征 | Gamma | Theta | 适合环境 |
|---------|-------|-------|---------|
| 买入期权 | 正 | 负（付出） | 大幅波动 |
| 卖出期权 | 负 | 正（收取） | 横盘整理 |

### 3.3 Gamma风险的非线性

Gamma是一个**二阶导数**，它衡量的是Delta变化的速度。这意味着：

- 当组合处于**平值（At-the-Money, ATM）**附近时，Gamma最大
- 当组合处于深度实值或深度虚值时，Gamma接近零
- Gamma在到期日临近时急剧增大，尤其是ATM期权

这解释了为什么**卖出大量临近到期的ATM期权**是极其危险的——你面临巨大的Gamma风险（也称为"Gamma炸弹"），一个小幅价格波动就可能导致巨额亏损。

### 3.4 组合Gamma管理

管理组合Gamma的原则：

1. **监控Portfolio Gamma的正负号**：确保你知道组合在大幅波动时的表现
2. **注意Gamma在到期前的加速效应**：如果组合中有大量即将到期的ATM期权，要特别警惕
3. **使用远月期权对冲Gamma**：远月期权的Gamma较小，可以平滑组合的Gamma暴露
4. **设置Gamma警戒线**：当Portfolio Gamma超过某个阈值时，采取行动

## 四、Portfolio Theta——组合每日时间损益

### 4.1 基本概念

**Portfolio Theta**（组合Theta）是所有头寸Theta的加总：

```
Portfolio Theta = Σ(每个头寸的Theta × 合约乘数 × 持仓张数)
```

Theta衡量的是：**在其他条件不变的情况下，每过一天组合的价值变化。**

- **正Theta**：每过一天赚钱（通常是卖方策略）
- **负Theta**：每过一天亏钱（通常是买方策略）

### 4.2 Theta与时间衰减

期权的时间价值衰减不是线性的，而是加速的。可以用以下经验法则：

- 距到期30天以上：时间衰减较慢
- 距到期30天以内：时间衰减加速
- 距到期7天以内：时间衰减极快
- 距到期最后一天：ATM期权可能损失剩余时间价值的50%以上

这意味着：
- **卖方**倾向于卖出30天以内到期的期权，以获取快速的时间衰减
- **买方**倾向于买入更长期限的期权，以减缓时间衰减

### 4.3 Theta-Gamma对称性

在Black-Scholes模型框架下，对于Delta中性的组合，存在一个重要的近似关系：

```
Theta ≈ -½ × Γ × S² × σ²
```

其中S是标的资产价格，σ是隐含波动率。

这个公式告诉我们：
- **正Gamma意味着负Theta**：你享受价格波动的好处，但要付出时间衰减的代价
- **负Gamma意味着正Theta**：你收取时间衰减，但承担价格波动的风险

这个对称性是期权交易的核心权衡，理解它对组合管理至关重要。

### 4.4 每日Theta收益的追踪

建议每天记录组合的Theta，并与实际盈亏对比。长期来看，如果你是卖方策略，实际的每日收益应该接近你的Portfolio Theta（扣除交易成本后）。

如果实际收益持续偏离Theta预期，可能意味着：
- 你的Gamma暴露导致了额外波动
- 隐含波动率发生了变化（Vega的影响）
- 你的Greeks计算可能不准确

## 五、Portfolio Vega——组合波动率风险

### 5.1 基本概念

**Portfolio Vega**（组合Vega）是所有头寸Vega的加总：

```
Portfolio Vega = Σ(每个头寸的Vega × 合约乘数 × 持仓张数)
```

Vega衡量的是：**当隐含波动率（Implied Volatility, IV）变动1个百分点（1 vol point）时，组合的价值变化。**

### 5.2 Vega的重要性

许多期权交易新手过度关注方向（Delta），却忽略了波动率（Vega）对组合的影响。实际上，在很多策略中，Vega的影响可能比Delta更大。

例如，如果你卖出了一组ATM跨式组合（Short Straddle），你的Delta接近零，但Vega可能非常负。如果市场虽然没怎么动，但V突然从15%飙升到30%，你的组合可能亏损惨重。

### 5.3 Vega与波动率曲面

更精确的Vega分析需要考虑**波动率曲面（Volatility Surface）**的变化模式：

1. **平行移动（Parallel Shift）**：所有行权价的IV同时上升或下降1个点
2. **偏度变化（Skew Change）**：低行权价和高行权价的IV差异发生变化
3. **期限结构变化（Term Structure Change）**：不同到期日的IV变化不同

Portfolio Vega假设的是平行移动，但实际中波动率曲面的变化往往更加复杂。高级交易员会使用**Vega Bucket**（分桶Vega）来监控不同行权价和到期日的Vega暴露。

### 5.4 Vega风险管理

管理Vega暴露的原则：

1. **了解你的Vega方向**：正Vega从IV上升中获利，负Vega从IV下降中获利
2. **注意事件风险**：财报、美联储会议等事件前后，IV通常大幅波动
3. **使用不同到期日的期权对冲Vega**：近月和远月期权对IV变化的敏感度不同
4. **监控VIX指数**：VIX是市场整体恐慌情绪的指标，与大多数期权的IV正相关

## 六、VaR（Value at Risk）——在险价值

### 6.1 什么是VaR？

**VaR（Value at Risk，在险价值）**是最广泛使用的市场风险度量指标之一。它回答的问题是：

> 在给定的置信水平下，在给定的时间内，我的组合最多可能亏损多少？

例如，"95% VaR = 50,000元"意味着：在正常市场条件下，有95%的概率，你的组合在指定时间段内的亏损不会超过50,000元。换句话说，有5%的概率亏损会超过50,000元。

### 6.2 VaR的计算方法

VaR有三种主要计算方法：

**方法一：参数法（Parametric Method）**

假设组合收益率服从正态分布：

```
VaR = Portfolio Value × Z × σ × √T
```

其中：
- Z是标准正态分布的分位数（95%置信水平下为1.645，99%为2.326）
- σ是组合收益率的标准差（年化）
- T是时间期限（以年为单位）

优点：计算简单快速
缺点：假设正态分布，无法捕捉尾部风险

**方法二：历史模拟法（Historical Simulation）**

使用历史收益率数据，按实际发生的盈亏排列，取对应分位数。

优点：不需要分布假设，反映真实市场
缺点：依赖历史数据，历史可能不代表未来

**方法三：蒙特卡洛模拟法（Monte Carlo Simulation）**

通过大量随机模拟来生成可能的组合盈亏分布。

优点：灵活性最高，可以处理复杂衍生品
缺点：计算量大，需要准确的模型参数

### 6.3 期权组合VaR的特殊性

期权组合的VaR比股票组合更复杂，因为：

1. **非线性收益**：期权的收益不是标的价格的线性函数
2. **多因子风险**：价格、波动率、时间都在变化
3. **尾部风险**：期权卖方的亏损可能远超正态分布的预期

对于期权组合，推荐使用**历史模拟法**或**蒙特卡洛模拟法**，因为参数法的正态分布假设往往不适用。

### 6.4 VaR的局限性

VaR有一个重要的局限：它只告诉你"在正常情况下最多亏多少"，但不告诉你"如果情况不正常会亏多少"。这就是为什么我们需要Expected Shortfall。

## 七、Expected Shortfall——条件风险

### 7.1 什么是Expected Shortfall？

**Expected Shortfall（ES，预期尾部损失）**，也称为**Conditional VaR（CVaR，条件在险价值）**，它回答的问题是：

> 如果亏损超过了VaR，平均会亏多少？

例如，"95% ES = 80,000元"意味着：当亏损超过VaR（即那5%的极端情况下），平均亏损为80,000元。

### 7.2 ES vs VaR

| 特征 | VaR | Expected Shortfall |
|------|-----|-------------------|
| 衡量内容 | 亏损的分位数 | 超过VaR后的平均亏损 |
| 尾部风险 | 不反映 | 充分反映 |
| 一致性 | 不是一致风险度量 | 是一致风险度量 |
| 直觉性 | 更容易理解 | 略抽象但更有意义 |

**一致风险度量（Coherent Risk Measure）**是数学上的一个重要概念。简单来说，ES满足次可加性（两个组合的ES之和大于等于合并后组合的ES），而VaR不满足。这意味着ES在组合层面是更合理的风险度量。

### 7.3 为什么期权交易者需要关注ES？

期权卖方策略的收益分布通常是**负偏态（Negative Skew）**的——大多数时候小赚，偶尔大亏。VaR可能无法充分捕捉这种尾部风险，而ES能更好地反映极端亏损的预期。

例如，一个卖出虚值看跌期权的策略，可能95% VaR看起来很小，但95% ES可能大得多，因为一旦发生极端下跌，亏损可能是灾难性的。

## 八、最大回撤（Max Drawdown）

### 8.1 什么是最大回撤？

**最大回撤（Maximum Drawdown, MDD）**是指从组合的历史最高点到最低点的最大百分比下降：

```
MDD = (Peak Value - Trough Value) / Peak Value × 100%
```

### 8.2 为什么最大回撤很重要？

最大回撤是衡量**实际体验**的风险指标。VaR和ES是理论上的风险度量，而最大回撤是你真正"感受到"的亏损。

一个策略可能年化收益率20%，但如果最大回撤达到50%，大多数投资者在回撤20%的时候就已经恐慌离场了。

### 8.3 回撤相关指标

除了最大回撤，还有几个重要的回撤相关指标：

1. **回撤持续时间（Drawdown Duration）**：从开始回撤到恢复到前高点的时间
2. **平均回撤（Average Drawdown）**：所有回撤的平均幅度
3. **Calmar比率**：年化收益率 / 最大回撤，衡量每单位最大回撤带来的收益

### 8.4 期权策略的最大回撤特征

不同期权策略的最大回撤特征差异很大：

| 策略 | 典型最大回撤 | 回撤形态 |
|------|------------|---------|
| 买入看涨 | 100%（全部权利金） | 缓慢下降后可能快速反弹 |
| 卖出看跌 | 理论无限（实际很大） | 快速大幅下降 |
| 铁鹰式 | 有限但频繁 | 小幅频繁回撤 |
| 保护性看跌 | 有限 | 与标的基本同步 |

## 九、风险度量在IBKR中的查看方法

### 9.1 Risk Navigator

盈透证券的**Risk Navigator**是最强大的组合风险管理工具。使用方法：

1. 在TWS中，点击菜单"分析" > "Risk Navigator"
2. 选择"Portfolio"视图
3. 你将看到以下关键指标：
   - **Portfolio Delta**：组合的总Delta
   - **Portfolio Gamma**：组合的总Gamma
   - **Portfolio Theta**：组合的每日Theta
   - **Portfolio Vega**：组合的总Vega
   - **Portfolio Value**：组合市值

### 9.2 风险报告定制

你可以定制Risk Navigator的报告：

- **按标的分组**：查看每个标的的风险贡献
- **按到期日分组**：查看不同到期日的风险暴露
- **按策略分组**：查看不同策略的风险特征

### 9.3 模拟场景分析

IBKR还提供**What-If分析**功能：

1. 在TWS中右键点击任意头寸
2. 选择"What-If Analysis"
3. 模拟标的价格变化、IV变化、时间流逝等情景
4. 查看组合在不同情景下的盈亏

这是最直观的压力测试工具，强烈建议经常使用。

### 9.4 预警设置

建议设置以下预警：

- Portfolio Delta超过阈值时预警
- 保证金使用率超过80%时预警
- 单一标的风险敞口超过总资金20%时预警
- 即将到期的头寸提醒

## 十、小结

本章介绍了组合风险管理的核心度量指标：

1. **Portfolio Greeks**（Delta、Gamma、Theta、Vega）是组合层面的Greeks汇总，是日常风险管理的基础
2. **VaR和Expected Shortfall**是统计风险度量，帮助理解组合的整体风险水平
3. **最大回撤**是实际体验风险的度量，对心理管理至关重要

关键要点：

- 不要只看单个头寸的风险，要看组合层面的整体风险
- Gamma-Theta权衡是期权交易的核心：你不可能同时享受正Gamma和正Theta
- VaR有局限性，Expected Shortfall更能反映尾部风险
- 最大回撤是你真正感受到的风险，要重视

## 思考题

1. 假设你同时卖出SPY的看涨和看跌期权各10张，ATM Delta分别为0.50和-0.50，你的Portfolio Delta是多少？你的Gamma和Theta是什么方向？

2. 你的组合95% VaR为30,000元，95% ES为50,000元。这意味着什么？如果你是卖出虚值看跌期权为主的策略，你更应该关注哪个指标？为什么？

3. 某策略年化收益率15%，最大回撤40%，另一策略年化收益率10%，最大回撤15%。你认为哪个策略更好？Calmar比率分别是多少？
