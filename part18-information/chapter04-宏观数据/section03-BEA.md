# BEA

## 定义

> **BEA（Bureau of Economic Analysis，经济分析局）**是美国商务部下属机构，负责发布GDP、贸易和个人收入数据。

## 网址

https://www.bea.gov

## 主要数据

| 数据 | 频率 | 重要性 |
|------|------|--------|
| **GDP** | 每季度 | ⭐⭐⭐⭐⭐ |
| **个人收入** | 每月 | ⭐⭐⭐⭐ |
| **个人支出** | 每月 | ⭐⭐⭐⭐ |
| **贸易数据** | 每月 | ⭐⭐⭐ |
| **企业利润** | 每季度 | ⭐⭐⭐ |

## GDP数据

| 指标 | 说明 |
|------|------|
| GDP增速 | 经济增长速度 |
| GDP构成 | 消费+投资+政府+净出口 |
| PCE | 个人消费支出（通胀指标） |

## 如何使用

| 用途 | 方法 |
|------|------|
| 经济增长 | 关注GDP增速 |
| 消费趋势 | 关注个人支出 |
| 通胀分析 | 关注PCE |
| 贸易分析 | 关注贸易数据 |
