# BLS

## 定义

> **BLS（Bureau of Labor Statistics，劳工统计局）**是美国劳工部下属机构，负责发布就业、物价和工资数据。

## 网址

https://www.bls.gov

## 主要数据

| 数据 | 频率 | 重要性 |
|------|------|--------|
| **非农就业** | 每月 | ⭐⭐⭐⭐⭐ |
| **失业率** | 每月 | ⭐⭐⭐⭐⭐ |
| **CPI** | 每月 | ⭐⭐⭐⭐⭐ |
| **PPI** | 每月 | ⭐⭐⭐⭐ |
| **平均时薪** | 每月 | ⭐⭐⭐⭐ |
| **职位空缺（JOLTS）** | 每月 | ⭐⭐⭐ |

## 非农就业数据

每月第一个周五发布，是最重要的经济数据之一。

| 指标 | 说明 |
|------|------|
| 新增就业 | 当月新增非农就业人数 |
| 失业率 | 失业人口占劳动力比例 |
| 劳动参与率 | 劳动力占适龄人口比例 |
| 平均时薪 | 时薪变化（通胀信号） |

## 如何使用

| 用途 | 方法 |
|------|------|
| 追踪就业 | 关注非农和失业率 |
| 通胀分析 | 关注CPI和PPI |
| 工资趋势 | 关注平均时薪 |
| 市场影响 | 数据发布时市场波动 |

2. 如何从BLS数据判断经济是否在衰退？
