# FRED

## 定义

> **FRED（Federal Reserve Economic Data）**是圣路易斯联邦储备银行维护的经济数据库，包含超过80万个经济时间序列。

## 网址

https://fred.stlouisfed.org

## 主要数据

| 数据 | 说明 |
|------|------|
| GDP | 国内生产总值 |
| CPI | 消费者物价指数 |
| PPI | 生产者物价指数 |
| 失业率 | 失业率 |
| 利率 | 联邦基金利率 |
| 国债收益率 | 各期限国债收益率 |
| 美元指数 | DXY |
| PMI | 采购经理指数 |

## 如何使用

| 用途 | 方法 |
|------|------|
| 查看数据 | 搜索指标名称 |
| 下载数据 | 导出CSV/Excel |
| 创建图表 | 可视化数据 |
| 对比分析 | 多指标对比 |

## 优势

| 优势 | 说明 |
|------|------|
| 免费 | 完全免费 |
| 权威 | 官方数据 |
| 全面 | 覆盖广泛 |
| 更新及时 | 数据更新快 |
