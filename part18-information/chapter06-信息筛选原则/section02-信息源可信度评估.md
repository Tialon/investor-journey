# 信息源可信度评估

## 评估维度

### 1. 来源权威性

| 来源 | 可信度 |
|------|--------|
| SEC文件 | ⭐⭐⭐⭐⭐ |
| 公司官方 | ⭐⭐⭐⭐⭐ |
| 知名媒体 | ⭐⭐⭐⭐ |
| 分析师报告 | ⭐⭐⭐ |
| 社交媒体 | ⭐⭐ |
| 匿名论坛 | ⭐ |

### 2. 作者专业性

| 因素 | 说明 |
|------|------|
| 专业背景 | 是否有金融/行业背景 |
| 历史记录 | 过去的分析是否准确 |
| 透明度 | 是否披露持仓 |
| 独立性 | 是否有利益冲突 |

### 3. 信息时效性

| 因素 | 说明 |
|------|------|
| 发布日期 | 是否最新 |
| 更新频率 | 是否定期更新 |
| 数据来源 | 是否引用原始数据 |

### 4. 论证质量

| 因素 | 说明 |
|------|------|
| 数据支持 | 是否有数据支撑 |
| 逻辑清晰 | 推理是否合理 |
| 多元视角 | 是否考虑不同观点 |
| 风险提示 | 是否提示风险 |

## 信息源评级

| 等级 | 来源 | 使用建议 |
|------|------|----------|
| **A级** | SEC文件、公司财报 | 直接使用 |
| **B级** | 知名媒体、权威机构 | 交叉验证后使用 |
| **C级** | 分析师报告 | 了解立场后参考 |
| **D级** | 社交媒体、论坛 | 仅作参考，谨慎使用 |

## 利益冲突识别

| 冲突类型 | 说明 |
|----------|------|
| 持仓 | 作者是否持有讨论的股票 |
| 广告 | 是否有商业合作 |
| 佣金 | 是否有佣金收入 |
| 立场 | 是否有看多/看空立场 |

## 实践框架

```
1. 来源是什么？
   └── 官方 vs 媒体 vs 社交

2. 作者是谁？
   └── 专业性、历史记录

3. 什么时候发布的？
   └── 是否最新

4. 有没有利益冲突？
   └── 持仓、广告、立场

5. 论证质量如何？
   └── 数据、逻辑、多元视角
```
