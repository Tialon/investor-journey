# Bloomberg

## 定义

> **Bloomberg**是全球领先的金融数据和新闻平台，提供实时市场数据、分析工具和新闻。

## 主要产品

| 产品 | 说明 | 费用 |
|------|------|------|
| **Bloomberg Terminal** | 专业终端 | ~$20,000/年 |
| **Bloomberg News** | 财经新闻 | 部分免费 |
| **Bloomberg Data** | 金融数据 | 需订阅 |

## Bloomberg Terminal功能

| 功能 | 说明 |
|------|------|
| 实时数据 | 全球市场实时报价 |
| 分析工具 | 估值、风险分析 |
| 新闻 | 实时财经新闻 |
| 通讯 | 即时通讯 |
| 交易 | 执行交易 |

## 如何使用

| 用途 | 方法 |
|------|------|
| 市场数据 | 实时报价和图表 |
| 公司分析 | 财务数据和估值 |
| 宏观分析 | 经济数据和分析 |
| 行业研究 | 行业报告和数据 |

## 替代方案

| 工具 | 说明 |
|------|------|
| Yahoo Finance | 免费替代 |
| Finviz | 筛选替代 |
| FRED | 宏观数据替代 |
| Seeking Alpha | 分析替代 |
