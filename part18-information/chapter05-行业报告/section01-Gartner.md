# Gartner

## 定义

> **Gartner**是全球领先的信息技术研究和咨询公司，提供IT行业的市场分析、预测和建议。

## 网址

https://www.gartner.com

## 主要产品

| 产品 | 说明 |
|------|------|
| **Magic Quadrant** | 市场领导者/挑战者/远见者/利基者 |
| **Hype Cycle** | 技术成熟度曲线 |
| **Market Share** | 市场份额数据 |
| **Forecasts** | 市场预测 |

## Magic Quadrant

将公司分为四类：

| 类型 | 特征 |
|------|------|
| **Leaders** | 执行力强+愿景完整 |
| **Challengers** | 执行力强+愿景不足 |
| **Visionaries** | 愿景完整+执行力不足 |
| **Niche Players** | 专注细分市场 |

## Hype Cycle

技术发展的五个阶段：

| 阶段 | 说明 |
|------|------|
| 技术萌芽期 | 新技术出现 |
| 期望膨胀期 | 炒作高峰 |
| 泡沫破裂谷底期 | 失望期 |
| 爬升恢复期 | 逐步成熟 |
| 生产力成熟期 | 广泛应用 |

## 如何使用

| 用途 | 方法 |
|------|------|
| 行业趋势 | 了解技术发展方向 |
| 竞争格局 | 了解公司在行业中的位置 |
| 投资决策 | 发现行业领导者 |
| 风险评估 | 了解技术风险 |

## 思考题

1. 为什么说Gartner的Magic Quadrant对投资有参考价值？
2. Hype Cycle的哪个阶段最适合投资？
3. Gartner的预测一定准确吗？
