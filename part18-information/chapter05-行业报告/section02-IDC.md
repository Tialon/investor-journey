# IDC

## 定义

> **IDC（International Data Corporation）**是全球领先的IT市场研究和咨询公司，提供IT行业的市场数据和预测。

## 网址

https://www.idc.com

## 主要数据

| 数据 | 说明 |
|------|------|
| 市场规模 | 各细分市场规模 |
| 市场份额 | 公司市场份额 |
| 预测 | 市场增长预测 |
| 季度追踪 | 季度出货量数据 |

## 与Gartner的区别

| 维度 | IDC | Gartner |
|------|-----|---------|
| 重点 | 市场数据 | 战略咨询 |
| 优势 | 量化数据 | 定性分析 |
| 产品 | 市场追踪 | Magic Quadrant |

## 如何使用

| 用途 | 方法 |
|------|------|
| 市场规模 | 了解TAM |
| 市场份额 | 了解竞争格局 |
| 增长预测 | 评估成长性 |
| 季度追踪 | 跟踪业绩 |
