# McKinsey

## 定义

> **McKinsey & Company（麦肯锡）**是全球最知名的管理咨询公司之一，发布大量行业研究报告。

## 网址

https://www.mckinsey.com

## 主要报告类型

| 类型 | 说明 |
|------|------|
| 行业报告 | 各行业深度分析 |
| 技术趋势 | AI、数字化等 |
| 宏观经济 | 全球经济展望 |
| 管理实践 | 最佳管理实践 |

## 知名报告

| 报告 | 说明 |
|------|------|
| **McKinsey Global Institute** | 全球经济研究 |
| **McKinsey Quarterly** | 季度商业期刊 |
| **State of AI** | AI年度报告 |

## 如何使用

| 用途 | 方法 |
|------|------|
| 行业趋势 | 了解行业发展方向 |
| 技术趋势 | 了解技术影响 |
| 宏观分析 | 了解全球经济 |
| 投资决策 | 发现行业机会 |

## 优势与局限

| 优势 | 局限 |
|------|------|
| 深度分析 | 可能有咨询偏见 |
| 全球视野 | 免费内容有限 |
| 高质量 | 不是投资建议 |
| 权威性 | 更新频率不定 |
