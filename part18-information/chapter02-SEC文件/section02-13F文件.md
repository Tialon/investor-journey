# 13F文件

## 定义

> **13F**是管理超过1亿美元的机构投资者每季度向SEC提交的持仓报告，披露其持有的美国股票。

## 提交要求

| 要求 | 说明 |
|------|------|
| 门槛 | 管理资产 > 1亿美元 |
| 频率 | 每季度 |
| 截止日 | 季度结束后45天 |
| 内容 | 多头持仓（不含空头） |

## 主要内容

| 信息 | 说明 |
|------|------|
| 持仓股票 | 持有哪些股票 |
| 持仓数量 | 每只股票持有多少股 |
| 持仓市值 | 每只股票的市值 |
| 机构名称 | 哪个机构持有 |

## 如何使用13F

### 跟踪大师持仓

| 机构 | 说明 |
|------|------|
| Berkshire Hathaway | 巴菲特 |
| Bridgewater | 达利奥 |
| Renaissance Technologies | 量化基金 |
| Tiger Global | 科技股 |

### 分析方法

| 方法 | 说明 |
|------|------|
| 新增持仓 | 大师在买什么 |
| 减少持仓 | 大师在卖什么 |
| 集中度 | 持仓是否集中 |
| 跟投 | 跟随大师投资 |

## 局限性

| 局限性 | 说明 |
|--------|------|
| 延迟45天 | 不是实时数据 |
| 只有美国股票 | 不含海外持仓 |
| 只有多头 | 不含空头持仓 |
| 可能过时 | 大师可能已经卖出 |

## 获取方式
