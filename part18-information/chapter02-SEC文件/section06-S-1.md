# S-1（IPO招股书）

## 定义

> **S-1**是公司首次公开募股（IPO）前向SEC提交的注册声明，包含公司最全面的业务和财务信息。

## 主要内容

| 部分 | 内容 |
|------|------|
| **Business** | 公司业务描述 |
| **Risk Factors** | 风险因素 |
| **Financial Statements** | 历史财务数据 |
| **Use of Proceeds** | 募集资金用途 |
| **Management** | 管理层介绍 |
| **Selling Shareholders** | 老股东出售情况 |

## S-1 vs 10-K

| 维度 | S-1 | 10-K |
|------|-----|------|
| 时机 | IPO前 | 上市后每年 |
| 目的 | 募资 | 信息披露 |
| 详细程度 | 非常详细 | 详细 |
| 财务数据 | 3年历史 | 当年+历史 |

## 如何使用S-1

| 用途 | 方法 |
|------|------|
| 了解公司 | IPO前全面了解 |
| 评估估值 | 结合财务数据 |
| 发现风险 | 公司自己披露的风险 |
| 老股东行为 | 是否在套现 |

## 关注要点

### 积极信号

| 信号 | 说明 |
|------|------|
| 收入高增长 | 业务在扩张 |
| 毛利率上升 | 竞争力增强 |
| 老股东不卖 | 看好公司前景 |
| 资金用于研发 | 投资未来 |

### 消极信号

| 信号 | 说明 |
|------|------|
| 老股东大量套现 | 不看好公司 |
| 资金用于还债 | 财务压力大 |
| 风险因素过多 | 业务风险大 |
| 盈利模式不清 | 可持续性存疑 |
