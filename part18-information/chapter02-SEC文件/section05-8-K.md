# 8-K

## 定义

> **8-K**是美国上市公司在发生**重大事件**时必须向SEC提交的报告。它是投资者获取实时重大信息的重要来源。

## 需要报告的事件

| 事件类型 | 例子 |
|----------|------|
| **高管变动** | CEO辞职、CFO更换 |
| **重大收购** | 收购其他公司 |
| **资产出售** | 出售子公司 |
| **破产** | 申请破产保护 |
| **财报发布** | 季度/年度业绩 |
| **股息变化** | 增加/减少分红 |
| **股票回购** | 新的回购计划 |
| **诉讼** | 重大法律诉讼 |
| **信用评级变化** | 评级上调/下调 |
| **会计变更** | 更换审计师 |

## 如何使用8-K

| 用途 | 方法 |
|------|------|
| 实时跟踪 | 关注重大事件 |
| 发现催化剂 | 收购、回购等 |
| 识别风险 | 高管离职、诉讼 |
| 验证预期 | 财报是否达预期 |

## 获取方式

| 来源 | 说明 |
|------|------|
| SEC EDGAR | 官方来源 |
| 公司IR页面 | 通常第一时间发布 |
| 财经新闻 | 报道重大8-K |

## 红旗信号

| 信号 | 可能含义 |
|------|----------|
| CEO突然辞职 | 可能有内部问题 |
| 更换审计师 | 可能有分歧 |
| 信用评级下调 | 财务风险增加 |
| 大额资产出售 | 可能在筹资 |

2. 如何从8-K中发现投资机会或风险？
3. CEO突然辞职一定是坏事吗？
