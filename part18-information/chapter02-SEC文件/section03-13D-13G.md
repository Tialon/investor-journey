# 13D/13G

## 定义

> **13D**和**13G**是当投资者持有上市公司**超过5%**股份时必须向SEC提交的披露文件。

## 13D vs 13G

| 维度 | 13D | 13G |
|------|-----|-----|
| 提交者 | 激进投资者 | 被动投资者 |
| 意图 | 可能推动变革 | 纯投资 |
| 截止日 | 达到5%后10天 | 季度结束后45天 |
| 信号 | 强烈 | 温和 |

## 13D的信号意义

当激进投资者提交13D时，通常意味着：

| 信号 | 说明 |
|------|------|
| 可能推动变革 | 要求董事会席位 |
| 可能推动收购 | 寻求出售公司 |
| 可能推动回购 | 要求增加回购 |
| 可能推动分拆 | 要求拆分业务 |

## 典型激进投资者

| 投资者 | 风格 |
|--------|------|
| Carl Icahn | 推动公司变革 |
| Elliott Management | 激进投资 |
| Third Point | 事件驱动 |
| Starboard Value | 推动运营改善 |

## 如何使用13D/13G

| 用途 | 方法 |
|------|------|
| 发现机会 | 13D提交 → 可能有催化剂 |
| 验证投资 | 大股东增持 → 信心信号 |
| 风险提示 | 大股东减持 → 可能有问题 |
