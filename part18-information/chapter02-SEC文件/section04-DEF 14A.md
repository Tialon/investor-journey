# DEF 14A（代理声明）

## 定义

> **DEF 14A（Proxy Statement）**是公司在年度股东大会前向SEC提交的文件，包含股东投票事项、管理层薪酬、董事会信息等。

## 主要内容

| 部分 | 内容 | 重要性 |
|------|------|--------|
| **管理层薪酬** | CEO等高管的薪酬 | ⭐⭐⭐⭐ |
| **董事会信息** | 董事的背景和独立性 | ⭐⭐⭐ |
| **股东提案** | 股东提出的投票事项 | ⭐⭐⭐ |
| **审计师** | 审计费用和审计师 | ⭐⭐ |
| **股权结构** | 大股东持股比例 | ⭐⭐⭐ |

## 如何使用DEF 14A

### 管理层薪酬

| 关注点 | 说明 |
|--------|------|
| 薪酬总额 | 是否合理 |
| 薪酬结构 | 固定 vs 浮动 |
| 业绩挂钩 | 是否与业绩挂钩 |
| 与同行对比 | 是否高于同行 |

### 董事会

| 关注点 | 说明 |
|--------|------|
| 独立性 | 独立董事占比 |
| 多样性 | 背景是否多元 |
| 专业性 | 是否有行业专家 |

### 股东提案

| 关注点 | 说明 |
|--------|------|
| 提案内容 | 股东关心什么 |
| 管理层态度 | 支持还是反对 |
| 投票结果 | 提案是否通过 |

## 红旗信号

| 信号 | 可能含义 |
|------|----------|
| 管理层薪酬远高于同行 | 治理问题 |
| 独立董事占比低 | 缺乏监督 |
| 频繁的股东激进提案 | 股东不满 |
| 审计费用异常 | 可能有审计问题 |

## 思考题

2. 如何从管理层薪酬判断公司治理质量？
3. 股东提案通过率高说明什么？
