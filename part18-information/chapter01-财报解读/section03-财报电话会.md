# 财报电话会

## 定义

> **财报电话会（Earnings Call）**是公司在发布财报后举行的电话会议，管理层介绍业绩并回答分析师提问。

## 结构

```
1. 管理层陈述（~20分钟）
   ├── 业绩概览
   ├── 业务进展
   └── 未来展望

2. Q&A环节（~40分钟）
   └── 分析师提问 → 管理层回答
```

| 来源 | 说明 |
|------|------|
| 公司IR页面 | 官方直播和录音 |
| Seeking Alpha | 转录稿 |
| The Motley Fool | 转录稿 |

## 如何阅读财报电话会

### 管理层陈述

| 关注点 | 说明 |
|--------|------|
| 语气 | 自信还是回避？ |
| 重点 | 强调什么？回避什么？ |
| 展望 | 对未来的判断 |
| 数据 | 关键指标 |

### Q&A环节

| 关注点 | 说明 |
|--------|------|
| 分析师的问题 | 尖锐还是温和？ |
| 管理层的回答 | 直接还是绕弯？ |
| 回避的问题 | 什么问题不回答？ |
| 新信息 | 是否有新披露？ |

## 关键信号

### 积极信号

| 信号 | 说明 |
|------|------|
| 管理层自信 | 语气坚定，数据支持 |
| 上调指引 | 提高全年预期 |
| 新业务进展 | 有具体数据 |
| 回购加速 | 对公司有信心 |

### 消极信号

| 信号 | 说明 |
|------|------|
| 管理层回避 | 不正面回答问题 |
| 下调指引 | 降低全年预期 |
| 高管离职 | 核心团队变动 |
| 找借口 | 将问题归咎于外部 |

## 名言

> "电话会上管理层说什么不重要，怎么说才重要。"

## 思考题

1. 为什么说"Q&A环节比管理层陈述更重要"？
2. 如何从财报电话会中发现管理层的"言外之意"？
3. 一家公司下调指引但股价上涨，可能是什么原因？
