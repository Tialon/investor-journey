# 10-Q季报

## 定义

> **10-Q**是美国上市公司向SEC提交的**季度报告**，包含季度财务数据和业务更新。

## 10-Q vs 10-K

| 维度 | 10-Q | 10-K |
|------|------|------|
| 频率 | 每年3次（Q1/Q2/Q3） | 每年1次 |
| 详细程度 | 较简略 | 最详细 |
| 审计 | 未经审计 | 经审计 |
| 内容 | 财务数据+业务更新 | 全面信息 |

## 主要内容

| 部分 | 内容 |
|------|------|
| **财务报表** | 季度三大报表 |
| **MD&A** | 季度业务讨论 |
| **风险更新** | 新增风险因素 |
| **法律诉讼** | 诉讼进展 |

## 如何使用10-Q

| 目的 | 方法 |
|------|------|
| 跟踪业绩 | 对比季度数据 |
| 发现趋势 | 连续季度对比 |
| 识别风险 | 关注新增风险 |
| 验证预期 | 对比分析师预期 |

## 关注要点

1. **收入增速** — 是否达到预期
2. **利润率变化** — 是否在恶化
3. **现金流** — 利润质量
4. **管理层展望** — 对未来的判断
5. **新增风险** — 是否有新问题

2. 如何用10-Q数据验证公司是否在按计划执行？
3. Q3的10-Q和Q1/Q2有什么不同？
