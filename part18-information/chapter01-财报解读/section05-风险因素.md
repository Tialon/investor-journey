# 风险因素

## 定义

> **风险因素（Risk Factors）**是10-K中公司自己披露的可能影响业务的风险。它是公司"自揭其短"的部分。

## 风险类型

| 类型 | 说明 | 例子 |
|------|------|------|
| **业务风险** | 核心业务的风险 | 客户集中度高 |
| **竞争风险** | 行业竞争 | 新进入者 |
| **技术风险** | 技术变化 | 被替代的风险 |
| **监管风险** | 政策变化 | 反垄断、数据隐私 |
| **财务风险** | 财务相关 | 汇率、利率 |
| **运营风险** | 运营相关 | 供应链、人才 |

## 如何使用风险因素

### 投资分析

| 用途 | 方法 |
|------|------|
| **了解风险** | 公司面临的主要风险 |
| **验证投资** | 是否能接受这些风险 |
| **发现变化** | 新增了哪些风险 |
| **对比同行** | 风险是否与同行一致 |

### 风险变化分析

| 变化 | 含义 |
|------|------|
| 风险因素增加 | 业务可能面临新挑战 |
| 风险因素措辞变化 | 可能更严重或已缓解 |
| 风险因素删除 | 可能已解决或不想提 |

## 典型风险因素

### 科技公司

- 技术变革风险
- 人才竞争风险
- 网络安全风险
- 监管风险

### 消费品公司

- 消费者偏好变化
- 原材料价格波动
- 竞争加剧
- 品牌声誉风险

### 金融公司

- 信用风险
- 利率风险
- 监管风险
- 市场风险

## 红旗信号

| 信号 | 可能含义 |
|------|----------|
| 风险因素大幅增加 | 业务环境恶化 |
| 新增监管风险 | 可能面临政策变化 |
| 新增诉讼风险 | 可能有法律问题 |
| 风险因素过于笼统 | 可能在隐藏具体问题 |

## 思考题

1. 为什么说"风险因素是公司自揭其短"？
2. 如何从风险因素的变化中发现投资机会？
3. 风险因素是否应该作为不投资的理由？
