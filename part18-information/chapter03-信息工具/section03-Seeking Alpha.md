# Seeking Alpha

## 定义

> **Seeking Alpha**是一个以深度股票分析著称的平台，提供分析师文章、财报转录和投资观点。

## 网址

https://seekingalpha.com

## 主要功能

| 功能 | 说明 |
|------|------|
| 分析文章 | 社区分析师的深度文章 |
| 财报转录 | Earnings Call Transcript |
| 量化评级 | SA自己的股票评级 |
| 新闻 | 财经新闻 |
| 分组 | 投资组合跟踪 |

## 分析文章

| 类型 | 说明 |
|------|------|
| 深度分析 | 对公司的全面分析 |
| 事件分析 | 财报、收购等事件 |
| 宏观分析 | 市场和经济分析 |
| 投资策略 | 投资方法论 |

## 量化评级

Seeking Alpha有自己的量化评级系统：

| 维度 | 说明 |
|------|------|
| 估值 | PE、PB等 |
| 增长 | 收入、利润增速 |
| 盈利能力 | ROE、利润率 |
| 动量 | 股价趋势 |
| 修正 | 分析师预期修正 |

## 财报转录

免费获取Earnings Call Transcript。

## 优势与局限

| 优势 | 局限 |
|------|------|
| 深度分析 | 需要付费订阅 |
| 多元观点 | 文章质量参差 |
| 财报转录免费 | 可能有偏见 |
| 量化评级 | 不构成投资建议 |

## 思考题

1. 如何判断Seeking Alpha上的分析文章是否可信？
2. Seeking Alpha的量化评级有什么参考价值？
3. 如何利用Seeking Alpha的财报转录进行分析？
