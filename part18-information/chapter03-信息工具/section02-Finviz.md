# Finviz

## 定义

> **Finviz（Financial Visualizations）**是一个以股票筛选和可视化著称的金融数据平台。

## 网址

https://finviz.com

## 主要功能

| 功能 | 说明 |
|------|------|
| 股票筛选 | 多条件筛选股票 |
| 热力图 | 市场可视化 |
| 内幕交易 | 内幕人士买卖 |
| 新闻聚合 | 财经新闻 |
| 股票报价 | 基本数据 |

## 股票筛选器

### 可筛选条件

| 类别 | 条件 |
|------|------|
| 基本面 | PE、PB、ROE、利润率 |
| 技术面 | RSI、移动平均线 |
| 行业 | 行业分类 |
| 市值 | 大盘/中盘/小盘 |
| 成交量 | 日均成交量 |

### 筛选示例

```
ROE > 15%
PE < 20
收入增速 > 10%
市值 > 100亿
```

## 热力图

以图形方式展示市场表现：
- 颜色深浅 = 涨跌幅
- 面积大小 = 市值
- 分行业/分指数

## 内幕交易

追踪公司高管和大股东的买卖行为。

| 信号 | 含义 |
|------|------|
| 内幕大量买入 | 可能看好公司 |
| 内幕大量卖出 | 可能不看好 |

## 优势与局限

| 优势 | 局限 |
|------|------|
| 免费版功能强 | 免费版有延迟 |
| 筛选功能强大 | 深度分析有限 |
| 可视化好 | 数据可能有误 |
| 更新及时 | 无法导出数据 |

2. 内幕交易数据对投资有什么参考价值？
3. Finviz的筛选结果是否可以直接作为投资依据？
