# Yahoo Finance

## 定义

> **Yahoo Finance**是最受欢迎的免费金融数据平台之一，提供股票报价、财务数据、新闻和分析。

## 网址

https://finance.yahoo.com

## 主要功能

| 功能 | 说明 |
|------|------|
| 股票报价 | 实时/延迟报价 |
| 财务数据 | 三大报表、关键指标 |
| 图表 | 股价走势图 |
| 新闻 | 财经新闻聚合 |
| 分析 | 分析师评级和预期 |
| 历史数据 | 历史股价下载 |

## 如何使用

| 用途 | 方法 |
|------|------|
| 查看股价 | 输入股票代码 |
| 查看财务数据 | Financials标签 |
| 查看分析师预期 | Analysis标签 |
| 下载历史数据 | Historical Data标签 |
| 查看新闻 | News标签 |

## 优势与局限

| 优势 | 局限 |
|------|------|
| 免费 | 数据可能有延迟 |
| 全面 | 深度分析有限 |
| 易用 | 广告较多 |
| 历史数据丰富 | 无法自定义筛选 |

## 替代方案

| 工具 | 特点 |
|------|------|
| Google Finance | 类似功能 |
| Finviz | 筛选功能强 |
| Seeking Alpha | 深度分析 |
| TradingView | 图表功能强 |
