# 公司投资者关系

## 定义

> **投资者关系（Investor Relations，IR）页面**是公司官方网站上专门为投资者提供信息的页面。

## 典型内容

| 内容 | 说明 |
|------|------|
| 财报 | 季度/年度财报 |
| SEC文件 | 10-K、10-Q、8-K等 |
| 财报电话会 | 直播和录音 |
| 新闻稿 | 公司新闻 |
| 股票信息 | 股价、分红等 |
| 公司治理 | 董事会、管理层 |
| ESG | 可持续发展报告 |

## 如何使用IR页面

| 用途 | 方法 |
|------|------|
| 获取最新信息 | 第一时间获取财报和新闻 |
| 听电话会 | 直播或回放 |
| 下载文件 | 财报、演示文稿 |
| 了解公司 | 公司介绍、业务描述 |

## IR页面 vs SEC EDGAR

| 维度 | IR页面 | SEC EDGAR |
|------|--------|-----------|
| 更新速度 | 更快 | 较慢 |
| 内容范围 | 更广泛 | 仅SEC文件 |
| 用户体验 | 更好 | 较差 |
| 可靠性 | 官方 | 官方 |

## 获取方式

通常格式：
- `https://investor.company.com`
- `https://company.com/investors`
