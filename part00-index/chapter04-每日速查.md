# ⚡ 每日速查

## 常用公式

### 期权
| 公式 | 说明 |
|------|------|
| 盈亏平衡(Call) | 行权价 + 权利金 |
| 盈亏平衡(Put) | 行权价 - 权利金 |
| 等效股票 | Delta × 100 × 张数 |
| 时间衰减 | Theta × 天数 |
| 最大盈利(Iron Condor) | 净权利金 |
| 最大亏损(Credit Spread) | 行权价间距 - 净权利金 |

### 估值
| 公式 | 说明 |
|------|------|
| PE | 股价 / EPS |
| PEG | PE / 增长率 |
| EV/EBITDA | 企业价值 / EBITDA |
| FCF Yield | FCF / 市值 |
| DCF | Σ(FCF/(1+r)^t) + 终值 |

### 财务
| 公式 | 说明 |
|------|------|
| ROE | 净利润 / 股东权益 |
| ROIC | NOPAT / 投入资本 |
| 毛利率 | (收入-成本) / 收入 |
| 净利率 | 净利润 / 收入 |
| 自由现金流 | 经营现金流 - 资本支出 |

### 风险管理
| 公式 | 说明 |
|------|------|
| Kelly公式 | f* = (bp - q) / b |
| 夏普比率 | (Rp - Rf) / σp |
| 最大回撤 | (峰值 - 谷值) / 峰值 |
