# DCF基础


## 核心思想

> 一家公司的价值 = 未来所有自由现金流的现值之和

1902\text{企业价值} = \sum_{t=1}^{n} \frac{FCF_t}{(1 + WACC)^t} + \frac{终值}{(1 + WACC)^n}1902

## DCF的三个核心输入

| 输入 | 说明 |
|------|------|
| **FCF预测** | 未来每年的自由现金流 |
| **折现率(WACC)** | 资本成本 |
| **终值** | 永续价值 |

## DCF的优缺点

| 优点 | 缺点 |
|------|------|
| 理论最正确 | 对假设极敏感 |
| 关注现金流 | 预测困难 |
| 不受市场情绪影响 | 垃圾进垃圾出 |
