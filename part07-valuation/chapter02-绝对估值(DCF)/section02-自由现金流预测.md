# 自由现金流预测


## 预测方法

### 方法一：收入驱动

1902FCF = 收入 \times FCF利润率 - CapEx - \Delta NWC1902

### 方法二：直接预测

直接预测FCF的增长率。

## 预测期

| 期限 | 说明 |
|------|------|
| 显式预测期 | 通常5-10年 |
| 终值 | 永续价值 |

## 关键假设

| 假设 | 影响 |
|------|------|
| 收入增速 | 驱动FCF增长 |
| 利润率 | 驱动FCF水平 |
| CapEx | 驱动FCF扣除 |
| 营运资本 | 驱动FCF调整 |

## 思考题

1. 为什么预测期通常选5-10年？
2. 如何合理假设收入增速？
3. 收入增速和利润率哪个更重要？
