# SOTP分部估值法


## 定义

> **SOTP（Sum of the Parts）**是将公司各业务板块分别估值，然后加总的方法。

## 适用场景

- 多元化集团
- 有独立业务板块的公司
- 有隐藏资产的公司

## 步骤

1. 识别各业务板块
2. 分别估值（可用不同方法）
3. 加总
4. 减去净负债/加净现金

## 例子：Amazon

| 业务板块 | 估值方法 | 估值 |
|----------|----------|------|
| 电商 | PS | 00B |
| AWS | EV/EBITDA | T |
| 广告 | PS | 00B |
| 减：净负债 | | -0B |
| **总计** | | **.95T** |
