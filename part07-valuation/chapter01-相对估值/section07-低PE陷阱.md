# 低PE陷阱


## 什么是低PE陷阱？

低PE不一定意味着便宜。有时PE低是因为市场预期公司未来盈利会大幅下降。

## 常见原因

| 原因 | 说明 |
|------|------|
| 周期顶部 | 盈利高但即将下滑 |
| 行业衰退 | 行业基本面恶化 |
| 一次性收益 | 非经常性利润推高EPS |
| 管理层问题 | 治理风险折价 |

## 如何避免

1. 分析盈利可持续性
2. 关注行业周期位置
3. 检查非经常性损益
4. 对比同行PE水平

## 思考题

1. 为什么周期股在PE最低时最贵？
2. 如何判断低PE是"价值陷阱"还是"真正便宜"？
3. 哪些行业容易出现低PE陷阱？
