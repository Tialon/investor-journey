# 银行股估值


## 特点

- 高杠杆
- 资产以金融资产为主
- 利差驱动

## 常用指标

| 指标 | 说明 |
|------|------|
| PB | 最常用 |
| PE | 辅助 |
| 股息率 | 重要参考 |

## 为什么银行股用PB？

- 资产以金融资产为主，账面价值接近公允价值
- PE受周期影响大
- PB更稳定
