# REIT估值


## 特点

- 强制高分红（90%以上应税收入）
- 资产密集
- 现金流稳定

## 常用指标

| 指标 | 说明 |
|------|------|
| P/FFO | 最常用（类似PE） |
| P/AFFO | 考虑维护性CapEx |
| 股息率 | 重要参考 |
| PB | 辅助 |

## FFO vs AFFO

| 指标 | 公式 |
|------|------|
| FFO | 净利润 + 折旧 - 资产出售收益 |
| AFFO | FFO - 维护性CapEx |
