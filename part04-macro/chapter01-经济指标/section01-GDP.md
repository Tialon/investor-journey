# GDP（国内生产总值）

## 定义

> **GDP（Gross Domestic Product）**是一个国家在特定时期内生产的所有最终商品和服务的市场价值总和。它是衡量经济规模和健康程度的最重要指标。

## 计算方法

### 支出法

$$\text{GDP} = C + I + G + (X - M)$$

| 组成 | 含义 | 占比（美国） |
|------|------|-------------|
| **C** | 消费支出 | ~68% |
| **I** | 投资支出 | ~18% |
| **G** | 政府支出 | ~17% |
| **X-M** | 净出口 | ~-3% |

### 收入法

$$\text{GDP} = 工资 + 利润 + 利息 + 租金 + 间接税 - 补贴 + 折旧$$

## GDP数据发布

| 项目 | 说明 |
|------|------|
| 发布机构 | 美国经济分析局（BEA） |
| 发布频率 | 每季度（初值→修正值→终值） |
| 发布时间 | 季度结束后约30天 |
| 数据网站 | https://www.bea.gov |

## GDP增速

$$\text{GDP增速} = \frac{\text{本期GDP} - \text{上期GDP}}{\text{上期GDP}} \times 100\%$$

| 增速 | 评价 |
|------|------|
| > 3% | 强劲增长 |
| 2-3% | 温和增长 |
| 0-2% | 低迷 |
| < 0% | 衰退（连续两个季度负增长） |

## GDP的局限性

| 局限性 | 说明 |
|--------|------|
| 不含非市场活动 | 家务劳动、志愿服务 |
| 不考虑收入分配 | GDP增长但贫富差距加大 |
| 不计环境成本 | 污染、资源耗竭 |
| 不反映幸福感 | GDP高不一定幸福 |

## 对投资的影响

| GDP信号 | 市场影响 |
|---------|----------|
| GDP超预期增长 | 股市利好，利率可能上升 |
| GDP低于预期 | 股市利空，利率可能下降 |
| GDP连续负增长 | 衰退信号，避险资产受益 |
