# CPI（消费者物价指数）

## 定义

> **CPI（Consumer Price Index）**衡量的是消费者购买一篮子商品和服务的价格变化。它是最常用的通胀指标。

## 计算方式

$$\text{CPI} = \frac{\text{当期一篮子价格}}{\text{基期一篮子价格}} \times 100$$

$$\text{CPI同比} = \frac{\text{本期CPI} - \text{去年同期CPI}}{\text{去年同期CPI}} \times 100\%$$

## CPI构成（美国）

| 类别 | 权重 | 说明 |
|------|------|------|
| 住房 | ~35% | 房租、业主等价租金 |
| 交通 | ~17% | 汽油、二手车 |
| 食品饮料 | ~14% | 食品、饮料 |
| 医疗 | ~8% | 医疗服务、药品 |
| 教育通信 | ~6% | 学费、电话费 |
| 娱乐 | ~6% | 娱乐商品和服务 |
| 其他 | ~14% | 服装、个人护理等 |

## 核心CPI vs 总体CPI

| 指标 | 包含内容 | 关注点 |
|------|----------|--------|
| **总体CPI** | 所有商品和服务 | 反映消费者实际感受 |
| **核心CPI** | 剔除食品和能源 | 反映基础通胀趋势 |

**为什么剔除食品和能源？** 这两项价格波动大，容易掩盖基础通胀趋势。

## 数据发布

| 项目 | 说明 |
|------|------|
| 发布机构 | 美国劳工统计局（BLS） |
| 发布频率 | 每月 |
| 发布时间 | 月中（约每月12日） |

## CPI对市场的影响

| CPI信号 | 市场影响 |
|---------|----------|
| CPI高于预期 | 利率上升，股市利空，债市利空 |
| CPI低于预期 | 利率下降，股市利好，债市利好 |
| CPI持续下降 | 降息预期升温 |
| CPI持续上升 | 加息预期升温 |

## CPI vs PPI vs PCE

| 指标 | 衡量对象 | 美联储关注度 |
|------|----------|-------------|
| **CPI** | 消费者价格 | 高 |
| **PPI** | 生产者价格 | 中（先行指标） |
| **PCE** | 个人消费支出 | 最高（首选指标） |
