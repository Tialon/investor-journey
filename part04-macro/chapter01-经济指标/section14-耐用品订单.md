# 耐用品订单

## 定义

> **耐用品订单（Durable Goods Orders）**是制造商收到的使用寿命超过3年的商品订单。它是衡量企业投资意愿的重要指标。

## 耐用品分类

| 类别 | 例子 |
|------|------|
| 交通运输 | 飞机、汽车 |
| 机械设备 | 工业设备 |
| 电子产品 | 计算机、通信设备 |
| 金属制品 | 钢铁产品 |

## 核心耐用品订单

$$\text{核心耐用品} = \text{耐用品} - \text{交通运输}$$

**为什么剔除交通运输？** 飞机订单波动大（单笔大额订单影响大）。

## 数据发布

| 项目 | 说明 |
|------|------|
| 发布机构 | 美国人口普查局 |
| 发布频率 | 每月 |
| 发布时间 | 月末 |

## 对投资的影响

| 信号 | 市场影响 |
|------|----------|
| 订单增加 | 企业投资增加，经济向好 |
| 订单减少 | 企业投资减少，经济放缓 |
| 核心订单稳定 | 经济基础稳固 |
