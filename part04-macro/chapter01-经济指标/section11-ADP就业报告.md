# ADP就业报告

## 定义

> **ADP就业报告**是美国自动数据处理公司（ADP）发布的私营部门就业数据。它比非农早两天发布，被称为"小非农"。

## 数据特点

| 特点 | 说明 |
|------|------|
| 发布时间 | 每月第一个**周三**（比非农早2天） |
| 覆盖范围 | 仅私营部门 |
| 数据来源 | ADP处理的工资单数据 |

## ADP vs 非农

| 维度   | ADP  | 非农   |
| ---- | ---- | ---- |
| 发布时间 | 周三   | 周五   |
| 覆盖范围 | 私营部门 | 全部门  |
| 数据来源 | 工资单  | 调查   |
| 准确性  | 预测性  | 官方数据 |
| 市场影响 | 较小   | 很大   |

## 对投资的影响

| 信号 | 市场影响 |
|------|----------|
| ADP强劲 | 非农可能强劲，加息预期 |
| ADP疲软 | 非农可能疲软，降息预期 |

**注意**：ADP和非农有时会大幅背离，不能完全替代。

## 思考题

1. 为什么ADP被称为"小非农"？
2. ADP和非农数据背离时应该以哪个为准？
3. ADP数据对市场的影响为什么比非农小？
