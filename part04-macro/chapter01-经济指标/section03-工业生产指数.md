# 工业生产指数

## 定义

> **工业生产指数（Industrial Production Index）**衡量制造业、采矿业和公用事业的实际产出变化。

## 构成

| 行业 | 权重 |
|------|------|
| 制造业 | ~75% |
| 采矿业 | ~14% |
| 公用事业 | ~11% |

## 数据发布

| 项目 | 说明 |
|------|------|
| 发布机构 | 美联储 |
| 发布频率 | 每月 |
| 发布时间 | 月中 |

## 对投资的影响

| 信号 | 市场影响 |
|------|----------|
| 产出增加 | 经济扩张 |
| 产出减少 | 经济放缓 |
| 产出持续下降 | 衰退信号 |
