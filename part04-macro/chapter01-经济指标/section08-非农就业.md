# 非农就业

## 定义

> **非农就业（Non-Farm Payrolls, NFP）**是每月新增的非农业就业人数。它是衡量劳动力市场健康程度的最重要指标，也是对市场影响最大的经济数据之一。

## 数据构成

| 指标 | 说明 |
|------|------|
| 新增就业人数 | 当月新增非农就业 |
| 失业率 | 失业人口占劳动力比例 |
| 劳动参与率 | 劳动力占适龄人口比例 |
| 平均时薪 | 时薪变化（通胀信号） |

## 数据发布

| 项目 | 说明 |
|------|------|
| 发布机构 | 美国劳工统计局（BLS） |
| 发布频率 | 每月 |
| 发布时间 | **每月第一个周五** |
| 数据覆盖 | 上月数据 |

## 非农数据的意义

| 信号 | 含义 |
|------|------|
| 新增 > 20万 | 劳动力市场强劲 |
| 新增 15-20万 | 温和增长 |
| 新增 < 10万 | 劳动力市场疲软 |
| 新增 < 0 | 就业减少，衰退信号 |

## 非农对市场的影响

| 非农信号 | 股市 | 债市 | 美元 | 黄金 |
|----------|------|------|------|------|
| 超预期强劲 | 短期涨 | 跌 | 涨 | 跌 |
| 低于预期 | 跌 | 涨 | 跌 | 涨 |
| 远低于预期 | 大跌 | 大涨 | 大跌 | 大涨 |

**逻辑**：就业强劲 → 经济好 → 加息预期 → 利率上升 → 股市承压

## 平均时薪的重要性

$$\text{时薪增速} → \text{通胀压力} → \text{美联储反应}$$

| 时薪增速 | 含义 |
|----------|------|
| > 4% | 通胀压力大 |
| 3-4% | 温和 |
| < 3% | 通胀压力小 |
