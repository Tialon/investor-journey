# PPI（生产者物价指数）

## 定义

> **PPI（Producer Price Index）**衡量的是生产者（企业）出售商品和服务的价格变化。它是CPI的先行指标——生产端价格变化最终会传导到消费端。

## PPI vs CPI

| 维度 | PPI | CPI |
|------|-----|-----|
| 衡量对象 | 生产者出售价格 | 消费者购买价格 |
| 传导方向 | 上游→下游 | 终端 |
| 先行性 | 先行指标 | 同步/滞后指标 |
| 波动性 | 更大 | 较小 |

## PPI构成

| 阶段 | 说明 |
|------|------|
| **原材料** | 未加工的商品 |
| **中间品** | 半成品 |
| **成品** | 最终商品 |

## 传导机制

$$\text{原材料涨价} → \text{中间品涨价} → \text{成品涨价} → \text{CPI上升}$$

**但传导不是100%**——企业可能压缩利润、提高效率来消化成本。

## 数据发布

| 项目 | 说明 |
|------|------|
| 发布机构 | 美国劳工统计局（BLS） |
| 发布频率 | 每月 |
| 发布时间 | CPI前一天 |

## PPI对市场的影响

| PPI信号 | 市场影响 |
|---------|----------|
| PPI高于预期 | 通胀压力上升，利率可能上升 |
| PPI低于预期 | 通胀压力缓解，利率可能下降 |
| PPI持续上升 | 可能预示CPI上升 |
| PPI持续下降 | 可能预示经济放缓 |
