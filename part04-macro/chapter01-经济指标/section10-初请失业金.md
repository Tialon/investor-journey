# 初请失业金

## 定义

> **初请失业金（Initial Jobless Claims）**是每周首次申请失业保险的人数。它是劳动力市场的**高频先行指标**。

## 数据特点

| 特点 | 说明 |
|------|------|
| 频率 | **每周四**发布 |
| 覆盖 | 上周数据 |
| 先行性 | 比月度数据更及时 |

## 水平参考

| 水平 | 含义 |
|------|------|
| < 20万 | 劳动力市场非常强劲 |
| 20-25万 | 正常 |
| 25-30万 | 偏弱 |
| > 30万 | 劳动力市场恶化 |
| > 50万 | 衰退信号 |

## 对投资的影响

| 信号 | 市场影响 |
|------|----------|
| 初请大幅上升 | 经济放缓信号，降息预期 |
| 初请持续下降 | 经济好转，加息预期 |
