# 失业率

## 定义

> **失业率（Unemployment Rate）**是失业人口占劳动力总数的比例。

$$\text{失业率} = \frac{\text{失业人口}}{\text{劳动力总数}} \times 100\%$$

## 失业率水平

| 水平 | 评价 |
|------|------|
| < 4% | 充分就业 |
| 4-5% | 正常 |
| 5-7% | 偏高 |
| > 7% | 高失业 |
| > 10% | 严重失业 |

## 自然失业率

> **自然失业率**是指在不引发通胀的情况下，经济能达到的最低失业率。美国约为4%左右。

当失业率 < 自然失业率 → 工资上涨压力 → 通胀压力

## 数据发布

| 项目 | 说明 |
|------|------|
| 发布机构 | 美国劳工统计局（BLS） |
| 发布频率 | 每月 |
| 发布时间 | 与非农同时（每月第一个周五） |

## 失业率的局限性

| 局限性 | 说明 |
|--------|------|
| 不含放弃找工作的人 | 劳动参与率下降时失业率可能"虚低" |
| 不含兼职想找全职的人 | U-6失业率更全面 |
| 滞后指标 | 经济好转后企业才开始招人 |

## U-3 vs U-6

| 指标 | 包含内容 |
|------|----------|
| **U-3** | 官方失业率 |
| **U-6** | U-3 + 想找全职的兼职者 + 放弃找工作的人 |
