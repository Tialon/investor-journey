# ISM制造业指数

## 定义

> **ISM制造业指数**是美国供应管理协会（ISM）发布的制造业PMI。它是美国最权威的PMI数据。

## 数据构成

| 分项 | 说明 |
|------|------|
| 新订单 | 需求 |
| 产出 | 生产 |
| 就业 | 劳动力 |
| 供应商交货 | 供应链 |
| 库存 | 库存 |
| 价格 | 价格压力 |
| 新出口订单 | 出口需求 |
| 进口 | 进口 |

## 数据发布

| 项目 | 说明 |
|------|------|
| 发布机构 | 美国供应管理协会（ISM） |
| 发布频率 | 每月 |
| 发布时间 | 每月第一个工作日 |

## ISM vs 标普全球PMI

| 维度 | ISM | 标普全球 |
|------|-----|----------|
| 调查对象 | 采购经理 | 采购经理 |
| 覆盖范围 | 美国 | 全球 |
| 权重方法 | 等权重 | 加权 |
| 市场影响 | 更大 | 较大 |
