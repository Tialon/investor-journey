# 美元指数（DXY）

## 定义

> **美元指数（DXY）**衡量美元相对于一篮子主要货币的汇率变化。

## 构成

| 货币 | 权重 |
|------|------|
| 欧元 | 57.6% |
| 日元 | 13.6% |
| 英镑 | 11.9% |
| 加元 | 9.1% |
| 瑞典克朗 | 4.2% |
| 瑞士法郎 | 3.6% |

## 影响因素

| 因素 | 影响 |
|------|------|
| 利率差异 | 美国利率高→美元强 |
| 经济表现 | 美国经济好→美元强 |
| 避险需求 | 危机时→美元强 |
| 贸易差额 | 逆差大→美元弱 |

## 强美元 vs 弱美元

| 美元 | 对美股 | 对黄金 | 对新兴市场 |
|------|--------|--------|-----------|
| 强 | 中性偏负 | 利空 | 利空 |
| 弱 | 利好 | 利好 | 利好 |

## 思考题

1. 为什么欧元占美元指数的57.6%？
2. 强美元对美国企业盈利有什么影响？
3. 避险时为什么美元会走强？
