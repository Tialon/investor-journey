# 利率与股市的关系

## 核心逻辑

$$\text{利率上升} → \text{资金成本上升} → \text{投资/消费减少} → \text{企业盈利下降} → \text{股市承压}$$

## 影响渠道

| 渠道 | 说明 |
|------|------|
| 企业融资成本 | 利率高→借贷成本高→利润减少 |
| 消费者支出 | 利率高→房贷/消费贷成本高→消费减少 |
| 估值模型 | 利率高→折现率高→股票估值下降 |
| 资金流向 | 利率高→债券吸引力增加→资金流出股市 |

## 估值影响

$$\text{股票估值} = \frac{\text{未来盈利}}{\text{折现率} - \text{增长率}}$$

利率上升 → 折现率上升 → 估值下降

## 历史规律

| 利率周期 | 股市表现 |
|----------|----------|
| 加息初期 | 通常上涨（经济好） |
| 加息末期 | 通常承压 |
| 降息初期 | 通常上涨 |
| 降息末期 | 取决于经济 |

## 思考题

1. 为什么加息初期股市可能继续上涨？
2. 哪些行业对利率最敏感？
3. 利率上升时哪些行业可能受益？
