# 国债收益率

## 定义

> **国债收益率**是投资者持有美国国债获得的回报率。它是全球金融市场的"无风险利率"基准。

## 关键期限

| 期限 | 含义 |
|------|------|
| **2年** | 反映短期利率预期 |
| **5年** | 中期预期 |
| **10年** | 长期经济预期，最受关注 |
| **30年** | 超长期预期 |

## 影响因素

| 因素 | 影响 |
|------|------|
| 美联储政策 | 加息→短端上升 |
| 通胀预期 | 通胀高→收益率上升 |
| 经济预期 | 经济好→收益率上升 |
| 避险需求 | 危机→收益率下降 |

## 10年期国债的重要性

| 影响 | 说明 |
|------|------|
| 房贷利率 | 基于10年期国债 |
| 企业融资 | 基于国债利率+信用利差 |
| 股票估值 | 折现率的基准 |

## 对投资的影响

| 信号 | 市场影响 |
|------|----------|
| 收益率上升 | 股市承压，成长股尤甚 |
| 收益率下降 | 股市利好，成长股受益 |
| 收益率急升 | 市场恐慌 |

## 思考题

1. 为什么10年期国债收益率最受关注？
2. 国债收益率上升为什么对成长股影响更大？
3. 如何用国债收益率判断市场情绪？
