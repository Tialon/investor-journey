# 收益率曲线倒挂

## 定义

> **收益率曲线倒挂**是指短期国债收益率高于长期国债收益率，即2年期 > 10年期。

## 为什么是衰退信号？

| 逻辑 | 说明 |
|------|------|
| 市场预期 | 投资者预期未来经济不好 |
| 降息预期 | 预期美联储将降息刺激经济 |
| 避险需求 | 买入长期国债避险，推高价格压低收益率 |

## 历史记录

| 倒挂时间 | 衰退开始 | 间隔 |
|----------|----------|------|
| 1998年 | 2001年 | ~2年 |
| 2006年 | 2007年 | ~1年 |
| 2019年 | 2020年 | ~1年 |
| 2022年 | 未衰退 | — |

## 注意事项

| 要点 | 说明 |
|------|------|
| 不是即时信号 | 倒挂后可能1-2年才衰退 |
| 不是100%准确 | 2022年倒挂但未衰退 |
| 关注利差深度 | 利差越深信号越强 |

## 对投资的影响

| 信号 | 市场影响 |
|------|----------|
| 曲线倒挂 | 避险情绪升温 |
| 曲线恢复正常 | 可能即将衰退（或已衰退） |

3. 倒挂后应该做什么投资决策？
