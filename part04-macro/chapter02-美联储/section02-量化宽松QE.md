# 量化宽松（QE）

## 定义

> **量化宽松（Quantitative Easing, QE）**是美联储在利率降至零附近后，通过购买国债和抵押贷款支持证券（MBS）向市场注入流动性的非常规货币政策。

## QE的运作机制

```
美联储购买国债/MBS → 推高债券价格 → 压低长期利率 → 刺激借贷和投资
                → 增加市场流动性 → 推高资产价格
```

## 历史QE

| 时期 | 操作 | 背景 |
|------|------|------|
| QE1 (2008-2010) | 购买1.75万亿 | 金融危机 |
| QE2 (2010-2011) | 购买6000亿 | 经济复苏缓慢 |
| QE3 (2012-2014) | 每月850亿 | 持续刺激 |
| QE4 (2020-2022) | 无上限购买 | 疫情 |

## QE对市场的影响

| 影响 | 说明 |
|------|------|
| 股市上涨 | 流动性推高估值 |
| 债券收益率下降 | 购买推高债券价格 |
| 美元走弱 | 货币供应增加 |
| 黄金上涨 | 避险+通胀预期 |
| 房价上涨 | 低利率刺激购房 |

## QE vs QT

| 操作 | 方向 | 市场影响 |
|------|------|----------|
| QE | 购买资产 | 资产价格上涨 |
| QT（缩表） | 出售资产 | 资产价格承压 |
