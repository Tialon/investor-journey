# FOMC会议

## 定义

> **FOMC（Federal Open Market Committee，联邦公开市场委员会）**是美联储制定货币政策的核心机构。每年召开8次会议。

## FOMC组成

| 成员       | 人数        |
| -------- | --------- |
| 美联储理事会成员 | 7人        |
| 纽约联储主席   | 1人        |
| 其他地区联储主席 | 4人（轮换）    |
| **总计**   | **12人投票** |

## 会议日程

| 时间 | 内容 |
|------|------|
| 第一天 | 经济讨论 |
| 第二天 | 政策决定、声明发布 |
| 2:30 PM | 发布声明 |
| 2:30 PM | 美联储主席新闻发布会 |

## FOMC声明的关键内容

| 内容 | 说明 |
|------|------|
| 利率决定 | 加息/降息/维持 |
| 经济评估 | 对经济的判断 |
| 前瞻指引 | 未来政策方向 |
| 点阵图 | 委员的利率预期 |

## 对市场的影响

| 信号 | 市场影响 |
|------|----------|
| 鸽派（降息预期） | 股市涨，美元跌 |
| 鹰派（加息预期） | 股市跌，美元涨 |
| 符合预期 | 市场平稳 |
| 超出预期 | 市场大幅波动 |
