# 前瞻指引

## 定义

> **前瞻指引（Forward Guidance）**是美联储通过言语沟通来管理市场对未来政策路径预期的工具。

## 类型

| 类型 | 说明 | 例子 |
|------|------|------|
| **日历型** | 指定时间 | "在2025年底前不会加息" |
| **数据型** | 依据数据 | "在通胀降至2%前不会降息" |
| **定性型** | 模糊表述 | "在一段时间内保持宽松" |

## 为什么前瞻指引重要？

| 原因 | 说明 |
|------|------|
| 管理预期 | 市场提前反映政策 |
| 减少波动 | 避免政策突袭 |
| 增强效果 | 言语本身就能影响市场 |

## 美联储措辞解读

| 措辞 | 含义 |
|------|------|
| "耐心" | 短期内不会行动 |
| "视数据而定" | 接近行动 |
| "适当时候" | 可能很快行动 |
| "坚定承诺" | 强烈信号 |
